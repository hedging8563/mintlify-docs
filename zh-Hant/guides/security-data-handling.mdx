## 概述

在 AI Sonar 整合中安全處理 API key、使用者媒體、日誌與生成資產。

使用 AI Sonar 公開 API 與模型契約；不要依賴供應商執行細節。

## 關鍵規則

- 在向使用者返回成功前持久化 API key scope, media URL lifetime, audit events。
- 使用 AI Sonar 公開 API 與模型契約；不要依賴供應商執行細節。
- 讓重試保持冪等，避免重複生成或重複計費。

## 生產檢查清單

- 在向使用者返回成功前持久化 API key scope, media URL lifetime, audit events。
- 使用 AI Sonar 公開 API 與模型契約；不要依賴供應商執行細節。
- 讓重試保持冪等，避免重複生成或重複計費。
- 日誌要便於支援排障，但不要儲存密鑰或私有 payload。
- 生產流量變更前先在預發環境驗證行為。
