---
title: "重排文件"
openapi: "POST /v1/rerank"
description: "根據與查詢的相關性對文件進行重排"
---

使用語義相似度模型對文件進行重排。適用於優化搜尋結果和 RAG 應用程式。

## 請求主體

**同步請求逾時：** 這個非聊天端點會等待路由到的模型完成處理。大型輸入、長音訊或大量批次可能超過常見的 30s 用戶端預設逾時，因此請將 HTTP 用戶端逾時設定為至少 `120s`。

<ParamField body="model" type="string" required>
  要使用的重排模型 ID（例如：`BAAI/bge-reranker-v2-m3`、`qwen3-rerank`）。
</ParamField>

<ParamField body="query" type="string" required>
  用於對文件進行排名的查詢語句。最大長度：`32,000` 個字元。
</ParamField>

<ParamField body="documents" type="array" required>
  要進行重排的文件列表（字串）。限制：最多 `1,000` 份文件；每份文件最多 `100,000` 個字元；所有文件合計最多 `2,000,000` 個字元。
</ParamField>

<ParamField body="top_n" type="integer">
  要返回的前幾個結果數量。預設為所有文件。必須至少為 `1`，且不能大於 `documents.length`。AI Sonar 目前沒有已治理的供應商專屬更低硬上限；若供應商日後發布此類上限，必須先加入 rerank request-shape truth，再對外文件化或執行。
</ParamField>

<ParamField body="return_documents" type="boolean" default="false">
  是否在回應中包含原始文件文本。
</ParamField>

## 回應

<ResponseField name="results" type="array">
  帶有評分的已排序文件列表。

  每個結果包含：
  - `index` (integer)：原始文件索引
  - `relevance_score` (number)：相關性評分 (0-1)
  - `document` (string)：原始文本（若 `return_documents=true`）
</ResponseField>

<ResponseField name="model" type="string">
  用於重排的模型。
</ResponseField>

<ResponseField name="usage" type="object">
  Token 使用量統計。
</ResponseField>

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model": "BAAI/bge-reranker-v2-m3",
        "query": "What is machine learning?",
        "documents": []string{
            "Machine learning is a subset of AI",
            "The weather is nice today",
            "Deep learning uses neural networks",
        },
        "top_n": 2,
    }
    body, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.aisonar.dev/v1/rerank", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Println(result["results"])
}
```
