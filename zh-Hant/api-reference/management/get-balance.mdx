## 概覽

這個端點返回目前管理令牌所屬組織的餘額彙總。它使用與計費和結算相同的組織餘額事實源。

## 注意事項

- 金額真值欄位以 USD 返回：`balance`、`total_recharge`、`total_used`。
- `preferred_currency` 表示組織在 Dashboard 中的展示偏好，不會改變 USD 真值欄位。

## 範例

<RequestExample>
```bash cURL
curl -X GET "https://api.aisonar.dev/v1/management/balance" \
  -H "Authorization: Bearer mt-your-management-token"
```
</RequestExample>

## 請求 / 回應

使用上方的互動式 OpenAPI 面板以取得確切的架構。

## 回應範例

<ResponseExample>
```json 200 OK
{
  "object": "management.balance",
  "currency": "USD",
  "available_balance": "42.50",
  "pending_usage": "1.25",
  "updated_at": "2026-06-27T00:00:00.000Z"
}
```
</ResponseExample>

## 重要欄位

<ResponseField name="available_balance" type="string">用於計費對帳的 USD 十進位字串。</ResponseField>
<ResponseField name="pending_usage" type="string">用於計費對帳的 USD 十進位字串。</ResponseField>
<ResponseField name="updated_at" type="string">API 返回的 ISO 時間或 Unix 時間戳。</ResponseField>
