---
title: "列出模型"
openapi: "GET /v1beta/models"
description: "使用 Google Gemini API 格式列出可用模型"
---

以 Google Gemini API 格式傳回可用模型列表。

## 查詢參數

<ParamField query="pageSize" type="integer">
  要傳回的模型最大數量。預設值：`1000`，最大值：`1000`。
</ParamField>

<ParamField query="pageToken" type="string">
  上一頁回傳的不透明續頁權杖。將它原樣帶回即可取得下一頁。
</ParamField>

## 身份驗證

選填。支援與其他 Gemini 端點相同的身份驗證方法：
- `?key=YOUR_API_KEY` 查詢參數
- `x-goog-api-key: YOUR_API_KEY` 標頭
- `Authorization: Bearer YOUR_API_KEY` 標頭

## 回應

<ResponseField name="models" type="array">
  模型物件陣列。
</ResponseField>

<ResponseField name="nextPageToken" type="string">
  當仍有更多模型時回傳。將此值作為 `pageToken` 傳回即可繼續分頁。
</ResponseField>

## 錯誤

- 無效的 `pageToken` 會回傳 `400 INVALID_ARGUMENT`。
- 非預期的列表失敗會回傳 `500 INTERNAL`。

<RequestExample>
```bash cURL
curl "https://api.aisonar.dev/v1beta/models?pageSize=5" \
  -H "x-goog-api-key: sk-your-api-key"
```

```python Python
import google.generativeai as genai

genai.configure(
    api_key="sk-your-api-key",
    transport="rest",
    client_options={"api_endpoint": "api.aisonar.dev"}
)

for model in genai.list_models():
    print(model.name)
```

```javascript JavaScript
const response = await fetch(
  "https://api.aisonar.dev/v1beta/models?pageSize=5",
  { headers: { "x-goog-api-key": "sk-your-api-key" } }
);
const { models } = await response.json();
models.forEach(m => console.log(m.name));
```
</RequestExample>
