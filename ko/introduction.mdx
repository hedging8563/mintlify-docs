## AI Sonar란?

AI Sonar는 OpenAI, Anthropic, Google, DeepSeek, Moonshot, MiniMax 등과 같은 제공업체의 **수백 개의 AI 모델**에 대한 통합 접근을 제공하는 AI API 통합 플랫폼입니다.

## 지원되는 기능

공개 카탈로그는 시간이 지나며 바뀝니다. 현재 사용 가능 여부는 `/v1/models`, `recommended_for`, 또는 모델 페이지에서 확인하세요. 아래 예시는 현재 공개 모델 ID를 사용합니다.

| 기능 | 설명 | 예시 모델 |
|---|---|---|
| 채팅 및 추론 | 텍스트 생성, 계획, 도구 사용 | `gpt-5.4`, `claude-sonnet-4-6`, `deepseek-r1` |
| 비전 및 멀티모달 | 이미지 이해와 멀티모달 채팅 | `gpt-4o`, `claude-sonnet-4-6`, `gemini-3.5-flash` |
| 이미지 생성 및 편집 | 텍스트-이미지 생성과 이미지 편집 | `gpt-image-2`, `flux-pro`, `qwen-image` |
| 비디오 생성 | 텍스트/이미지/참조 비디오 기반 생성과 모션 제어 | `veo3.1`, `seedance-2.0`, `kling-3.0-video` |
| 오디오 및 Realtime | TTS, 전사, 번역, 실시간 세션 | `gpt-4o-mini-tts`, `whisper-1`, `gpt-realtime` |
| Embeddings 및 rerank | 텍스트 벡터화와 재정렬 | `text-embedding-3-small`, `qwen3-vl-rerank` |
| 번역 | 텍스트 및 음성 번역 워크플로 | `qwen-mt-plus`, `gpt-realtime-translate` |
| 음악 및 3D | 음악과 3D asset 생성 | `suno-music`, `tripo3d-v2.5` |
| Worlds | World 생성, 상태 polling, media asset 워크플로 | 현재 사용 가능 여부는 `/v1/worlds/generations`와 Models API로 확인하세요 |

## 요청 형식 선택

필요한 동작과 일치하는 경로로 시작하세요. 공급자 네이티브 기능에는 네이티브 Anthropic/Gemini 경로를 사용하고, 기존 OpenAI 스타일 채팅 코드를 마이그레이션할 때는 OpenAI 호환 `/v1/chat/completions`를 사용하세요.

<CodeGroup>

```bash cURL
curl https://api.aisonar.dev/v1/chat/completions \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-5.4",
    "messages": [
      {"role": "user", "content": "Explain what AI Sonar does in one sentence."}
    ]
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.aisonar.dev/v1"
)

response = client.chat.completions.create(
    model="gpt-5.4",
    messages=[{"role": "user", "content": "Explain what AI Sonar does in one sentence."}]
)

print(response.choices[0].message.content)
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.aisonar.dev/v1'
});

const response = await client.chat.completions.create({
  model: 'gpt-5.4',
  messages: [{ role: 'user', content: 'Explain what AI Sonar does in one sentence.' }],
});

console.log(response.choices[0].message.content);
```

<Note>
명시적으로 Responses 고유의 동작이 필요할 때만 `POST /v1/responses`를 사용하세요. 모든 모델에 걸친 기본 호환 경로는 아닙니다.
</Note>

</CodeGroup>
