## AI Sonar란?

AI Sonar는 OpenAI, Anthropic, Google, DeepSeek, Moonshot, MiniMax 등과 같은 제공업체의 **수백 개의 AI 모델**에 대한 통합 접근을 제공하는 AI API 통합 플랫폼입니다.

## 지원되는 기능

| 기능 | 설명 | 예시 모델 |
|------------|-------------|----------------|
| **채팅 및 추론** | 텍스트 생성, 계획 수립, 도구 활용 | `gpt-5.4`, `claude-sonnet-4-6`, `deepseek-r1` |
| **비전** | 이미지 이해 및 멀티모달 채팅 | `gpt-4o`, `claude-sonnet-4-6`, `gemini-2.5-flash` |
| **이미지 생성** | 텍스트-투-이미지 및 이미지 편집 | `dall-e-3`, `flux-pro`, `midjourney` |
| **오디오** | TTS 및 전사 | `gpt-4o-mini-tts`, `whisper-1`, `gpt-4o-transcribe` |
| **임베딩** | 텍스트 벡터화 | `text-embedding-3-small`, `text-embedding-3-large` |
| **음악** | AI 음악 생성 | `suno` |
| **3D** | 3D 자산 생성 | `tripo3d` |

## 요청 형식 선택

필요한 동작과 일치하는 경로로 시작하세요. 공급자 네이티브 기능에는 네이티브 Anthropic/Gemini 경로를 사용하고, 기존 OpenAI 스타일 채팅 코드를 마이그레이션할 때는 OpenAI 호환 `/v1/chat/completions`를 사용하세요.

<CodeGroup>

```bash cURL
curl https://api.aisonar.dev/v1/chat/completions \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-5.4",
    "messages": [
      {"role": "user", "content": "Explain what AI Sonar does in one sentence."}
    ]
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.aisonar.dev/v1"
)

response = client.chat.completions.create(
    model="gpt-5.4",
    messages=[{"role": "user", "content": "Explain what AI Sonar does in one sentence."}]
)

print(response.choices[0].message.content)
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.aisonar.dev/v1'
});

const response = await client.chat.completions.create({
  model: 'gpt-5.4',
  messages: [{ role: 'user', content: 'Explain what AI Sonar does in one sentence.' }],
});

console.log(response.choices[0].message.content);
```

<Note>
명시적으로 Responses 고유의 동작이 필요할 때만 `POST /v1/responses`를 사용하세요. 모든 모델에 걸친 기본 호환 경로는 아닙니다.
</Note>

</CodeGroup>
