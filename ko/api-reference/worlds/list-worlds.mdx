AI Sonar에서 만든 완료된 World Labs Marble world를 나열합니다. 결과는 조직별로 격리되며 공유 상위 계정의 전체 목록은 노출하지 않습니다.

대기 중이거나 실패한 생성 작업은 `/v1/worlds/generations/{id}`를 사용하세요. 이 목록은 생성 후 AI Sonar이 색인한 완료 world만 반환합니다.

## 요청 본문

<ParamField body="page_size" type="integer" default="20">
  반환할 world 수, 1~100.
</ParamField>

<ParamField body="page_token" type="string">
  이전 응답의 페이지 토큰.
</ParamField>

<ParamField body="status" type="string">
  선택 상태 필터. AI Sonar은 `SUCCEEDED`에 대해 완료된 world를 반환하며 다른 상태는 생성 작업 엔드포인트에서 확인합니다.
</ParamField>

<ParamField body="model" type="string">
  선택 Marble 모델 필터.
</ParamField>

<ParamField body="tags" type="array">
  선택 태그 필터.
</ParamField>

<ParamField body="is_public" type="boolean">
  선택 공개 여부 필터.
</ParamField>

<ParamField body="created_after" type="string">
  이 시각 이후 생성된 world만 반환합니다.
</ParamField>

<ParamField body="created_before" type="string">
  이 시각 이전 생성된 world만 반환합니다.
</ParamField>

<ParamField body="sort_by" type="string" default="created_at">
  `created_at` 또는 `updated_at` 기준 정렬.
</ParamField>

<ParamField body="sort_order" type="string" default="desc">
  정렬 방향: `asc` 또는 `desc`.
</ParamField>

## 응답

<ResponseField name="worlds" type="array">
  World 레코드 배열.
</ResponseField>

<ResponseField name="next_page_token" type="string">
  다음 페이지 토큰 또는 `null`.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.aisonar.dev/v1/worlds/list" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{"page_size": 20, "status": "SUCCEEDED", "model": "marble-1.1"}'
```

```python Python
import requests

response = requests.post(
    "https://api.aisonar.dev/v1/worlds/list",
    headers={"Authorization": "Bearer sk-your-api-key"},
    json={"page_size": 20, "status": "SUCCEEDED", "model": "marble-1.1"},
)
print(response.json()["worlds"])
```

```javascript JavaScript

    'Content-Type': 'application/json'
```
