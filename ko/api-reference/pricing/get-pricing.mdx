## 개요

이 엔드포인트는 활성 모델에 대한 현재 공개 가격 표면을 조회할 때 사용합니다.

단일 모델의 전체 가격 상세와 구조화된 가격, 출처까지 보려면 [모델 가격 조회](/ko/api-reference/models/get-model) 를 사용하세요.

## 쿼리 파라미터

<ParamField query="provider" type="string">
  선택 사항인 공급자 필터입니다. 예: `openai`, `anthropic`, `google`.
</ParamField>

<ParamField query="tag" type="string">
  선택 사항인 모델 태그 필터입니다. 예: `chat`, `image`, `video`, `embedding`, `translation`.
</ParamField>

## 응답

<ResponseField name="object" type="string">
  항상 `pricing`입니다.
</ResponseField>

<ResponseField name="updated_at" type="string">
  응답에 포함된 최신 가격 항목의 ISO 타임스탬프입니다.
</ResponseField>

<ResponseField name="currency" type="string">
  항상 `USD`입니다.
</ResponseField>

<ResponseField name="data" type="array">
  가격 항목 배열입니다. 각 항목에는 `model`, `provider`, `is_lock_price`, `capabilities`, `aliases`, 그리고 상단의 대화형 스키마가 반환하는 공개 스칼라 또는 구조화 가격 요약 필드가 포함됩니다.
</ResponseField>
