## 응답

<ResponseField name="object" type="string">
  항상 `list`입니다.
</ResponseField>

<ResponseField name="data" type="array">
  모델 객체 배열입니다.

  각 모델에는 다음이 포함됩니다:
  - `id` (string): 모델 식별자
  - `object` (string): `model`
  - `created` (integer): 생성 타임스탬프
  - `owned_by` (string): 모델 제공자
  - `aisonar.category` (string): 공개 모델 카테고리
  - `aisonar.pricing_unit` (string 또는 `null`): 공개 가격 단위
  - `aisonar.capability_flags` (object 또는 `null`): 공개 기능 요약
  - `aisonar.supported_operations` (array 또는 `null`): 공개 비채팅 작업 목록
  - `aisonar.pricing_provenance` (object 또는 `null`): 공개 가격 출처 메타데이터
  - `aisonar.public_contract_summary` (object 또는 `null`): `public_operations` 와 `request_endpoint` 만 포함하는 간단한 비채팅 탐색 요약
  - `aisonar.agent_preferences` (object): `recommended_for` 가 있을 때만 반환
</ResponseField>

<Note>
`GET /v1/models` 는 탐색에 최적화되어 있습니다. `capability_flags`, `pricing_provenance`, 전체 `public_contract` 같은 상세 전용 메타데이터는 `GET /v1/models/{model}` 에 있습니다.
</Note>

## 쿼리 파라미터

<ParamField query="category" type="string">
  선택 사항인 공개 카테고리 필터입니다. `chat`, `image`, `video`, `audio`, `tts`, `stt`, `music`, `3d`, `embedding`, `rerank`, `translation` 을 지원합니다.
</ParamField>

<ParamField query="recommended_for" type="string">
  선택 사항인 비채팅 추천 장면입니다. `image`, `video`, `music`, `3d`, `tts`, `stt`, `embedding`, `rerank`, `translation` 을 지원합니다.
</ParamField>

<ParamField query="provider" type="string">
  선택 사항인 공급자 필터입니다. 예: `openai`, `anthropic`, `google`, `deepseek`.
</ParamField>

<ParamField query="tag" type="string">
  선택 사항인 모델 태그 필터입니다. 예: `chat`, `image`, `video`, `embedding`, `translation`.
</ParamField>

<Note>
`recommended_for` 가 있을 때 `/v1/models` 는 비채팅 모델을 최신 캐시된 24시간 성공률 스냅샷 기준으로 정렬합니다. `status = "insufficient_samples"` 인 모델은 계속 보이지만 점수화된 모델 뒤에 정렬됩니다.
</Note>

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.aisonar.dev/v1"
)

models = client.models.list()

for model in models.data:
    print(f"{model.id} ({model.owned_by})")
```

```go Go
package main

import (
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    req, _ := http.NewRequest("GET", "https://api.aisonar.dev/v1/models", nil)
    req.Header.Set("Authorization", "Bearer sk-your-api-key")

    resp, _ := http.DefaultClient.Do(req)
    defer resp.Body.Close()

    var payload struct {
        Data []struct {
            ID      string `json:"id"`
            OwnedBy string `json:"owned_by"`
        } `json:"data"`
    }

    json.NewDecoder(resp.Body).Decode(&payload)

    for _, model := range payload.Data {
        fmt.Printf("%s (%s)\n", model.ID, model.OwnedBy)
    }
}
```
