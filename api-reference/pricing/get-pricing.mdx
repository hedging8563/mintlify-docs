title: "Get Pricing"
openapi: "GET /v1/pricing"
description: "Get pricing information for models"
---

## Overview

Use this endpoint to list the current public pricing surface for active models.

For one model's full pricing detail, including structured pricing and provenance, prefer [Get Model Pricing](/api-reference/models/get-model).

## Query Parameters

<ParamField query="provider" type="string">
  Optional provider filter such as `openai`, `anthropic`, or `google`.
</ParamField>

<ParamField query="tag" type="string">
  Optional model tag filter such as `chat`, `image`, `video`, `embedding`, or `translation`.
</ParamField>

## Response

<ResponseField name="object" type="string">
  Always `pricing`.
</ResponseField>

<ResponseField name="updated_at" type="string">
  ISO timestamp for the newest pricing entry included in the response.
</ResponseField>

<ResponseField name="currency" type="string">
  Always `USD`.
</ResponseField>

<ResponseField name="data" type="array">
  Array of pricing entries. Each item includes `model`, `provider`, `is_lock_price`, `capabilities`, `aliases`, and the public scalar or structured pricing summary fields returned by the interactive schema above.
</ResponseField>
