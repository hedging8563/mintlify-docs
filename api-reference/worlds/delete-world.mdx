---
title: "Delete World"
openapi: "DELETE /v1/worlds/{id}"
description: "Deletes a World Labs Marble world"
---

Delete a Marble world created through AI Sonar. AI Sonar forwards the deletion to World Labs and marks the local world record as deleted.

Deletion is permanent.

## Path Parameters

<ParamField path="id" type="string" required>
  World Labs world ID.
</ParamField>

## Response

<ResponseField name="deleted" type="boolean">
  Whether the deletion was accepted.
</ResponseField>

<ResponseField name="world_id" type="string">
  World Labs world ID.
</ResponseField>

<RequestExample>
```bash cURL
curl -X DELETE "https://api.aisonar.dev/v1/worlds/world_123" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
import requests

response = requests.delete(
    "https://api.aisonar.dev/v1/worlds/world_123",
    headers={"Authorization": "Bearer sk-your-api-key"},
)
print(response.json())
```

```javascript JavaScript
const response = await fetch('https://api.aisonar.dev/v1/worlds/world_123', {
  method: 'DELETE',
  headers: { Authorization: 'Bearer sk-your-api-key' }
});
console.log(await response.json());
```
</RequestExample>
