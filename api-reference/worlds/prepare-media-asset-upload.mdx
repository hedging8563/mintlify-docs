---
title: "Prepare Media Asset Upload"
openapi: "POST /v1/worlds/media-assets/prepare-upload"
description: "Creates a World Labs media asset upload target"
---

Create a World Labs media asset record and receive an upload URL for an image or video. Upload the binary file to `upload_info.upload_url`, then reference the returned `media_asset.media_asset_id` from `world_prompt` with `source: "media_asset"`.

AI Sonar records the asset owner. Only the same organization can retrieve or use this media asset in world generation.

## Request Body

<ParamField body="file_name" type="string" required>
  Original file name, up to 64 characters.
</ParamField>

<ParamField body="kind" type="string" required>
  Media type: `image` or `video`.
</ParamField>

<ParamField body="extension" type="string">
  Optional extension such as `png`, `jpg`, or `mp4`.
</ParamField>

<ParamField body="metadata" type="object">
  Optional metadata object stored with the upstream media asset.
</ParamField>

## Response

<ResponseField name="media_asset" type="object">
  Prepared World Labs media asset metadata.
</ResponseField>

<ResponseField name="upload_info.upload_url" type="string">
  Signed upstream URL where the binary file should be uploaded.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.aisonar.dev/v1/worlds/media-assets/prepare-upload" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "file_name": "reference.png",
    "kind": "image",
    "extension": "png"
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.aisonar.dev/v1/worlds/media-assets/prepare-upload",
    headers={"Authorization": "Bearer sk-your-api-key"},
    json={"file_name": "reference.png", "kind": "image", "extension": "png"},
)

print(response.json()["media_asset"]["media_asset_id"])
```

```javascript JavaScript
const response = await fetch('https://api.aisonar.dev/v1/worlds/media-assets/prepare-upload', {
  method: 'POST',
  headers: {
    Authorization: 'Bearer sk-your-api-key',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({ file_name: 'reference.png', kind: 'image', extension: 'png' })
});

const prepared = await response.json();
console.log(prepared.media_asset.media_asset_id, prepared.upload_info.upload_url);
```
