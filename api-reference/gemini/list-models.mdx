---
title: "List Models"
openapi: "GET /v1beta/models"
description: "List available models using Google Gemini API format"
---

Returns a list of available models in Google Gemini API format.

## Query Parameters

<ParamField query="pageSize" type="integer">
  Maximum number of models to return. Default: `1000`, maximum: `1000`.
</ParamField>

<ParamField query="pageToken" type="string">
  Continuation token returned by a previous page. Pass it back unchanged to fetch the next slice.
</ParamField>

## Authentication

Authentication is not required for the public model catalog read endpoints.

Gemini-style auth hints such as `?key=` or `x-goog-api-key` do not change the response contract here.

## Response

<ResponseField name="models" type="array">
  Array of model objects.
</ResponseField>

<ResponseField name="nextPageToken" type="string">
  Returned when more models are available. Pass this value as `pageToken` to continue pagination.
</ResponseField>

## Errors

- Invalid `pageToken` returns `400 INVALID_ARGUMENT`.
- Unexpected list failures return `500 INTERNAL`.

<RequestExample>
```bash cURL
curl "https://api.aisonar.dev/v1beta/models?pageSize=5" \
  -H "x-goog-api-key: sk-your-api-key"
```

```python Python
import google.generativeai as genai

genai.configure(
    api_key="sk-your-api-key",
    transport="rest",
    client_options={"api_endpoint": "api.aisonar.dev"}
)

for model in genai.list_models():
    print(model.name)
```

```javascript JavaScript
const response = await fetch(
  "https://api.aisonar.dev/v1beta/models?pageSize=5",
  { headers: { "x-goog-api-key": "sk-your-api-key" } }
);
const { models } = await response.json();
models.forEach(m => console.log(m.name));
```
</RequestExample>
