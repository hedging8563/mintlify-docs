## Why Coding Agents Are Expensive

Coding agents like Claude Code, Cursor, and Codex CLI are powerful — but they consume tokens at a rate that surprises most teams.

**Long context per request.** Every call includes a system prompt, project files, and conversation history — often 10,000–100,000+ tokens before the model even starts thinking.

**High-frequency calls.** A single coding session triggers dozens of API calls: code generation, search, review, autocomplete, and tool use. A 1-hour session can easily hit 200+ requests.

**Conversation accumulation.** Each turn resends the full message history. By turn 20, you're paying for the same context 20 times over.

<Info>
A typical 1-hour Claude Code session can consume 2–5M tokens. At direct API rates, that's $6–30+ per hour depending on the model.
</Info>
