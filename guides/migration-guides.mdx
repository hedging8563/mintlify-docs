## Overview

Move OpenAI, Anthropic, Gemini, OpenRouter, LiteLLM, and media workloads to AI Sonar with minimal behavior drift.

Use public AI Sonar APIs and model contracts; do not depend on provider execution details.

## Key rules

- Persist old model ID, new model ID, rollout owner before returning success to users.
- Use public AI Sonar APIs and model contracts; do not depend on provider execution details.
- Make retries idempotent and avoid duplicate billable work.

## Production checklist

- Persist old model ID, new model ID, rollout owner before returning success to users.
- Use public AI Sonar APIs and model contracts; do not depend on provider execution details.
- Make retries idempotent and avoid duplicate billable work.
- Keep logs useful for support without storing secrets or private payloads.
- Verify behavior in staging before changing production traffic.
