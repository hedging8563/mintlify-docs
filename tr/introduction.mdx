## AI Sonar Nedir?

## Desteklenen Yetenekler

| Yetenek | Açıklama | Örnek Modeller |
|------------|-------------|----------------|
| **Sohbet ve Muhakeme** | Metin üretimi, planlama, araçlar | `gpt-5.4`, `claude-sonnet-4-6`, `deepseek-r1` |
| **Görsel** | Görüntü anlama ve multimodal sohbet | `gpt-4o`, `claude-sonnet-4-6`, `gemini-2.5-flash` |
| **Görüntü Üretimi** | Metinden görüntüye ve görüntü düzenleme | `dall-e-3`, `flux-pro`, `midjourney` |
| **Ses** | TTS ve transkripsiyon | `gpt-4o-mini-tts`, `whisper-1`, `gpt-4o-transcribe` |
| **Gömlemeler** | Metin vektörleştirme | `text-embedding-3-small`, `text-embedding-3-large` |
| **Müzik** | AI ile müzik üretimi | `suno` |
| **3B** | 3B varlık üretimi | `tripo3d` |

## İstek Formatını Seçin

İhtiyacınız olan davranışa uyan rotayla başlayın. Sağlayıcıya özgü özellikler için yerel Anthropic/Gemini rotalarını kullanın; mevcut OpenAI tarzı sohbet kodunu taşırken OpenAI uyumlu `/v1/chat/completions` kullanın.

<CodeGroup>

```bash cURL
curl https://api.aisonar.dev/v1/chat/completions \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-5.4",
    "messages": [
      {"role": "user", "content": "Explain what AI Sonar does in one sentence."}
    ]
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.aisonar.dev/v1"
)

response = client.chat.completions.create(
    model="gpt-5.4",
    messages=[{"role": "user", "content": "Explain what AI Sonar does in one sentence."}]
)

print(response.choices[0].message.content)
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.aisonar.dev/v1'
});

const response = await client.chat.completions.create({
  model: 'gpt-5.4',
  messages: [{ role: 'user', content: 'Explain what AI Sonar does in one sentence.' }],
});

console.log(response.choices[0].message.content);
```

<Note>
Yalnızca Responses'a özgü davranışa açıkça ihtiyaç duyduğunuzda `POST /v1/responses` kullanın. Bu tüm modeller için varsayılan uyumluluk yolu değildir.
</Note>

</CodeGroup>
