## Desteklenen Yetenekler

Public katalog zamanla değişir. Güncel kullanılabilirliği `/v1/models`, `recommended_for` veya Models sayfasından kontrol edin; aşağıdaki örnekler güncel public model ID’lerini kullanır.

| Yetenek | Açıklama | Örnek modeller |
|---|---|---|
| Chat ve akıl yürütme | Metin üretimi, planlama, araçlar | `gpt-5.4`, `claude-sonnet-4-6`, `deepseek-r1` |
| Vision ve multimodal | Görüntü anlama ve multimodal sohbet | `gpt-4o`, `claude-sonnet-4-6`, `gemini-3.5-flash` |
| Görüntü üretimi ve düzenleme | Metinden görüntü ve görüntü düzenleme | `gpt-image-2`, `flux-pro`, `qwen-image` |
| Video üretimi | Metin/görüntü/referanstan video ve hareket kontrolü | `veo3.1`, `seedance-2.0`, `kling-3.0-video` |
| Ses ve Realtime | TTS, transkripsiyon, çeviri ve realtime oturumlar | `gpt-4o-mini-tts`, `whisper-1`, `gpt-realtime` |
| Embeddings ve rerank | Metin vektörleştirme ve yeniden sıralama | `text-embedding-3-small`, `qwen3-vl-rerank` |
| Çeviri | Metin ve konuşma çevirisi iş akışları | `qwen-mt-plus`, `gpt-realtime-translate` |
| Müzik ve 3D | Müzik ve 3D asset üretimi | `suno-music`, `tripo3d-v2.5` |
| Worlds | World üretimi, durum polling ve media asset iş akışları | Güncel kullanılabilirliği `/v1/worlds/generations` ve Models API ile kontrol edin |

## İstek Formatını Seçin

İhtiyacınız olan davranışa uyan rotayla başlayın. Sağlayıcıya özgü özellikler için yerel Anthropic/Gemini rotalarını kullanın; mevcut OpenAI tarzı sohbet kodunu taşırken OpenAI uyumlu `/v1/chat/completions` kullanın.

<CodeGroup>

```bash cURL
curl https://api.aisonar.dev/v1/chat/completions \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-5.4",
    "messages": [
      {"role": "user", "content": "Explain what AI Sonar does in one sentence."}
    ]
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.aisonar.dev/v1"
)

response = client.chat.completions.create(
    model="gpt-5.4",
    messages=[{"role": "user", "content": "Explain what AI Sonar does in one sentence."}]
)

print(response.choices[0].message.content)
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.aisonar.dev/v1'
});

const response = await client.chat.completions.create({
  model: 'gpt-5.4',
  messages: [{ role: 'user', content: 'Explain what AI Sonar does in one sentence.' }],
});

console.log(response.choices[0].message.content);
```

<Note>
Yalnızca Responses'a özgü davranışa açıkça ihtiyaç duyduğunuzda `POST /v1/responses` kullanın. Bu tüm modeller için varsayılan uyumluluk yolu değildir.
</Note>

</CodeGroup>
