## Genel Bakış

Bu endpoint, AI Sonar'nın OpenAI uyumlu Batch API'sine hizmet eder; yüklenen batch girişlerini ve oluşturulan output veya error dosyalarını kapsar.

İstek Anthropic Files uyumluluk moduna geçtiğinde, aynı route bağlı workspace için kullanılabilen Anthropic tarzı dosyaları da listeleyebilir.

## Notlar

- Batch giriş dosyaları `purpose=batch` kullanmalıdır.
- Oluşturulan output dosyalarına odaklanmak için `purpose=batch_output` kullanın.
- `completion_window` şu anda `24h`'dir.
- Output sırası garanti edilmez; her zaman `custom_id` ile eşleştirin.
- Streaming, batch öğelerinin içinde desteklenmez.

## Örnek

<RequestExample>
```bash cURL
curl -X GET "https://api.aisonar.dev/v1/files?purpose=batch&limit=20" \
  -H "Authorization: Bearer sk-your-api-key"
```
</RequestExample>

## Operasyonel İpuçları

- Downstream mutabakatı idempotent yapmak için `custom_id` kullanın.
- `output_file_id` ve `error_file_id` yalnızca worker batch'i tamamladıktan sonra beklenmelidir.
- `isBatchRequest=true` indirim kuralları uygulandığı için batch fiyatlandırması senkron fiyatlandırmadan farklı olabilir.
