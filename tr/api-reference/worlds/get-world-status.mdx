---
title: "World Durumunu Al"
openapi: "GET /v1/worlds/generations/{id}"
description: "World Labs Marble dünya üretim görevinin durumunu ve sonucunu alır"
---

World Labs Marble üretim görevinin durumunu kontrol etmek için bu endpointi poll edin. Oluşturma yanıtı `poll_url` içeriyorsa tam olarak o URLyi kullanmayı tercih edin.

## Yol Parametreleri

<ParamField path="id" type="string" required>
  Dünya üretim görevi IDsi.
</ParamField>

## Yanıt

<ResponseField name="id" type="string">
  Genel görev IDsi.
</ResponseField>

<ResponseField name="task_id" type="string">
  Asenkron görev tanımlayıcı aliası.
</ResponseField>

<ResponseField name="operation_id" type="string">
  Upstream World Labs operation IDsi.
</ResponseField>

<ResponseField name="status" type="string">
  Görev durumu: `pending`, `processing`, `completed`, `failed` veya `timeout`.
</ResponseField>

<ResponseField name="world_id" type="string">
  Tamamlandığında upstream World Labs world IDsi.
</ResponseField>

<ResponseField name="world_marble_url" type="string">
  Tamamlandığında oluşturulan Marble dünya URLsi.
</ResponseField>

<ResponseField name="model_url" type="string">
  Birincil oluşturulan dünya/model URLsi için alias.
</ResponseField>

<ResponseField name="glb_url" type="string">
  Varsa collider mesh GLB URLsi.
</ResponseField>

<ResponseField name="pano_url" type="string">
  Varsa panorama görüntü URLsi.
</ResponseField>

<ResponseField name="thumbnail_url" type="string">
  Varsa thumbnail görüntü URLsi.
</ResponseField>

<ResponseField name="caption" type="string">
  Oluşturulan dünya için upstream caption.
</ResponseField>

<ResponseField name="cost" type="object">
  Varsa `total_credits` dahil upstream operation maliyet ayrıntıları.
</ResponseField>

<ResponseField name="error" type="string">
  Görev başarısız olduğunda hata mesajı.
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.aisonar.dev/v1/worlds/generations/ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
import requests
import time

task_id = "ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa"

while True:
    response = requests.get(
        f"https://api.aisonar.dev/v1/worlds/generations/{task_id}",
        headers={"Authorization": "Bearer sk-your-api-key"},
    )
    result = response.json()
    if result["status"] in ["completed", "failed", "timeout"]:
        print(result)
        break
    time.sleep(10)
```

```javascript JavaScript
const taskId = 'ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa';
const response = await fetch(`https://api.aisonar.dev/v1/worlds/generations/${taskId}`, {
  headers: { Authorization: 'Bearer sk-your-api-key' }
});

const result = await response.json();
console.log(result.status, result.world_marble_url);
```
