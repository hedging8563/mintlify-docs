---
title: "Worlds listele"
openapi: "POST /v1/worlds/list"
description: "AI Sonar ile oluşturulmuş tamamlanan World Labs Marble world kayıtlarını listeler"
---

AI Sonar ile oluşturulmuş tamamlanan World Labs Marble world kayıtlarını listeleyin. Sonuçlar organizasyonunuzla sınırlıdır; AI Sonar paylaşılan upstream hesabının genel listesini açmaz.

Bekleyen veya başarısız üretim görevleri için `/v1/worlds/generations/{id}` kullanın. Bu liste yalnızca üretimden sonra AI Sonar tarafından indekslenen tamamlanmış world kayıtlarını döndürür.

## İstek gövdesi

<ParamField body="page_size" type="integer" default="20">
  Döndürülecek world sayısı, 1 ile 100 arası.
</ParamField>

<ParamField body="page_token" type="string">
  Önceki yanıttaki sayfalama tokenı.
</ParamField>

<ParamField body="status" type="string">
  İsteğe bağlı durum filtresi. AI Sonar `SUCCEEDED` için tamamlanan world kayıtlarını döndürür; diğer durumlar üretim görevi endpointine aittir.
</ParamField>

<ParamField body="model" type="string">
  İsteğe bağlı Marble model filtresi.
</ParamField>

<ParamField body="tags" type="array">
  İsteğe bağlı etiket filtresi.
</ParamField>

<ParamField body="is_public" type="boolean">
  İsteğe bağlı görünürlük filtresi.
</ParamField>

<ParamField body="created_after" type="string">
  Yalnızca bu zamandan sonra oluşturulan world kayıtlarını döndürür.
</ParamField>

<ParamField body="created_before" type="string">
  Yalnızca bu zamandan önce oluşturulan world kayıtlarını döndürür.
</ParamField>

<ParamField body="sort_by" type="string" default="created_at">
  `created_at` veya `updated_at` ile sırala.
</ParamField>

<ParamField body="sort_order" type="string" default="desc">
  Sıralama yönü: `asc` veya `desc`.
</ParamField>

## Yanıt

<ResponseField name="worlds" type="array">
  World kayıtları dizisi.
</ResponseField>

<ResponseField name="next_page_token" type="string">
  Sonraki sayfa tokenı veya `null`.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.aisonar.dev/v1/worlds/list" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{"page_size": 20, "status": "SUCCEEDED", "model": "marble-1.1"}'
```

```python Python
import requests

response = requests.post(
    "https://api.aisonar.dev/v1/worlds/list",
    headers={"Authorization": "Bearer sk-your-api-key"},
    json={"page_size": 20, "status": "SUCCEEDED", "model": "marble-1.1"},
)
print(response.json()["worlds"])
```
