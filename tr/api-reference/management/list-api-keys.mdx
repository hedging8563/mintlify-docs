## Genel Bakış

Geçerli yönetim belirteciyle ilişkili kuruluştaki kullanıcı API anahtarlarını döndürür.

## Notlar

- Sistem tarafından yönetilen keyler sonuçlara dahil edilmez.

## Örnek

<RequestExample>
```bash cURL
curl -X GET "https://api.aisonar.dev/v1/management/api-keys" \
  -H "Authorization: Bearer mt-your-management-token"
```
</RequestExample>

## İstek / Yanıt

Kesin şema için yukarıdaki etkileşimli OpenAPI panelini kullanın.

## Yanıt örneği

<ResponseExample>
```json 200 OK
{
  "object": "list",
  "data": [{
    "id": "ak_abc123",
    "name": "Production app",
    "status": "active",
    "created_at": "2026-06-27T00:00:00.000Z",
    "last_used_at": "2026-06-27T01:00:00.000Z"
  }]
}
```
</ResponseExample>

## Önemli alanlar

<ResponseField name="data" type="array">Bu yanıtta dönen liste veya payload.</ResponseField>
<ResponseField name="status" type="string">Bu kaynağın güncel lifecycle durumu.</ResponseField>
