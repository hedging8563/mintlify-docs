## Genel Bakış

Bu uç nokta, seçilen API anahtarı için tüketime dayalı faturalandırma toplamlarını döndürür. Buna toplam maliyet, toplam istek sayısı, faturalandırma birimine göre kullanım ile istekte gönderdiğiniz model adı (`model`), sağlayıcı, `scene`, `accessChannel`, cache sınıfı ve günlük maliyet eğilimine göre kırılımlar dahildir.

## Notlar

- Yükleme, fatura ve kuruluş düzeyindeki diğer finansal kayıtlar dahil değildir.

## Örnek

<RequestExample>
```bash cURL
curl -X GET "https://api.aisonar.dev/v1/management/api-keys/key_abc123def456/billing?startDate=2026-01-01&endDate=2026-01-31" \
  -H "Authorization: Bearer mt-your-management-token"
```
</RequestExample>

## İstek / Yanıt

Kesin şema için yukarıdaki etkileşimli OpenAPI panelini kullanın.

## Yanıt örneği

<ResponseExample>
```json 200 OK
{
  "object": "management.billing",
  "api_key_id": "ak_abc123",
  "currency": "USD",
  "total_cost": "12.3400",
  "data": [{
    "billing_transaction_id": "billtxn_abc123",
    "request_id": "req_abc123",
    "amount": "0.0012",
    "created_at": "2026-06-27T00:00:00.000Z"
  }]
}
```
</ResponseExample>

## Önemli alanlar

<ResponseField name="total_cost" type="string">Billing mutabakatı için USD decimal string.</ResponseField>
<ResponseField name="data" type="array">Bu yanıtta dönen liste veya payload.</ResponseField>
<ResponseField name="billing_transaction_id" type="string">Varsa billing settlement identifier.</ResponseField>
