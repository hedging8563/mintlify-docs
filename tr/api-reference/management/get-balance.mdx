## Genel Bakış

Bu uç nokta, yönetim tokenıyla ilişkili kuruluşun geçerli bakiye toplamlarını döndürür. Faturalandırma ve mutabakatla aynı kuruluş bakiyesi doğruluk kaynağını kullanır.

## Notlar

- Parasal doğruluk alanları USD olarak döner: `balance`, `total_recharge` ve `total_used`.
- `preferred_currency`, kuruluş Dashboard görüntüleme tercihini yansıtır; USD doğruluk alanlarını değiştirmez.

## Örnek

<RequestExample>
```bash cURL
curl -X GET "https://api.aisonar.dev/v1/management/balance" \
  -H "Authorization: Bearer mt-your-management-token"
```
</RequestExample>
