## Yol Parametreleri

<ParamField path="id" type="string" required>
  İlk görüntü oluşturma isteğinden döndürülen görev kimliği (task ID).
</ParamField>

<Info>
Eğer oluşturma yanıtı `poll_url` içeriyorsa, yoklama (polling) için tam olarak bu URL'yi çağırmayı tercih edin. Bazı görüntü görevleri, görüntüye özel durum yolu yerine `/v1/tasks/{id}` altında bir `poll_url` sunabilir.

`ldtask_...` gibi herkese açık görsel görev ID’lerini `/v1/images/{id}` üzerinden de sorgulayabilirsiniz. Bu uyumluluk alias’ı `/v1/images/generations/{id}` ile aynı yetkilendirme, ownership kontrolü, faturalama başlıkları, terminal snapshot’ları ve yanıt şeklini kullanır.


Görev artık yoksa veya genel async-task sözleşmesi üzerinden artık çözümlenemiyorsa, AI Sonar `async_task_not_found` döndürür ve `Task not found or no longer available.` mesajını verir.
</Info>

## Yanıt

<ResponseField name="created" type="integer">
  Oluşturulma zamanının Unix zaman damgası.
</ResponseField>

<ResponseField name="task_id" type="string">
  Görev tanımlayıcı.
</ResponseField>

<ResponseField name="status" type="string">
  Görev durumu: `pending`, `processing`, `completed` veya `failed`.
</ResponseField>

<ResponseField name="data" type="array">
  Oluşturulan görüntülerin dizisi (`status` değeri `completed` olduğunda doldurulur).

  Her nesne şunları içerir:
  - `url` (string): Oluşturulan görüntünün URL'si
  - `revised_prompt` (string): Oluşturma için kullanılan istem (prompt)
</ResponseField>

<ResponseField name="error" type="string">
  Hata mesajı (yalnızca `status` değeri `failed` olduğunda mevcuttur).
</ResponseField>

## Yoklama (Polling) İçin En İyi Uygulamalar

<Tip>
**Önerilen yoklama aralığı**: 3-5 saniye. Çoğu görüntü oluşturma görevi, modele ve yönlendirilen sağlayıcı yoluna bağlı olarak 30-120 saniye içinde tamamlanır.
</Tip>

```python
import requests
import time

def poll_image_task(task_id, api_key, max_wait=300, interval=3):
    """Poll for image generation result with timeout."""
    url = f"https://api.aisonar.dev/v1/tasks/{task_id}"
    headers = {"Authorization": f"Bearer {api_key}"}

    start_time = time.time()
    while time.time() - start_time < max_wait:
        response = requests.get(url, headers=headers)
        data = response.json()

        if data["status"] == "completed":
            return data["data"][0]["url"]
        elif data["status"] == "failed":
            raise Exception(data.get("error", "Generation failed"))

        time.sleep(interval)

    raise TimeoutError(f"Task {task_id} did not complete within {max_wait}s")

# Usage
image_url = poll_image_task("ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa", "sk-your-api-key")
print(f"Generated image: {image_url}")
```
