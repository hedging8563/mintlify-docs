## Genel Bakış

`/v1/translations` endpoint'ini **metinden metne çeviri** için kullanın.

<Note>
Bu endpoint, ses dosyası alan ve her zaman İngilizce metin çıkaran [Ses Çevirisi](/tr/api-reference/audio/create-translation) endpoint'inden farklıdır.
</Note>

Agent akışları için önce güncel önerilen çeviri modellerini keşfedin:

```bash
curl "https://api.aisonar.dev/v1/models?recommended_for=translation" \
  -H "Authorization: Bearer sk-your-api-key"
```

Ardından seçilen modeli açıkça `/v1/translations` endpoint'ine gönderin.

<Note>
`recommended_for=translation` yalnızca **metin çevirisi** (`POST /v1/translations`) için geçerlidir. [ses çevirisine](/tr/api-reference/audio/create-translation) uygulanmaz.
</Note>

## İstek Gövdesi

<ParamField body="model" type="string" required>
  `/v1/models?recommended_for=translation` tarafından döndürülen çeviri model ID'si.
</ParamField>

<ParamField body="text" type="string" required>
  Çevrilecek kaynak metin.
</ParamField>

<ParamField body="target_language" type="string" required>
  Modele bağlı olarak hedef dil kodu veya dil adı.
</ParamField>

<ParamField body="source_language" type="string">
  İsteğe bağlı kaynak dil ipucu.
</ParamField>

<ParamField body="mime_type" type="string" default="text/plain">
  Kaynak metin formatı. Desteklenen değerler: `text/plain`, `text/html`.
</ParamField>

<ParamField body="user" type="string">
  Abuse izleme ve request takibi için isteğe bağlı son kullanıcı identifier'ı.
</ParamField>

## Yanıt

<ResponseField name="text" type="string">
  Çevrilmiş metin.
</ResponseField>

<ResponseField name="model" type="string">
  Çeviriyi üreten model.
</ResponseField>

<ResponseField name="source_language" type="string | null">
  Model tarafından tespit edilen veya kabul edilen kaynak dil.
</ResponseField>

<ResponseField name="target_language" type="string">
  Request için kullanılan hedef dil.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.aisonar.dev/v1/translations" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini-translation-pro",
    "text": "你好，欢迎使用 AI Sonar。",
    "target_language": "en"
  }'
```

```python Python
from openai import OpenAI
import requests

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.aisonar.dev/v1"
)

response = requests.post(
    "https://api.aisonar.dev/v1/translations",
    headers={
        "Authorization": "Bearer sk-your-api-key",
        "Content-Type": "application/json",
    },
    json={
        "model": "gemini-translation-pro",
        "text": "你好，欢迎使用 AI Sonar。",
        "target_language": "en",
    },
    timeout=30,
)

print(response.json()["text"])
```
</RequestExample>
