## Genel Bakış

Bu endpoint, yerel Anthropic Messages API uyumluluğu sağlar. Bunu, genişletilmiş düşünme gibi özelliklerle Claude modelleri için kullanın.


Bu endpoint yerel Anthropic sözleşmesini korur. `messages`, yalnızca `user` / `assistant` mesajlarından oluşan bir dizi olmalıdır; `system` üst seviye `system` alanına yazılır ve `max_tokens` zorunludur. Payload `messages` içinde OpenAI rolleri olan `system`, `developer` veya `tool` kullanıyorsa bunu `/v1/chat/completions` endpoint’ine gönderin.
<Note>
Anthropic SDK için Base URL: `https://api.aisonar.dev` (`/v1` soneki yok)
</Note>

## İstek Başlıkları

<ParamField header="x-api-key" type="string" required>
  AI Sonar API anahtarınız. Bearer token'a alternatif.
</ParamField>

<ParamField header="anthropic-version" type="string" required>
  Anthropic API sürümü. `2023-06-01` kullanın.
</ParamField>

## İstek Gövdesi

<ParamField body="model" type="string" required>
  Claude model ID'si (ör. `claude-sonnet-4-6` veya `claude-opus-4-6`).
</ParamField>

<ParamField body="messages" type="array" required>
  `role` ve `content` içeren mesaj nesneleri dizisi.
</ParamField>

<ParamField body="max_tokens" type="integer" required>
  Oluşturulacak maksimum token sayısı.
</ParamField>

<ParamField body="system" type="string">
  System prompt (`messages` dizisinden ayrı).
</ParamField>

<ParamField body="temperature" type="number" default="1">
  Örnekleme sıcaklığı (0-1).
</ParamField>

<ParamField body="stream" type="boolean" default="false">
  Streaming yanıtlarını etkinleştirir.
</ParamField>

<ParamField body="thinking" type="object">
  Genişletilmiş düşünme yapılandırması (Claude Opus 4.5).

  - `type` (string): Etkinleştirmek için `"enabled"`
  - `budget_tokens` (integer): Düşünme için token bütçesi
</ParamField>

<ParamField body="tools" type="array">
  Model için kullanılabilir araçlar.
</ParamField>

<ParamField body="tool_choice" type="object">
  Modelin araçları nasıl kullanması gerektiği. Seçenekler: `auto`, `any`, `tool` (belirli araç).
</ParamField>

<ParamField body="top_p" type="number">
  Nucleus sampling parametresi. Hem temperature hem de top_p birlikte değil, yalnızca biri kullanılmalıdır.
</ParamField>

<ParamField body="top_k" type="integer">
  Her token için yalnızca en iyi K seçenek arasından örnekleme yapar.
</ParamField>

<ParamField body="stop_sequences" type="array">
  Modelin üretimi durdurmasına neden olacak özel durdurma dizileri.
</ParamField>

<ParamField body="metadata" type="object">
  İzleme amaçları için isteğe eklenecek metadata.
</ParamField>

## Yanıt

<ResponseField name="id" type="string">
  Benzersiz mesaj tanımlayıcısı.
</ResponseField>

<ResponseField name="type" type="string">
  Her zaman `message`.
</ResponseField>

<ResponseField name="role" type="string">
  Her zaman `assistant`.
</ResponseField>

<ResponseField name="content" type="array">
  İçerik blokları dizisi (`text`, `thinking`, `tool_use`).
</ResponseField>

<ResponseField name="model" type="string">
  Kullanılan model.
</ResponseField>

<ResponseField name="stop_reason" type="string">
  Üretimin neden durduğu (`end_turn`, `max_tokens`, `tool_use`).
</ResponseField>

<ResponseField name="usage" type="object">
  `input_tokens` ve `output_tokens` ile token kullanımı.
</ResponseField>

## Genişletilmiş Düşünme Örneği

```python
message = client.messages.create(
    model="claude-opus-4-6",
    max_tokens=16000,
    thinking={
        "type": "enabled",
        "budget_tokens": 10000
    },
    messages=[{"role": "user", "content": "Solve this math problem..."}]
)

for block in message.content:
    if block.type == "thinking":
        print(f"Thinking: {block.thinking}")
    elif block.type == "text":
        print(f"Response: {block.text}")
```
## Anthropic Mesaj Toplu İşlemleri

AI Sonar artık `/v1/messages` yanında yerel Anthropic Message Batches akışını da sunuyor.

Kullanılabilir rotalar:

- `POST /v1/messages/batches`
- `GET /v1/messages/batches`
- `GET /v1/messages/batches/:message_batch_id`
- `GET /v1/messages/batches/:message_batch_id/results`
- `POST /v1/messages/batches/:message_batch_id/cancel`
- `DELETE /v1/messages/batches/:message_batch_id`

Operasyon notları:

- Aynı AI Sonar API anahtarını ve Anthropic yerel başlıklarını kullanın.
- Batch öğeleri `file_id` referansı içeriyorsa `anthropic-beta: files-api-2025-04-14` başlığını da ekleyin.
- Batch işlerinin istek/yanıt biçimi Anthropic yerel formatında kalır; AI Sonar ise iç settlement yaşam döngüsünü takip eder.
