## Yol Parametreleri

<ParamField path="model" type="string" required>
  Getirilecek modelin ID'si (örneğin `gpt-5.4`, `claude-sonnet-4-6`).
</ParamField>

## Yanıt

<ResponseField name="id" type="string">
  Model tanımlayıcısı.
</ResponseField>

<ResponseField name="object" type="string">
  Her zaman `model`.
</ResponseField>

<ResponseField name="created" type="integer">
  Oluşturulma zaman damgası.
</ResponseField>

<ResponseField name="owned_by" type="string">
  Model sağlayıcısı.
</ResponseField>

<ResponseField name="aisonar" type="object">
  AI Sonar'ya özgü herkese açık metadata. Detay route'u `category`, `pricing_unit`, `capability_flags`, `supported_operations`, `pricing_provenance`, `public_contract_summary`, tam `public_contract` ve chat dışı öneri snapshot'ları için sahne düzeyinde `agent_preferences` içerebilir.
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.aisonar.dev/v1/models/gpt-5.4" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.aisonar.dev/v1"
)

model = client.models.retrieve("gpt-5.4")
print(f"Model: {model.id}, Provider: {model.owned_by}")
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.aisonar.dev/v1'
});

const model = await client.models.retrieve('gpt-5.4');
console.log(`Model: ${model.id}, Provider: ${model.owned_by}`);
```

```go Go
package main

import (
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    req, _ := http.NewRequest("GET", "https://api.aisonar.dev/v1/models/gpt-5.4", nil)
    req.Header.Set("Authorization", "Bearer sk-your-api-key")

    resp, _ := http.DefaultClient.Do(req)
    defer resp.Body.Close()

    var model map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&model)
    fmt.Printf("Model: %s, Provider: %s\n", model["id"], model["owned_by"])
}
```

## Hata İşleme

Model yoksa 404 hatası alırsınız:

```json
{
  "error": {
    "message": "Model 'invalid-model' not found",
    "type": "invalid_request_error",
    "code": "model_not_found"
  }
}
```

## Chat Dışı Öneri Metadata'sı

Chat dışı modeller için detay route'u ayrıca `aisonar.agent_preferences.<scene>` döndürür.

Bunu şunları incelemek için kullanın:

- `preferred_rank`
- `success_rate_24h`
- `sample_count_24h`
- `status`
- `updated_at`

Herkese açık yanıt, ajanların öneri sinyalini iç yönlendirme tanımlayıcılarını öğrenmeden tüketebilmesi için `basis.channel_id` ve `basis.physical_model` alanlarını kasıtlı olarak `null` bırakır.

## Herkese Açık Detay Metadata'sı

`GET /v1/models` ile karşılaştırıldığında, detay route'u ayrıca şunları döndürebilir:

- `aisonar.capability_flags`
- `aisonar.supported_operations`
- `aisonar.pricing_provenance`
- `aisonar.public_contract_summary`
- `aisonar.public_contract`
