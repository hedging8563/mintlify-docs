## Yanıt

<ResponseField name="object" type="string">
  Her zaman `list`.
</ResponseField>

<ResponseField name="data" type="array">
  Model nesneleri dizisi.

  Her model şunları içerir:
  - `id` (string): Model tanımlayıcısı
  - `object` (string): `model`
  - `created` (integer): Oluşturulma zaman damgası
  - `owned_by` (string): Model sağlayıcısı
  - `tokenlab.category` (string): Genel model kategorisi
  - `tokenlab.pricing_unit` (string veya `null`): Varsa genel fiyatlandırma birimi
  - `tokenlab.capability_flags` (object veya `null`): Genel yetenek özeti
  - `tokenlab.supported_operations` (array veya `null`): Model için genel chat dışı işlemler
  - `tokenlab.pricing_provenance` (object veya `null`): Genel fiyat kaynağı metadata'sı
  - `tokenlab.request_format_summary` (object veya `null`): Yalnızca `public_operations` ve `request_endpoint` içeren hafif chat dışı discovery özeti
  - `tokenlab.agent_preferences` (object): Yalnızca `recommended_for` mevcut olduğunda döner
  - `tokenlab.lifecycle` (object): yaşam döngüsü aşaması, yayın/kullanımdan kaldırma tarihleri, yerine geçecek model ve latest rozeti kaynağı
  - `tokenlab.commercial` (object): kullanıcı ücret politikası, ücretsiz nedeni ve isteğe bağlı ücretsiz bitiş zamanı
  - `tokenlab.badges` (array): lifecycle ve commercial meta verilerinden türetilen rozetler
</ResponseField>

<Note>
`GET /v1/models` keşif için optimize edilmiştir. `capability_flags`, `pricing_provenance` ve tam `request_format_details` gibi yalnızca detay sayfasına ait metadata `GET /v1/models/{model}` üzerinde bulunur.
</Note>

## Query Parametreleri

<ParamField query="category" type="string">
  İsteğe bağlı genel kategori filtresi. `chat`, `image`, `video`, `audio`, `tts`, `stt`, `music`, `3d`, `embedding`, `rerank` ve `translation` desteklenir.
</ParamField>

<ParamField query="recommended_for" type="string">
  İsteğe bağlı chat dışı öneri senaryosu. `image`, `video`, `music`, `3d`, `tts`, `stt`, `embedding`, `rerank` ve `translation` desteklenir.
</ParamField>

<ParamField query="provider" type="string">
  `openai`, `anthropic`, `google` veya `deepseek` gibi isteğe bağlı provider filtresi.
</ParamField>

<ParamField query="tag" type="string">
  `chat`, `image`, `video`, `embedding` veya `translation` gibi isteğe bağlı model tag filtresi.
</ParamField>

<Note>
`recommended_for` mevcut olduğunda, `/v1/models` chat dışı modelleri en son cache'lenmiş 24 saatlik başarı oranı snapshot'una göre sıralar. `status = "insufficient_samples"` olan modeller görünür kalır ancak puanlanmış modellerin arkasına yerleşir.
</Note>

<RequestExample>
```bash cURL
curl "https://api.aisonar.dev/v1/models" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.aisonar.dev/v1"
)

models = client.models.list()

for model in models.data:
    print(f"{model.id} ({model.owned_by})")
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.aisonar.dev/v1'
});

const models = await client.models.list();

for (const model of models.data) {
  console.log(`${model.id} (${model.owned_by})`);
}
```

```go Go
package main

import (
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    req, _ := http.NewRequest("GET", "https://api.aisonar.dev/v1/models", nil)
    req.Header.Set("Authorization", "Bearer sk-your-api-key")

    resp, _ := http.DefaultClient.Do(req)
    defer resp.Body.Close()

    var payload struct {
        Data []struct {
            ID      string `json:"id"`
            OwnedBy string `json:"owned_by"`
        } `json:"data"`
    }

    json.NewDecoder(resp.Body).Decode(&payload)

    for _, model := range payload.Data {
        fmt.Printf("%s (%s)\n", model.ID, model.OwnedBy)
    }
}
```

## Sağlayıcıya Göre Filtreleme

```python
# Get all OpenAI models
openai_models = [m for m in models.data if m.owned_by == "openai"]

# Get all Anthropic models
anthropic_models = [m for m in models.data if m.owned_by == "anthropic"]
```
