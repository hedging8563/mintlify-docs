## Coding Agents Neden Pahalıdır

Claude Code, Cursor ve Codex CLI gibi coding agents çok güçlüdür — ancak çoğu ekibi şaşırtan bir hızda token tüketirler.

**İstek başına uzun bağlam.** Her çağrı bir sistem istemi, proje dosyaları ve konuşma geçmişi içerir — model düşünmeye başlamadan önce genellikle 10.000–100.000+ token.

**Yüksek frekanslı çağrılar.** Tek bir coding oturumu düzinelerce API çağrısını tetikler: kod üretimi, arama, inceleme, otomatik tamamlama ve araç kullanımı. 1 saatlik bir oturum kolayca 200+ isteğe ulaşabilir.

**Konuşma birikimi.** Her tur tam mesaj geçmişini yeniden gönderir. 20. turda, aynı bağlam için 20 kez ödeme yapıyorsunuz.

<Info>
Tipik bir 1 saatlik Claude Code oturumu 2–5 milyon token tüketebilir. Doğrudan API fiyatlarıyla bu, modele bağlı olarak saatte $6–30+ demektir.
</Info>
