## Genel bakış

OpenAI, Anthropic ve Gemini uyumlu rotalarda JSON Schema, tool calls ve native format özelliklerini kullanın.

AI Sonar public API’leri ve model sözleşmeleri kullanın; provider execution detaylarına bağlı kalmayın.

## Temel kurallar

- Kullanıcıya başarı dönmeden önce schema version, tool name, validated arguments bilgisini saklayın.
- AI Sonar public API’leri ve model sözleşmeleri kullanın; provider execution detaylarına bağlı kalmayın.
- Retry işlemlerini idempotent yapın ve çift generation veya çift billing önleyin.
