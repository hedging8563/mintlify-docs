## Genel Bakış



<Note>
**Tür**: Kodlama aracı

**Birincil yol**: Gemini-native

**Destek seviyesi**: En iyi çaba uyumluluk
</Note>

Google Gemini CLI, Gemini modelleriyle etkileşim kurmak için kullanılan bir komut satırı aracıdır. AI Sonar, Gemini CLI'yı 300'den fazla modele erişimle kullanmanıza olanak tanıyan uyumlu bir uç nokta (endpoint) sağlar.


<Warning>
Bu entegrasyonu deneysel kabul edin: Gemini CLI şu anda AI Sonar benzeri özel Base URL iş akışını kararlı bir resmi yapılandırma olarak belgelemiyor. Desteklenen bir AI Sonar terminal kurulumu istiyorsanız Codex CLI, Claude Code veya OpenCode kullanın.
</Warning>

## Sistem Gereksinimleri

- **Node.js**: Sürüm 20.0+
- **OS**: Windows 10/11, macOS 10.15+, Ubuntu 20.04+ veya Debian 10+

## Kurulum

```bash
npm install -g @google/gemini-cli
```

Kurulumu doğrulayın:

```bash
gemini --version
```

## Yapılandırma

### Adım 1: API Anahtarınızı Alın

1. [AI Sonar Dashboard](https://console.aisonar.dev) sayfasına giriş yapın
2. [API Keys](https://console.aisonar.dev/api) bölümüne gidin
3. API anahtarınızı oluşturun ve kopyalayın (format: `sk-...`)

### Adım 2: Ortam Değişkenlerini Ayarlayın

**Geçici (mevcut oturum):**

```bash
export GEMINI_API_KEY="sk-your-api-key"
export GOOGLE_GEMINI_BASE_URL="https://api.aisonar.dev"
```

**Kalıcı yapılandırma:**

Kabuk (shell) yapılandırma dosyanıza ekleyin:

<Tabs>
  <Tab title="Bash">
    ```bash
    echo 'export GEMINI_API_KEY="sk-your-api-key"' >> ~/.bashrc
    echo 'export GOOGLE_GEMINI_BASE_URL="https://api.aisonar.dev"' >> ~/.bashrc
    source ~/.bashrc
    ```
  </Tab>
  <Tab title="Zsh">
    ```bash
    echo 'export GEMINI_API_KEY="sk-your-api-key"' >> ~/.zshrc
    echo 'export GOOGLE_GEMINI_BASE_URL="https://api.aisonar.dev"' >> ~/.zshrc
    source ~/.zshrc
    ```
  </Tab>
  <Tab title="Fish">
    ```bash
    set -Ux GEMINI_API_KEY "sk-your-api-key"
    set -Ux GOOGLE_GEMINI_BASE_URL "https://api.aisonar.dev"
    ```
  </Tab>
</Tabs>

<Warning>
Değişikliklerin etkili olması için yapılandırmadan sonra terminalinizi yeniden başlatın.
</Warning>

<Note>
**Bilinen kısıtlama**: Bazı durumlarda, önbelleğe alınmış bir Google oturumu varsa Gemini CLI, `GOOGLE_GEMINI_BASE_URL` değişkenini dikkate almayabilir. Bağlantı sorunları yaşıyorsanız, CLI önbelleğini temizlemeyi veya yeni bir terminal oturumu başlatmayı deneyin.
</Note>

## Temel Kullanım

Gemini CLI'yı proje dizininizden başlatın:

```bash
cd your-project
gemini
```

İlk çalıştırmada şunları yapacaksınız:
1. Bir tema seçin
2. Güvenlik bildirimini onaylayın
3. Çalışma dizinine güvenin
