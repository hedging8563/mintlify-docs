## Skill ne yapar

- AI Sonar chat, image, audio, video, translation ve diğer API aileleri için minimum çalıştırılabilir örnekler üretir.
- OpenAI uyumlu client’lar için `https://api.aisonar.dev/v1` kullanır ve native Anthropic veya Gemini route’larına ne zaman geçileceğini açıklar.
- Eskimiş listelerden tahmin etmek yerine `/v1/models`, `/llms.txt` ve `recommended_for` kısa listeleriyle modelleri keşfeder.
- Non-chat istekleri tekrar denemeden önce modelin public contract bilgisini okur; desteklenmeyen alanlar sessizce atılmaz.
- `did_you_mean`, `suggestions`, `retry_after`, `recommended_request` gibi Agent-First hata ipuçlarını işler.

## Önerilen Agent iş akışı

1. Kullanıcının istediği dil ve API ailesiyle eşleşen en küçük çalışan örnekle başlayın.
2. Model seçimi açıksa bir modeli hard-code etmeden önce `/v1/models` veya `https://api.aisonar.dev/llms.txt` çağırın.
3. Non-chat işlerde `/v1/models?recommended_for=<scene>` çağırın; `<scene>` `image`, `video`, `music`, `3d`, `tts`, `stt`, `embedding`, `rerank` veya `translation` olabilir.
4. Başarısız non-chat isteği değiştirmeden önce `/v1/models/:model` okuyun ve `supported_operations`, `supported_parameters`, `request_endpoint`, `request_shape_mode`, `recommended_request` ile hizalayın.
5. API Agent-First hata döndürürse isteği yapılandırılmış alanlarla düzeltin ve yalnızca hata retryable ise yeniden deneyin.

## Minimum chat örneği

Bu örnek OpenAI Python SDK ile AI Sonar base URL kullanır:

```python
import os
from openai import OpenAI

client = OpenAI(
    api_key=os.environ["TOKENLAB_API_KEY"],
    base_url="https://api.aisonar.dev/v1"
)

response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)
```

Şununla çalıştırın:

```bash
pip install openai
export TOKENLAB_API_KEY="sk-your-api-key"
python app.py
```

## Model keşfi ve public contract’lar

Eskimiş paket listeleri yerine canlı keşfi tercih edin:

```bash
# Makine tarafından okunabilir API özeti
curl https://api.aisonar.dev/llms.txt

# Modelleri listele
curl "https://api.aisonar.dev/v1/models" -H "Authorization: Bearer sk-KEY"
curl "https://api.aisonar.dev/v1/models?category=image" -H "Authorization: Bearer sk-KEY"

# Sıralı non-chat kısa listeleri
curl "https://api.aisonar.dev/v1/models?recommended_for=image" -H "Authorization: Bearer sk-KEY"
curl "https://api.aisonar.dev/v1/models?recommended_for=video" -H "Authorization: Bearer sk-KEY"
curl "https://api.aisonar.dev/v1/models?recommended_for=translation" -H "Authorization: Bearer sk-KEY"

# Non-chat isteği tekrar denemeden önce tek modelin public contract bilgisini oku
curl "https://api.aisonar.dev/v1/models/gpt-image-2" -H "Authorization: Bearer sk-KEY"

# Yalnızca fiyat ayrıntısı
curl "https://api.aisonar.dev/v1/models/gpt-image-2/pricing" -H "Authorization: Bearer sk-KEY"
```

## Native routing ipuçları

Yaygın chat, Responses, image, embedding, audio ve rerank örnekleri için OpenAI uyumlu `/v1` varsayılan yoldur. Native Anthropic veya Gemini route’larını yalnızca istek sağlayıcıya özel davranış gerektirdiğinde ya da AI Sonar the native endpoint hint header optimizasyon ipucu döndürdüğünde kullanın.



## Agent-First hata toparlama

Hatalar, coding agent’ların dokümanı kazımadan ayrıştırabileceği alanlar içerir:

| Hata | Yararlı alanlar | Agent eylemi |
|---|---|---|
| Yanlış model adı | `did_you_mean`, `suggestions`, `hint` | Düzeltilmiş modelle veya önerilen bir alternatifle yeniden deneyin |
| Yetersiz bakiye | `balance_usd`, `estimated_cost_usd`, `suggestions` | Onay isteyin veya uygun maliyetli bir modele geçin |
| Hız sınırı veya geçici kullanılamama | `retryable`, `retry_after`, `alternatives` | Bekleyin, yeniden deneyin veya listelenen bir alternatifi seçin |
| Sohbet dışı contract uyuşmazlığı | `supported_operations`, `supported_parameters`, `request_endpoint`, `request_shape_mode`, `recommended_request` | İsteği public contract üzerinden yeniden kurun; niyeti değiştiren alanlarda fail closed uygulayın |

## Desteklenen API aileleri

| Aile | Birincil yol |
|---|---|
| Chat ve Responses | `/v1/chat/completions`, `/v1/responses` |
| Claude-native messages | `/v1/messages` |
| Gemini-native istekler | `/v1beta/models/{model}:generateContent` |
| Görseller | `/v1/images/generations`, `/v1/images/edits` |
| Video | `/v1/videos/generations` |
| Müzik | `/v1/music/generations` |
| Worlds | `/v1/worlds/generations` ve world status ile media asset endpoint’leri |
| 3D | `/v1/3d/generations` |
| Ses | `/v1/audio/speech`, `/v1/audio/transcriptions`, `/v1/audio/translations` |
| Realtime | `/v1/realtime?model={model}` |
| Embeddings ve rerank | `/v1/embeddings`, `/v1/rerank` |
| Metin çevirisi | `/v1/translations` |
