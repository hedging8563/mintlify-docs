## Genel Bakış



<Note>
**Tür**: Kodlama aracı

**Birincil yol**: OpenAI ile uyumlu

**Destek seviyesi**: Sınırlı olarak desteklenir
</Note>

[Cursor](https://cursor.sh), bring-your-own-key (BYOK) sağlayıcılarını destekler, ancak özel API anahtarları yalnızca standart chat modelleriyle çalışır. AI Sonar'yı burada, chat tarzı editör iş akışları için OpenAI uyumlu bir uç nokta olarak kullanın; Cursor'daki her model özelliğinin tam bir yerine geçecek çözüm olarak değil.

## Yapılandırma

1. Cursor Ayarları'nı açın (`Cmd/Ctrl + ,`)
2. Sol kenar çubuğunda **Models** bölümüne gidin
3. **OpenAI API Key** bölümüne kadar aşağı kaydırın
4. AI Sonar API anahtarınızı girin
5. **Override OpenAI Base URL** seçeneğini etkinleştirin
6. Base URL'yi şu şekilde ayarlayın:

```
https://api.aisonar.dev/v1
```

<Warning>
Cursor özel API anahtarları yalnızca standart chat modelleri için geçerlidir. Tab Completion ve Cursor tarafından yönetilen diğer özelleşmiş özellikler, Cursor'un yerleşik sağlayıcılarını kullanmaya devam edecektir.
</Warning>

<Warning>
Cursor’un özel API anahtarı akışı yalnızca standart chat modelleri için geçerlidir. Tab Completion ve bazı akıl yürütme odaklı akışlar gibi Cursor tarafından yönetilen özelleşmiş modeller gerektiren özellikler, Cursor’un yerleşik sağlayıcılarını kullanmaya devam edecektir. Cursor ayrıca, isteklerin son prompt oluşturma işlemi için yine de kendi backend'i üzerinden geçtiğini belirtir.
</Warning>

## Model Seçimi

Yapılandırmadan sonra, AI Sonar üzerinden sunulan standart chat modellerini tercih edin:

### Kodlama için Önerilen Modeller

| Model | En Uygun Olduğu Alan |
|-------|----------|
| `gpt-5.4` | Genel kodlama, debugging, chat |
| `gpt-5-mini` | Hızlı düzenlemeler ve hafif prompt'lar |

## En İyi Çalışanlar

### Chat ve editör akışları

Cursor üzerinden AI Sonar'yı şunlar için kullanın:

- Kod açıklamaları
- Hata düzeltme
- Refactoring önerileri
- Dokümantasyon oluşturma

### Cursor tarafından yönetilen modellerde kalanlar

Bu özellikler AI Sonar endpoint'inize güvenilir şekilde geçmez:

- Sekme Tamamlama
- Özelleşmiş standart dışı chat modelleri
- Cursor'un yalnızca yerleşik modellerde etkinleştirdiği sağlayıcıya özgü özellikler
