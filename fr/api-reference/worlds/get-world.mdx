---
title: "Récupérer un World"
openapi: "GET /v1/worlds/{id}"
description: "Récupère un World World Labs Marble terminé"
---

Récupérez un Marble World terminé créé via AI Sonar. Le world doit appartenir à la même organisation que la clé API.

## Paramètres de chemin

<ParamField path="id" type="string" required>
  ID World Labs world.
</ParamField>

## Réponse

<ResponseField name="world_id" type="string">
  ID World Labs world.
</ResponseField>

<ResponseField name="world_marble_url" type="string">
  World Labs Marble URL.
</ResponseField>

<ResponseField name="assets" type="object">
  Assets World générés lorsqu’ils sont disponibles.
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.aisonar.dev/v1/worlds/world_123" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
import requests

response = requests.get(
    "https://api.aisonar.dev/v1/worlds/world_123",
    headers={"Authorization": "Bearer sk-your-api-key"},
)
print(response.json()["world_id"])
```

```javascript JavaScript
const response = await fetch('https://api.aisonar.dev/v1/worlds/world_123', {
  headers: { Authorization: 'Bearer sk-your-api-key' }
});
console.log(await response.json());
```
</RequestExample>
