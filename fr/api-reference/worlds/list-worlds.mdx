---
title: "Lister les Worlds"
openapi: "POST /v1/worlds/list"
description: "Liste les Worlds World Labs Marble terminés créés via AI Sonar"
---

Listez les Worlds World Labs Marble terminés créés via AI Sonar. Les résultats sont limités à votre organisation; AI Sonar n’expose pas la liste globale du compte upstream partagé.

Utilisez `/v1/worlds/generations/{id}` pour les tâches en attente ou échouées. Cette liste renvoie uniquement les Worlds terminés que AI Sonar a indexés après génération.

## Corps de requête

<ParamField body="page_size" type="integer" default="20">
  Nombre de worlds à renvoyer, de 1 à 100.
</ParamField>

<ParamField body="page_token" type="string">
  Token de pagination de la réponse précédente.
</ParamField>

<ParamField body="status" type="string">
  Filtre de statut optionnel. AI Sonar renvoie les worlds terminés pour `SUCCEEDED`; les autres statuts relèvent de l’endpoint de tâche de génération.
</ParamField>

<ParamField body="model" type="string">
  Filtre optionnel sur le modèle Marble.
</ParamField>

<ParamField body="tags" type="array">
  Filtre optionnel sur les tags.
</ParamField>

<ParamField body="is_public" type="boolean">
  Filtre optionnel de visibilité.
</ParamField>

<ParamField body="created_after" type="string">
  Ne renvoyer que les worlds créés après cet horodatage.
</ParamField>

<ParamField body="created_before" type="string">
  Ne renvoyer que les worlds créés avant cet horodatage.
</ParamField>

<ParamField body="sort_by" type="string" default="created_at">
  Trier par `created_at` ou `updated_at`.
</ParamField>

<ParamField body="sort_order" type="string" default="desc">
  Ordre de tri: `asc` ou `desc`.
</ParamField>

## Réponse

<ResponseField name="worlds" type="array">
  Tableau d’enregistrements world.
</ResponseField>

<ResponseField name="next_page_token" type="string">
  Token de page suivante, ou `null`.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.aisonar.dev/v1/worlds/list" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{"page_size": 20, "status": "SUCCEEDED", "model": "marble-1.1"}'
```

```python Python
import requests

response = requests.post(
    "https://api.aisonar.dev/v1/worlds/list",
    headers={"Authorization": "Bearer sk-your-api-key"},
    json={"page_size": 20, "status": "SUCCEEDED", "model": "marble-1.1"},
)
print(response.json()["worlds"])
```
