---
title: "Obtenir le statut du World"
openapi: "GET /v1/worlds/generations/{id}"
description: "Récupère l’état et le résultat d’une tâche de génération de monde World Labs Marble"
---

Interrogez cet endpoint pour vérifier l’état d’une tâche de génération World Labs Marble. Si la réponse de création incluait `poll_url`, appelez de préférence cette URL exacte.

## Paramètres de chemin

<ParamField path="id" type="string" required>
  ID de la tâche de génération de monde.
</ParamField>

## Réponse

<ResponseField name="id" type="string">
  ID public de tâche.
</ResponseField>

<ResponseField name="task_id" type="string">
  Alias de l’identifiant de tâche asynchrone.
</ResponseField>

<ResponseField name="operation_id" type="string">
  ID d’operation World Labs upstream.
</ResponseField>

<ResponseField name="status" type="string">
  État de la tâche : `pending`, `processing`, `completed`, `failed` ou `timeout`.
</ResponseField>

<ResponseField name="world_id" type="string">
  ID World Labs world upstream une fois terminé.
</ResponseField>

<ResponseField name="world_marble_url" type="string">
  URL du monde Marble généré une fois terminé.
</ResponseField>

<ResponseField name="model_url" type="string">
  Alias de l’URL principale du monde/modèle généré.
</ResponseField>

<ResponseField name="glb_url" type="string">
  URL GLB du maillage collider si disponible.
</ResponseField>

<ResponseField name="pano_url" type="string">
  URL de l’image panoramique si disponible.
</ResponseField>

<ResponseField name="thumbnail_url" type="string">
  URL de miniature si disponible.
</ResponseField>

<ResponseField name="caption" type="string">
  Caption upstream du monde généré.
</ResponseField>

<ResponseField name="cost" type="object">
  Détails de coût de l’operation upstream si disponibles, y compris `total_credits`.
</ResponseField>

<ResponseField name="error" type="string">
  Message d’erreur si la tâche a échoué.
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.aisonar.dev/v1/worlds/generations/ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
import requests
import time

task_id = "ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa"

while True:
    response = requests.get(
        f"https://api.aisonar.dev/v1/worlds/generations/{task_id}",
        headers={"Authorization": "Bearer sk-your-api-key"},
    )
    result = response.json()
    if result["status"] in ["completed", "failed", "timeout"]:
        print(result)
        break
    time.sleep(10)
```

```javascript JavaScript

  headers: { Authorization: 'Bearer sk-your-api-key' }
```
