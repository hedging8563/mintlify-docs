---
title: "Supprimer un World"
openapi: "DELETE /v1/worlds/{id}"
description: "Supprime un World World Labs Marble"
---

Supprimez un Marble World créé via AI Sonar. AI Sonar transmet la suppression à World Labs et marque l’enregistrement local comme supprimé.

La suppression est permanente.

## Paramètres de chemin

<ParamField path="id" type="string" required>
  ID World Labs world.
</ParamField>

## Réponse

<ResponseField name="deleted" type="boolean">
  Indique si la suppression a été acceptée.
</ResponseField>

<ResponseField name="world_id" type="string">
  ID World Labs world.
</ResponseField>

<RequestExample>
```bash cURL
curl -X DELETE "https://api.aisonar.dev/v1/worlds/world_123" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
import requests

response = requests.delete(
    "https://api.aisonar.dev/v1/worlds/world_123",
    headers={"Authorization": "Bearer sk-your-api-key"},
)
print(response.json())
```

```javascript JavaScript
const response = await fetch('https://api.aisonar.dev/v1/worlds/world_123', {
  method: 'DELETE',
  headers: { Authorization: 'Bearer sk-your-api-key' }
});
console.log(await response.json());
```
</RequestExample>
