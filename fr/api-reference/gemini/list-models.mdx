---
title: "Lister les modèles"
openapi: "GET /v1beta/models"
description: "Lister les modèles disponibles en utilisant le format de l'API Google Gemini"
---

Renvoie une liste des modèles disponibles au format de l’API Google Gemini.

## Paramètres de requête

<ParamField query="pageSize" type="integer">
  Nombre maximum de modèles à retourner. Par défaut : `1000`, maximum : `1000`.
</ParamField>

<ParamField query="pageToken" type="string">
  Jeton de continuation renvoyé par une page précédente. Renvoyez-le tel quel pour récupérer le segment suivant.
</ParamField>

## Authentification

L’authentification n’est pas requise pour les points de lecture du catalogue public de modèles.

Les indications d’authentification de type Gemini comme `?key=` ou `x-goog-api-key` ne modifient pas le contrat de réponse ici.

## Réponse

<ResponseField name="models" type="array">
  Tableau d'objets de modèle.
</ResponseField>

<ResponseField name="nextPageToken" type="string">
  Renvoie une valeur lorsqu’il reste d’autres modèles. Passez cette valeur comme `pageToken` pour continuer la pagination.
</ResponseField>

## Erreurs

- Un `pageToken` invalide renvoie `400 INVALID_ARGUMENT`.
- Les échecs inattendus de lecture renvoient `500 INTERNAL`.

<RequestExample>
```bash cURL
curl "https://api.aisonar.dev/v1beta/models?pageSize=5" \
  -H "x-goog-api-key: sk-your-api-key"
```

```python Python
import google.generativeai as genai

genai.configure(
    api_key="sk-your-api-key",
    transport="rest",
    client_options={"api_endpoint": "api.aisonar.dev"}
)

for model in genai.list_models():
    print(model.name)
```

```javascript JavaScript
const response = await fetch(
  "https://api.aisonar.dev/v1beta/models?pageSize=5",
  { headers: { "x-goog-api-key": "sk-your-api-key" } }
);
const { models } = await response.json();
models.forEach(m => console.log(m.name));
```
</RequestExample>
