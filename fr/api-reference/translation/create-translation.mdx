---

## Aperçu

`/v1/translations` sert à la **traduction texte vers texte**.

<Note>
Cet endpoint est différent de [la traduction audio](/api-reference/audio/create-translation), qui accepte un fichier audio et renvoie toujours du texte en anglais.
</Note>

Pour les workflows d'agent, commencez par découvrir les modèles de traduction actuellement recommandés :

```bash
curl "https://api.aisonar.dev/v1/models?recommended_for=translation" \
  -H "Authorization: Bearer sk-your-api-key"
```

Envoyez ensuite explicitement le modèle choisi à `/v1/translations`.

<Note>
`recommended_for=translation` s'applique uniquement à la **traduction de texte** (`POST /v1/translations`). Il ne s'applique pas à [la traduction audio](/api-reference/audio/create-translation).
</Note>

## Corps de la requête

<ParamField body="model" type="string" required>
  L'ID du modèle de traduction renvoyé par `/v1/models?recommended_for=translation`.
</ParamField>

<ParamField body="text" type="string" required>
  Le texte source à traduire.
</ParamField>

<ParamField body="target_language" type="string" required>
  Code de langue ou nom de langue cible, selon le support du modèle.
</ParamField>

<ParamField body="source_language" type="string">
  Indication facultative de la langue source.
</ParamField>

<ParamField body="mime_type" type="string" default="text/plain">
  Format du texte source. Valeurs prises en charge : `text/plain`, `text/html`.
</ParamField>

<ParamField body="user" type="string">
  Identifiant facultatif de l'utilisateur final pour la surveillance des abus et le suivi des requêtes.
</ParamField>

## Réponse

<ResponseField name="text" type="string">
  Texte traduit.
</ResponseField>

<ResponseField name="model" type="string">
  Modèle qui a généré la traduction.
</ResponseField>

<ResponseField name="source_language" type="string | null">
  Langue source détectée ou acceptée par le modèle.
</ResponseField>

<ResponseField name="target_language" type="string">
  Langue cible utilisée pour la requête.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.aisonar.dev/v1/translations" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini-translation-pro",
    "text": "Bonjour, bienvenue sur AI Sonar.",
    "target_language": "en"
  }'
```

```python Python
from openai import OpenAI
import requests

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.aisonar.dev/v1"
)

response = requests.post(
    "https://api.aisonar.dev/v1/translations",
    headers={
        "Authorization": "Bearer sk-your-api-key",
        "Content-Type": "application/json",
    },
    json={
        "model": "gemini-translation-pro",
        "text": "Bonjour, bienvenue sur AI Sonar.",
        "target_language": "en",
    },
    timeout=30,
)

print(response.json()["text"])
```
</RequestExample>
