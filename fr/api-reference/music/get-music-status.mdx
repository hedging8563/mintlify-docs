---

<span data-mintlify-rebuild="2026-05-19-after-mdx-parse-fix" aria-hidden="true" />

Interrogez cet endpoint pour vérifier le statut de votre tâche de génération de musique.

<Info>
Si la réponse de création contient `poll_url`, privilégiez cette URL exacte pour le polling. `id` et `task_id` doivent être traités comme le même identifiant de tâche asynchrone.


Si la tâche n'existe plus ou ne peut plus être résolue via le contrat async-task public, AI Sonar renvoie `async_task_not_found` avec le message `Task not found or no longer available.`
</Info>

## Paramètres de chemin

<ParamField path="id" type="string" required>
  L'ID de la tâche de génération de musique.
</ParamField>

## Réponse

<ResponseField name="id" type="string">
  ID de la tâche.
</ResponseField>

<ResponseField name="task_id" type="string">
  Alias de l'identifiant de tâche asynchrone lorsqu'il est disponible.
</ResponseField>

<ResponseField name="status" type="string">
  Statut de la tâche : `pending`, `processing`, `completed` ou `failed`.
</ResponseField>

<ResponseField name="progress" type="integer">
  Pourcentage de progression (0-100).
</ResponseField>

<ResponseField name="audio_url" type="string">
  URL pour télécharger le fichier audio (une fois terminé).
</ResponseField>

<ResponseField name="video_url" type="string">
  URL vers la version vidéo avec visualiseur (une fois terminé).
</ResponseField>

<ResponseField name="title" type="string">
  Titre de la chanson générée.
</ResponseField>

<ResponseField name="lyrics" type="string">
  Paroles générées ou fournies.
</ResponseField>

<ResponseField name="error" type="string">
  Message d'erreur (en cas d'échec).
</ResponseField>

```python Python
import requests
import time

task_id = "ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa"

while True:
    response = requests.get(
        f"https://api.aisonar.dev/v1/tasks/{task_id}",
        headers={"Authorization": "Bearer sk-your-api-key"}
    )
    result = response.json()

    if result["status"] == "completed":
        print(f"Audio URL: {result['audio_url']}")
        break
    elif result["status"] == "failed":
        print(f"Error: {result['error']}")
        break

    time.sleep(5)  # Poll every 5 seconds
```

```javascript JavaScript
const taskId = 'ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa';

async function pollForMusic() {
  while (true) {
    const response = await fetch(
      `https://api.aisonar.dev/v1/tasks/${taskId}`,
      { headers: { 'Authorization': 'Bearer sk-your-api-key' } }
    );

    const result = await response.json();

    if (result.status === 'completed') {
      console.log(`Audio URL: ${result.audio_url}`);
      return result.audio_url;
    } else if (result.status === 'failed') {
      throw new Error(result.error);
    }

    await new Promise(r => setTimeout(r, 5000));
  }
}
```

```go Go
package main

import (
    "encoding/json"
    "fmt"
    "net/http"
    "time"
)

func main() {
    taskID := "ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa"

    for {
        req, _ := http.NewRequest("GET",
            fmt.Sprintf("https://api.aisonar.dev/v1/tasks/%s", taskID), nil)
        req.Header.Set("Authorization", "Bearer sk-your-api-key")

        client := &http.Client{}
        resp, _ := client.Do(req)

        var result map[string]interface{}
        json.NewDecoder(resp.Body).Decode(&result)
        resp.Body.Close()

        if result["status"] == "completed" {
            fmt.Printf("Audio URL: %s\n", result["audio_url"])
            break
        } else if result["status"] == "failed" {
            fmt.Printf("Error: %s\n", result["error"])
            break
        }

        time.Sleep(5 * time.Second)
    }
}
```
