## Aperçu

Cet endpoint retourne les totaux de solde actuels de l’organisation associée au jeton de gestion. Il utilise la même source de vérité de solde d’organisation que la facturation et le règlement.

## Remarques

- Les champs monétaires de référence sont renvoyés en USD : `balance`, `total_recharge` et `total_used`.
- `preferred_currency` reflète la préférence d’affichage du Dashboard de l’organisation ; elle ne modifie pas les champs de référence en USD.

## Exemple

<RequestExample>
```bash cURL
curl -X GET "https://api.aisonar.dev/v1/management/balance" \
  -H "Authorization: Bearer mt-your-management-token"
```
</RequestExample>

## Demande / Réponse

Utilisez le panneau OpenAPI interactif ci-dessus pour le schéma exact.

## Exemple de réponse

<ResponseExample>
```json 200 OK
{
  "object": "management.balance",
  "currency": "USD",
  "available_balance": "42.50",
  "pending_usage": "1.25",
  "updated_at": "2026-06-27T00:00:00.000Z"
}
```
</ResponseExample>

## Champs importants

<ResponseField name="available_balance" type="string">Chaîne décimale USD pour le rapprochement de facturation.</ResponseField>
<ResponseField name="pending_usage" type="string">Chaîne décimale USD pour le rapprochement de facturation.</ResponseField>
<ResponseField name="updated_at" type="string">Horodatage ISO ou Unix retourné par l’API.</ResponseField>
