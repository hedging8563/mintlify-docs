## لماذا تُعدّ Coding Agents مكلفة؟

أدوات مثل Claude Code وCursor وCodex CLI قوية للغاية — لكنها تستهلك الـ tokens بمعدل يفاجئ معظم الفرق.

**سياق طويل في كل طلب.** كل استدعاء يتضمن system prompt وملفات المشروع وسجل المحادثة — غالبًا 10,000–100,000+ token قبل أن يبدأ النموذج في التفكير.

**استدعاءات متكررة.** جلسة برمجة واحدة تُطلق عشرات استدعاءات API: توليد الكود، البحث، المراجعة، الإكمال التلقائي، واستخدام الأدوات. جلسة مدتها ساعة واحدة قد تتجاوز 200+ طلب.

**تراكم المحادثة.** كل دور يُعيد إرسال كامل سجل الرسائل. بحلول الدور العشرين، أنت تدفع مقابل نفس السياق 20 مرة.

<Info>
جلسة Claude Code نموذجية لمدة ساعة قد تستهلك 2–5 مليون token. بأسعار API المباشرة، هذا يعني $6–30+ في الساعة حسب النموذج.
</Info>
