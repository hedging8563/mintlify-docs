## نظرة عامة

استخدم JSON Schema وtool calls والميزات الأصلية عبر مسارات OpenAI وAnthropic وGemini المتوافقة.

استخدم واجهات AI Sonar العامة وعقود النماذج؛ ولا تعتمد على تفاصيل تنفيذ المزوّد.

## قواعد أساسية

- احفظ schema version, tool name, validated arguments قبل إرجاع النجاح للمستخدم.
- استخدم واجهات AI Sonar العامة وعقود النماذج؛ ولا تعتمد على تفاصيل تنفيذ المزوّد.
- اجعل retries آمنة للتكرار وتجنب التوليد أو الفوترة المزدوجة.

## قائمة فحص الإنتاج

- احفظ schema version, tool name, validated arguments قبل إرجاع النجاح للمستخدم.
- استخدم واجهات AI Sonar العامة وعقود النماذج؛ ولا تعتمد على تفاصيل تنفيذ المزوّد.
- اجعل retries آمنة للتكرار وتجنب التوليد أو الفوترة المزدوجة.
- يجب أن تساعد logs الدعم دون تخزين أسرار أو payloads خاصة.
- تحقق من السلوك في staging قبل تغيير حركة الإنتاج.
