قم بتوليد الموسيقى والكلمات باستخدام الذكاء الاصطناعي. هذه API غير متزامنة. تعيد استجابة الإنشاء هوية المهمة، وقد تتضمن أيضًا `poll_url` مفضلًا عند توفره.

## جسم الطلب

<ParamField body="model" type="string" default="suno_music">
  النموذج المستخدم: `suno_music` لتوليد الموسيقى، و `suno_lyrics` للكلمات فقط.
</ParamField>

<ParamField body="prompt" type="string" required>
  وصف للموسيقى المراد توليدها. يمكن أن يتضمن النمط، والمزاج، والآلات الموسيقية، وما إلى ذلك.
</ParamField>

<ParamField body="title" type="string">
  عنوان الأغنية التي تم توليدها.
</ParamField>

<ParamField body="tags" type="string">
  علامات النمط (مثل: "pop, upbeat, electronic").
</ParamField>

<ParamField body="action" type="string">
  نوع التوليد: `MUSIC` (افتراضي) أو `LYRICS`.
</ParamField>

<ParamField body="mv" type="string">
  إصدار النموذج المستخدم.
</ParamField>

<ParamField body="continue_clip_id" type="string">
  معرف (ID) لمقطع سابق للمتابعة منه.
</ParamField>

<ParamField body="continue_at" type="number">
  الطابع الزمني (بالثواني) للمتابعة منه.
</ParamField>

<ParamField body="user" type="string">
  معرف فريد للمستخدم النهائي.
</ParamField>

## الاستجابة

<ResponseField name="id" type="string">
  معرف المهمة (Task ID) للاستعلام عن الحالة.
</ResponseField>

<ResponseField name="task_id" type="string">
  الاسم البديل لمعرّف المهمة غير المتزامنة عندما يعيده المهايئ.
</ResponseField>

<ResponseField name="poll_url" type="string">
  رابط polling المفضّل عندما يكون متاحًا.
</ResponseField>

<ResponseField name="status" type="string">
  حالة المهمة: `pending` أو `processing` أو `completed` أو `failed`.
</ResponseField>

<ResponseField name="created" type="integer">
  طابع Unix الزمني لوقت إنشاء المهمة.
</ResponseField>

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":  "suno_music",
        "prompt": "An upbeat electronic dance track with heavy bass",
        "title":  "Night Drive",
        "tags":   "electronic, EDM, energetic",
    }
    body, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.aisonar.dev/v1/music/generations", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Printf("Task ID: %s\n", result["id"])
}
```

```php PHP

        'tags' => 'electronic, EDM, energetic'
    ])
]);

curl_close($ch);

$data = json_decode($response, true);
echo "Task ID: " . $data['id'];
```
