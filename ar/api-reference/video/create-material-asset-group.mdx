ينشئ مجموعة مواد AI Sonar Seedance للمؤسسة الحالية.

أنشئ مجموعة عندما تريد تنظيم مواد Seedance 2.0 القابلة لإعادة الاستخدام حسب نوع المكتبة. تكون المجموعات ضمن نطاق المؤسسة الموثقة ولا يمكن استخدامها عبر حسابات أخرى.

يمكن لمعظم عمليات التكامل تخطي هذا الطلب وعدم إرسال `group_id` عند إنشاء مادة. سيستخدم AI Sonar أو ينشئ المجموعة الافتراضية للمؤسسة حسب `library_type` المحدد.

- `aigc_avatar`: مواد شخصية افتراضية أو صورة رمزية.
- `liveness_face`: مواد شخص حقيقي معتمد.

استخدم `id` المعاد كقيمة `group_id` أو `material_group_id` في [إنشاء مادة](/ar/api-reference/video/create-material-asset).
