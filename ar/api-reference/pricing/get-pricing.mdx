## نظرة عامة

استخدم هذا الـ endpoint لعرض سطح الأسعار العام الحالي للنماذج النشطة.

للاطلاع على تفاصيل التسعير الكاملة لنموذج واحد، بما في ذلك التسعير المنظم ومصدر البيانات، فضّل [الحصول على تسعير النموذج](/api-reference/models/get-model).

## معلمات الاستعلام

<ParamField query="provider" type="string">
  فلتر اختياري حسب المزود، مثل `openai` أو `anthropic` أو `google`.
</ParamField>

<ParamField query="tag" type="string">
  فلتر اختياري حسب وسم النموذج، مثل `chat` أو `image` أو `video` أو `embedding` أو `translation`.
</ParamField>

## الاستجابة

<ResponseField name="object" type="string">
  دائمًا `pricing`.
</ResponseField>

<ResponseField name="updated_at" type="string">
  طابع زمني بصيغة ISO لأحدث إدخال تسعير مضمن في الاستجابة.
</ResponseField>

<ResponseField name="currency" type="string">
  دائمًا `USD`.
</ResponseField>

<ResponseField name="data" type="array">
  مصفوفة من إدخالات التسعير. يتضمن كل عنصر `model` و `provider` و `is_lock_price` و `capabilities` و `aliases` وحقول الملخص العام للتسعير، سواء كانت قيمًا scalar أو بنية منظمة، كما تعيدها الواجهة التفاعلية أعلاه.
</ResponseField>

```python Python
import requests

response = requests.get(
    "https://api.aisonar.dev/v1/pricing",
    headers={"Authorization": "Bearer sk-your-api-key"},
    params={"provider": "openai", "tag": "chat"}
)

print(response.json())
```


```go Go
package main

import (
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    req, _ := http.NewRequest("GET", "https://api.aisonar.dev/v1/pricing?provider=openai&tag=chat", nil)
    req.Header.Set("Authorization", "Bearer sk-your-api-key")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Println(result)
}
```

<Note>
استخدم لوحة OpenAPI التفاعلية أعلاه للحصول على مخطط الاستجابة الدقيق. قد تكشف إدخالات التسعير العامة أيضًا `lock_price_per_request` و `has_complex_pricing` و `supported_operations` و `pricing_summary` و `pricing_provenance`.
</Note>
