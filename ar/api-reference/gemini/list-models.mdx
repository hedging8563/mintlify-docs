---

يُرجع قائمة بالنماذج المتاحة بتنسيق Google Gemini API.

## معلمات الاستعلام

<ParamField query="pageSize" type="integer">
  أقصى عدد من النماذج التي سيتم إرجاعها. القيمة الافتراضية: `1000`، الحد الأقصى: `1000`.
</ParamField>

<ParamField query="pageToken" type="string">
  رمز متابعة غير شفاف تعيده الصفحة السابقة. أرسله كما هو للحصول على الصفحة التالية.
</ParamField>

## المصادقة

لا يلزم التوثيق لنقاط قراءة كتالوج النماذج العامة.

مؤشرات المصادقة على نمط Gemini مثل `?key=` أو `x-goog-api-key` لا تغيّر عقد الاستجابة هنا.

## الاستجابة

<ResponseField name="models" type="array">
  مصفوفة من كائنات النماذج.
</ResponseField>

<ResponseField name="nextPageToken" type="string">
  يُعاد عند وجود المزيد من النماذج. مرّر هذه القيمة كـ `pageToken` لمتابعة التصفح.
</ResponseField>

## الأخطاء

- إعادة `400 INVALID_ARGUMENT` عند إرسال `pageToken` غير صالح.
- إعادة `500 INTERNAL` عند فشل غير متوقع في عرض القائمة.

<RequestExample>
```bash cURL
curl "https://api.aisonar.dev/v1beta/models?pageSize=5" \
  -H "x-goog-api-key: sk-your-api-key"
```

```python Python
import google.generativeai as genai

genai.configure(
    api_key="sk-your-api-key",
    transport="rest",
    client_options={"api_endpoint": "api.aisonar.dev"}
)

for model in genai.list_models():
    print(model.name)
```

```javascript JavaScript
const response = await fetch(
  "https://api.aisonar.dev/v1beta/models?pageSize=5",
  { headers: { "x-goog-api-key": "sk-your-api-key" } }
);
const { models } = await response.json();
models.forEach(m => console.log(m.name));
```
</RequestExample>
