## معاملات المسار

<ParamField path="model" type="string" required>
  معرّف النموذج المراد استرجاعه (على سبيل المثال، `gpt-5.4` أو `claude-sonnet-4-6`).
</ParamField>

## الاستجابة

<ResponseField name="id" type="string">
  معرّف النموذج.
</ResponseField>

<ResponseField name="object" type="string">
  دائمًا `model`.
</ResponseField>

<ResponseField name="created" type="integer">
  طابع زمني للإنشاء.
</ResponseField>

<ResponseField name="owned_by" type="string">
  مزود النموذج.
</ResponseField>

<ResponseField name="aisonar" type="object">
  بيانات وصفية عامة خاصة بـ AI Sonar. قد تتضمن مسار التفاصيل `category` و `pricing_unit` و `capability_flags` و `supported_operations` و `pricing_provenance` و `public_contract_summary` و `public_contract` الكامل و `agent_preferences` على مستوى السيناريو للقطات التوصية غير chat.
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.aisonar.dev/v1/models/gpt-5.4" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.aisonar.dev/v1"
)

model = client.models.retrieve("gpt-5.4")
print(f"Model: {model.id}, Provider: {model.owned_by}")
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.aisonar.dev/v1'
});

const model = await client.models.retrieve('gpt-5.4');
console.log(`Model: ${model.id}, Provider: ${model.owned_by}`);
```

```go Go
package main

import (
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    req, _ := http.NewRequest("GET", "https://api.aisonar.dev/v1/models/gpt-5.4", nil)
    req.Header.Set("Authorization", "Bearer sk-your-api-key")

    resp, _ := http.DefaultClient.Do(req)
    defer resp.Body.Close()

    var model map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&model)
    fmt.Printf("Model: %s, Provider: %s\n", model["id"], model["owned_by"])
}
```

```php PHP
<?php
$ch = curl_init('https://api.aisonar.dev/v1/models/gpt-5.4');

curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer sk-your-api-key'
    ]
]);

$response = curl_exec($ch);
curl_close($ch);

$model = json_decode($response, true);
echo "Model: {$model['id']}, Provider: {$model['owned_by']}\n";
```

## معالجة الأخطاء

إذا لم يكن النموذج موجودًا، فستتلقى خطأ 404:

```json
{
  "error": {
    "message": "Model 'invalid-model' not found",
    "type": "invalid_request_error",
    "code": "model_not_found"
  }
}
```

## بيانات التوصية للنماذج غير chat

بالنسبة للنماذج غير chat، تُرجع صفحة التفاصيل أيضًا `aisonar.agent_preferences.<scene>`.

استخدمها لفحص:

- `preferred_rank`
- `success_rate_24h`
- `sample_count_24h`
- `status`
- `updated_at`

تُبقي الاستجابة العامة `basis.channel_id` و `basis.physical_model` على `null` عمدًا حتى تتمكن الوكلاء من استهلاك إشارة التوصية دون معرفة معرّفات التوجيه الداخلية.

## بيانات التفاصيل العامة

بالمقارنة مع `GET /v1/models`، يمكن لصفحة التفاصيل أن تعيد أيضًا:

- `aisonar.capability_flags`
- `aisonar.supported_operations`
- `aisonar.pricing_provenance`
- `aisonar.public_contract_summary`
- `aisonar.public_contract`
