## نظرة عامة

`GET /v1/models` مهيأ للاكتشاف. أما البيانات الوصفية الخاصة بالتفاصيل فقط، مثل `capability_flags` و `pricing_provenance` و `public_contract` الكامل، فتوجد في `GET /v1/models/{model}`.

## معلمات الاستعلام

<ParamField query="category" type="string">
  فلتر اختياري للفئة العامة. يدعم `chat` و `image` و `video` و `audio` و `tts` و `stt` و `music` و `3d` و `embedding` و `rerank` و `translation`.
</ParamField>

<ParamField query="recommended_for" type="string">
  سيناريو اختياري للتوصية خارج chat. يدعم `image` و `video` و `music` و `3d` و `tts` و `stt` و `embedding` و `rerank` و `translation`.
</ParamField>

<ParamField query="provider" type="string">
  فلتر اختياري حسب المزود، مثل `openai` أو `anthropic` أو `google` أو `deepseek`.
</ParamField>

<ParamField query="tag" type="string">
  فلتر اختياري حسب وسم النموذج، مثل `chat` أو `image` أو `video` أو `embedding` أو `translation`.
</ParamField>

<Note>
عندما تكون `recommended_for` موجودة، يرتب `/v1/models` النماذج غير chat وفق أحدث لقطة مخزنة مؤقتًا لمعدل النجاح خلال 24 ساعة. تظل النماذج ذات `status = "insufficient_samples"` مرئية، لكنها تظهر بعد النماذج ذات النقاط.
</Note>

## الاستجابة

<ResponseField name="object" type="string">
  دائمًا `list`.
</ResponseField>

<ResponseField name="data" type="array">
  مصفوفة من كائنات النماذج.

  يحتوي كل نموذج على:
  - `id` (string): معرّف النموذج
  - `object` (string): `model`
  - `created` (integer): طابع زمني للإنشاء
  - `owned_by` (string): مزود النموذج
  - `aisonar.category` (string): الفئة العامة للنموذج
  - `aisonar.pricing_unit` (string or `null`): وحدة التسعير العامة عند توفرها
  - `aisonar.capability_flags` (object or `null`): ملخص عام للقدرات
  - `aisonar.supported_operations` (array or `null`): العمليات العامة غير chat التي يدعمها النموذج
  - `aisonar.pricing_provenance` (object or `null`): بيانات وصفية عامة لمصدر التسعير
  - `aisonar.public_contract_summary` (object or `null`): ملخص اكتشاف خفيف غير chat يحتوي فقط على `public_operations` و `request_endpoint`
  - `aisonar.agent_preferences` (object): يُعاد فقط عندما تكون `recommended_for` موجودة
</ResponseField>

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.aisonar.dev/v1"
)

models = client.models.list()

for model in models.data:
    print(f"{model.id} ({model.owned_by})")
```

```go Go
package main

import (
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    req, _ := http.NewRequest("GET", "https://api.aisonar.dev/v1/models", nil)
    req.Header.Set("Authorization", "Bearer sk-your-api-key")

    resp, _ := http.DefaultClient.Do(req)
    defer resp.Body.Close()

    var payload struct {
        Data []struct {
            ID      string `json:"id"`
            OwnedBy string `json:"owned_by"`
        } `json:"data"`
    }

    json.NewDecoder(resp.Body).Decode(&payload)

    for _, model := range payload.Data {
        fmt.Printf("%s (%s)\n", model.ID, model.OwnedBy)
    }
}
```

## التصفية حسب المزود

```python
# الحصول على كل نماذج OpenAI
openai_models = [m for m in models.data if m.owned_by == "openai"]

```
