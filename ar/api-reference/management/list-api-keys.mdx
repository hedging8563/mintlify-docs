## نظرة عامة

يعيد مفاتيح API الخاصة بالمستخدم في المؤسسة المرتبطة برمز الإدارة الحالي.

## ملاحظات

- يتم استبعاد المفاتيح التي يديرها النظام.

## مثال

<RequestExample>
```bash cURL
curl -X GET "https://api.aisonar.dev/v1/management/api-keys" \
  -H "Authorization: Bearer mt-your-management-token"
```
</RequestExample>

## الطلب / الاستجابة

استخدم لوحة OpenAPI التفاعلية أعلاه للحصول على المخطط الدقيق.

## مثال الاستجابة

<ResponseExample>
```json 200 OK
{
  "object": "list",
  "data": [{
    "id": "ak_abc123",
    "name": "Production app",
    "status": "active",
    "created_at": "2026-06-27T00:00:00.000Z",
    "last_used_at": "2026-06-27T01:00:00.000Z"
  }]
}
```
</ResponseExample>

## حقول مهمة

<ResponseField name="data" type="array">القائمة أو payload المعاد في هذه الاستجابة.</ResponseField>
