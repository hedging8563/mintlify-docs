<span data-mintlify-rebuild="2026-05-19-after-mdx-parse-fix" aria-hidden="true" />

توليد نماذج ثلاثية الأبعاد من النصوص أو الصور باستخدام Tripo3D ومزودين آخرين. هذه API غير متزامنة. تعيد استجابة الإنشاء هوية المهمة، وقد تتضمن أيضًا `poll_url` مفضلًا عند توفره.

## جسم الطلب

<ParamField body="model" type="string" default="tripo3d-v2.5">
  النموذج المراد استخدامه (على سبيل المثال، `tripo3d-v2.5` أو `tripo3d-v2`).
</ParamField>

<ParamField body="prompt" type="string" required>
  وصف نصي للنموذج ثلاثي الأبعاد المراد توليده.
</ParamField>

<ParamField body="image" type="string">
  صورة بترميز Base64 لتوليد نموذج ثلاثي الأبعاد من صورة.
</ParamField>

<ParamField body="image_url" type="string">
  رابط (URL) الصورة لتوليد نموذج ثلاثي الأبعاد من صورة.
</ParamField>

<ParamField body="format" type="string" default="glb">
  تنسيق المخرجات: `glb` أو `fbx` أو `obj` أو `usdz`.
</ParamField>

<ParamField body="quality" type="string" default="standard">
  مستوى الجودة: `draft` أو `standard` أو `high`.
</ParamField>

<ParamField body="style" type="string">
  نمط مسبق الإعداد للنموذج.
</ParamField>

<ParamField body="seed" type="integer">
  بذرة (seed) لضمان توليد نتائج متطابقة.
</ParamField>

<ParamField body="user" type="string">
  معرف فريد للمستخدم النهائي.
</ParamField>

## الاستجابة

<ResponseField name="id" type="string">
  معرف المهمة (Task ID) للاستعلام عن الحالة.
</ResponseField>

<ResponseField name="task_id" type="string">
  الاسم البديل لمعرّف المهمة غير المتزامنة عندما يعيده المهايئ.
</ResponseField>

<ResponseField name="poll_url" type="string">
  رابط polling المفضّل عندما يكون متاحًا.
</ResponseField>

<ResponseField name="status" type="string">
  حالة المهمة: `pending` أو `processing` أو `completed` أو `failed`.
</ResponseField>

<ResponseField name="created" type="integer">
  طابع زمني بتنسيق Unix لوقت إنشاء المهمة.
</ResponseField>

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":   "tripo3d-v2.5",
        "prompt":  "A detailed medieval castle with towers",
        "format":  "glb",
        "quality": "high",
    }
    body, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.aisonar.dev/v1/3d/generations", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer sk-your-api-key")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Printf("Task ID: %s\n", result["id"])
}
```

```php PHP
<?php
$ch = curl_init('https://api.aisonar.dev/v1/3d/generations');

curl_setopt_array($ch, [

    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Content-Type: application/json',
        'Authorization: Bearer sk-your-api-key'
    ],
    CURLOPT_POSTFIELDS => json_encode([
        'model' => 'tripo3d-v2.5',
        'prompt' => 'A detailed medieval castle with towers',
        'format' => 'glb',
        'quality' => 'high'
    ])
]);

$response = curl_exec($ch);
curl_close($ch);

$data = json_decode($response, true);
echo "Task ID: " . $data['id'];
```
