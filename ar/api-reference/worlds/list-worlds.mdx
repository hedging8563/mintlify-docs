---

يعرض World Labs Marble worlds المكتملة التي تم إنشاؤها عبر AI Sonar. النتائج مقيدة بمؤسستك وتشمل فقط الموارد المنشأة عبر AI Sonar.

استخدم `/v1/worlds/generations/{id}` للمهام قيد الانتظار أو الفاشلة. هذه القائمة تعيد فقط العوالم المكتملة التي فهرسها AI Sonar بعد الإنشاء.

## جسم الطلب

<ParamField body="page_size" type="integer" default="20">
  عدد worlds المراد إرجاعها، من 1 إلى 100.
</ParamField>

<ParamField body="page_token" type="string">
  رمز الصفحة من الاستجابة السابقة.
</ParamField>

<ParamField body="status" type="string">
  مرشح حالة اختياري. يعيد AI Sonar worlds مكتملة عند `SUCCEEDED`؛ الحالات الأخرى تخص نقطة مهام الإنشاء.
</ParamField>

<ParamField body="model" type="string">
  مرشح اختياري لنموذج Marble.
</ParamField>

<ParamField body="tags" type="array">
  مرشح اختياري للوسوم.
</ParamField>

<ParamField body="is_public" type="boolean">
  مرشح اختياري للرؤية.
</ParamField>

<ParamField body="created_after" type="string">
  إرجاع worlds المنشأة بعد هذا الوقت فقط.
</ParamField>

<ParamField body="created_before" type="string">
  إرجاع worlds المنشأة قبل هذا الوقت فقط.
</ParamField>

<ParamField body="sort_by" type="string" default="created_at">
  الترتيب حسب `created_at` أو `updated_at`.
</ParamField>

<ParamField body="sort_order" type="string" default="desc">
  اتجاه الترتيب: `asc` أو `desc`.
</ParamField>

## الاستجابة

<ResponseField name="worlds" type="array">
  مصفوفة سجلات world.
</ResponseField>

<ResponseField name="next_page_token" type="string">
  رمز الصفحة التالية أو `null`.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.aisonar.dev/v1/worlds/list" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{"page_size": 20, "status": "SUCCEEDED", "model": "marble-1.1"}'
```

```python Python
import requests

response = requests.post(
    "https://api.aisonar.dev/v1/worlds/list",
    headers={"Authorization": "Bearer sk-your-api-key"},
    json={"page_size": 20, "status": "SUCCEEDED", "model": "marble-1.1"},
)
print(response.json()["worlds"])
```
