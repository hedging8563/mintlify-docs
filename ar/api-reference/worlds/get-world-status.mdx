---
title: "الحصول على حالة World"
openapi: "GET /v1/worlds/generations/{id}"
description: "يجلب حالة ونتيجة مهمة توليد عالم World Labs Marble"
---

استخدم polling لهذا endpoint للتحقق من حالة مهمة توليد World Labs Marble. إذا تضمنت استجابة الإنشاء `poll_url`، فاستخدم هذا الرابط كما هو.

## معلمات المسار

<ParamField path="id" type="string" required>
  معرف مهمة توليد العالم.
</ParamField>

## الاستجابة

<ResponseField name="id" type="string">
  معرف مهمة عام.
</ResponseField>

<ResponseField name="task_id" type="string">
  اسم بديل لمعرف المهمة غير المتزامنة.
</ResponseField>

<ResponseField name="operation_id" type="string">
  معرف operation في World Labs upstream.
</ResponseField>

<ResponseField name="status" type="string">
  حالة المهمة: `pending` أو `processing` أو `completed` أو `failed` أو `timeout`.
</ResponseField>

<ResponseField name="world_id" type="string">
  معرف world في World Labs upstream عند الاكتمال.
</ResponseField>

<ResponseField name="world_marble_url" type="string">
  رابط عالم Marble الناتج عند الاكتمال.
</ResponseField>

<ResponseField name="model_url" type="string">
  اسم بديل لرابط العالم/النموذج الأساسي الناتج.
</ResponseField>

<ResponseField name="glb_url" type="string">
  رابط GLB لشبكة collider عند توفره.
</ResponseField>

<ResponseField name="pano_url" type="string">
  رابط صورة بانورامية عند توفرها.
</ResponseField>

<ResponseField name="thumbnail_url" type="string">
  رابط صورة مصغرة عند توفره.
</ResponseField>

<ResponseField name="caption" type="string">
  Caption من upstream للعالم الناتج.
</ResponseField>

<ResponseField name="cost" type="object">
  تفاصيل تكلفة operation من upstream عند توفرها، بما في ذلك `total_credits`.
</ResponseField>

<ResponseField name="error" type="string">
  رسالة الخطأ عند فشل المهمة.
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.aisonar.dev/v1/worlds/generations/ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
import requests
import time

task_id = "ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa"

while True:
    response = requests.get(
        f"https://api.aisonar.dev/v1/worlds/generations/{task_id}",
        headers={"Authorization": "Bearer sk-your-api-key"},
    )
    result = response.json()
    if result["status"] in ["completed", "failed", "timeout"]:
        print(result)
        break
    time.sleep(10)
```

```javascript JavaScript
const taskId = 'ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa';
const response = await fetch(`https://api.aisonar.dev/v1/worlds/generations/${taskId}`, {
  headers: { Authorization: 'Bearer sk-your-api-key' }
});

const result = await response.json();
console.log(result.status, result.world_marble_url);
```
