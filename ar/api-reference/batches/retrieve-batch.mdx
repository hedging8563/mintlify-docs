## نظرة عامة

تعتبر هذه النقطة جزءًا من واجهة برمجة التطبيقات الخاصة بالدفعات المتوافقة مع OpenAI من AI Sonar. تستخدم مهام الدفعات ملفات JSONL المرفوعة، وتعمل بشكل غير متزامن، ويمكن أن تعيد ملفات الإخراج/الأخطاء لاحقًا.

## ملاحظات

- يجب أن تستخدم ملفات الإدخال للدفعة `purpose=batch`.
- `completion_window` حاليًا `24h`.
- ترتيب الإخراج غير مضمون؛ يجب دائمًا المطابقة بواسطة `custom_id`.
- لا يتم دعم البث داخل عناصر الدفعة.

## مثال

<RequestExample>
```bash cURL
curl -X GET "https://api.aisonar.dev/v1/batches/batch_abc123" \
  -H "Authorization: Bearer sk-your-api-key"
```
</RequestExample>

## الطلب / الاستجابة

استخدم لوحة OpenAPI التفاعلية أعلاه للحصول على المخطط الدقيق.

## نصائح تشغيلية

- استخدم `custom_id` للمصالحة غير القابلة للتغيير في الأسفل.
- توقع `output_file_id` و `error_file_id` فقط بعد أن ينهي العامل الدفعة.
- قد تختلف أسعار الدفعات عن الأسعار المتزامنة لأن قواعد الخصم لـ `isBatchRequest=true` تنطبق.
