## ما هي AI Sonar؟

AI Sonar هي منصة تجميع واجهات برمجة تطبيقات الذكاء الاصطناعي تمنحك وصولاً موحّداً إلى **مئات نماذج الذكاء الاصطناعي** من مزودين مثل OpenAI وAnthropic وGoogle وDeepSeek وMoonshot وMiniMax والمزيد.

## الإمكانيات المدعومة

يتغير الكتالوج العام بمرور الوقت. تحقق من التوفر الحالي عبر `/v1/models` أو `recommended_for` أو صفحة النماذج؛ تستخدم الأمثلة أدناه IDs عامة حالية.

| الإمكانية | الوصف | نماذج أمثلة |
|---|---|---|
| المحادثة والاستدلال | توليد النص والتخطيط والأدوات | `gpt-5.4`, `claude-sonnet-4-6`, `deepseek-r1` |
| الرؤية ومتعدد الوسائط | فهم الصور والمحادثة متعددة الوسائط | `gpt-4o`, `claude-sonnet-4-6`, `gemini-3.5-flash` |
| توليد الصور وتحريرها | تحويل النص إلى صورة وتحرير الصور | `gpt-image-2`, `flux-pro`, `qwen-image` |
| توليد الفيديو | نص/صورة/مرجع إلى فيديو والتحكم بالحركة | `veo3.1`, `seedance-2.0`, `kling-3.0-video` |
| الصوت وRealtime | TTS والنسخ والترجمة والجلسات الفورية | `gpt-4o-mini-tts`, `whisper-1`, `gpt-realtime` |
| Embeddings وrerank | تمثيل النصوص وإعادة الترتيب | `text-embedding-3-small`, `qwen3-vl-rerank` |
| الترجمة | سير عمل ترجمة النص والكلام | `qwen-mt-plus`, `gpt-realtime-translate` |
| الموسيقى و3D | توليد الموسيقى وأصول 3D | `suno-music`, `tripo3d-v2.5` |
| Worlds | توليد worlds وتتبع الحالة وتدفقات media assets | تحقق من التوفر عبر `/v1/worlds/generations` وModels API |

## اختر صيغة الطلب

ابدأ بالمسار الذي يطابق السلوك الذي تحتاجه. استخدم المسارات الأصلية لـ Anthropic/Gemini للميزات الأصلية الخاصة بالمزوّد؛ واستخدم `/v1/chat/completions` المتوافق مع OpenAI عند ترحيل شيفرة الدردشة الحالية بأسلوب OpenAI.

<CodeGroup>

```bash cURL
curl https://api.aisonar.dev/v1/chat/completions \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-5.4",
    "messages": [
      {"role": "user", "content": "Explain what AI Sonar does in one sentence."}
    ]
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.aisonar.dev/v1"
)

response = client.chat.completions.create(
    model="gpt-5.4",
    messages=[{"role": "user", "content": "Explain what AI Sonar does in one sentence."}]
)

print(response.choices[0].message.content)
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.aisonar.dev/v1'
});

const response = await client.chat.completions.create({
  model: 'gpt-5.4',
  messages: [{ role: 'user', content: 'Explain what AI Sonar does in one sentence.' }],
});

console.log(response.choices[0].message.content);
```

<Note>
استخدم `POST /v1/responses` فقط عندما تحتاج صراحةً إلى سلوك مخصص لـ Responses. ليست هذه هي مسار التوافق الافتراضي عبر جميع النماذج.
</Note>

</CodeGroup>
