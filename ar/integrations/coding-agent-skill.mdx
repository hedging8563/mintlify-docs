## ما الذي تفعله المهارة

- تولّد أمثلة صغيرة قابلة للتشغيل لعائلات AI Sonar API مثل chat والصور والصوت والفيديو والترجمة وغيرها.
- تستخدم `https://api.aisonar.dev/v1` للعملاء المتوافقين مع OpenAI وتوضح متى يجب الانتقال إلى مسارات Anthropic أو Gemini الأصلية.
- تكتشف النماذج عبر `/v1/models` و`/llms.txt` وقوائم `recommended_for` بدل التخمين من قوائم قديمة.
- تقرأ العقد العام للنموذج قبل إعادة محاولة طلبات غير المحادثة حتى لا تُزال الحقول غير المدعومة بصمت.
- تتعامل مع تلميحات Agent-First مثل `did_you_mean` و`suggestions` و`retry_after` و`recommended_request`.

## الحصول على API Key

<Steps>
  <Step title="افتح AI Sonar">
    انتقل إلى [aisonar.dev](https://aisonar.dev)
  </Step>
  <Step title="سجّل الدخول">
    أنشئ حسابًا أو سجّل الدخول
  </Step>
  <Step title="احصل على API Key">
    افتح [Dashboard → API Keys](https://console.aisonar.dev/api) وأنشئ key جديدًا
  </Step>
  <Step title="انسخ key">

</Tip>

## سير عمل Agent الموصى به

1. ابدأ بأصغر مثال يعمل ويطابق لغة المستخدم وعائلة API المطلوبة.
2. عندما يكون اختيار النموذج مفتوحًا، استدعِ `/v1/models` أو `https://api.aisonar.dev/llms.txt` قبل تثبيت نموذج في الكود.
3. لأعمال غير المحادثة، استدعِ `/v1/models?recommended_for=<scene>` حيث تكون `<scene>` واحدة من `image` أو `video` أو `music` أو `3d` أو `tts` أو `stt` أو `embedding` أو `rerank` أو `translation`.
4. قبل تعديل طلب غير محادثة فشل، اقرأ `/v1/models/:model` ووافق `supported_operations` و`supported_parameters` و`request_endpoint` و`request_shape_mode` و`recommended_request`.
5. إذا أعادت API خطأ Agent-First، استخدم الحقول المنظمة لتصحيح الطلب ولا تعِد المحاولة إلا إذا كان الخطأ retryable.

## مثال محادثة بسيط

يستخدم هذا المثال OpenAI Python SDK مع base URL الخاص بـ AI Sonar:

```python
import os
from openai import OpenAI

client = OpenAI(
    api_key=os.environ["TOKENLAB_API_KEY"],
    base_url="https://api.aisonar.dev/v1"
)

response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)
```

شغّله باستخدام:

```bash
pip install openai
export TOKENLAB_API_KEY="sk-your-api-key"
python app.py
```

## اكتشاف النماذج والعقود العامة

فضّل الاكتشاف الحي بدل القوائم المضمّنة القديمة:

```bash
# نظرة عامة على API قابلة للقراءة آليًا
curl https://api.aisonar.dev/llms.txt

# عرض النماذج
curl "https://api.aisonar.dev/v1/models" -H "Authorization: Bearer sk-KEY"
curl "https://api.aisonar.dev/v1/models?category=image" -H "Authorization: Bearer sk-KEY"

# قوائم مختصرة مرتبة لغير المحادثة
curl "https://api.aisonar.dev/v1/models?recommended_for=image" -H "Authorization: Bearer sk-KEY"
curl "https://api.aisonar.dev/v1/models?recommended_for=video" -H "Authorization: Bearer sk-KEY"
curl "https://api.aisonar.dev/v1/models?recommended_for=translation" -H "Authorization: Bearer sk-KEY"

# اقرأ العقد العام لنموذج واحد قبل إعادة محاولة طلب غير محادثة
curl "https://api.aisonar.dev/v1/models/gpt-image-2" -H "Authorization: Bearer sk-KEY"

# تفاصيل التسعير فقط
curl "https://api.aisonar.dev/v1/models/gpt-image-2/pricing" -H "Authorization: Bearer sk-KEY"
```

## تلميحات التوجيه الأصلي

المسار المتوافق مع OpenAI `/v1` هو الافتراضي لأمثلة chat وResponses والصور وembedding والصوت وrerank. استخدم مسارات Anthropic أو Gemini الأصلية فقط عندما يحتاج الطلب إلى سلوك خاص بالمزوّد أو عندما يعيد AI Sonar تلميح التحسين the native endpoint hint header.



## استرداد أخطاء Agent-First

تتضمن الأخطاء حقولًا يمكن لوكلاء البرمجة تحليلها من دون استخراج النص من الوثائق:

| الخطأ | حقول مفيدة | إجراء agent |
|---|---|---|
| اسم النموذج غير صحيح | `did_you_mean`, `suggestions`, `hint` | أعِد المحاولة بالنموذج المصحح أو ببديل مقترح |
| الرصيد غير كافٍ | `balance_usd`, `estimated_cost_usd`, `suggestions` | اطلب موافقة أو انتقل إلى نموذج ميسور التكلفة |
| حد معدل أو عدم توفر مؤقت | `retryable`, `retry_after`, `alternatives` | انتظر ثم أعد المحاولة أو اختر بديلًا من القائمة |
| عدم تطابق عقد غير المحادثة | `supported_operations`, `supported_parameters`, `request_endpoint`, `request_shape_mode`, `recommended_request` | أعد بناء الطلب من العقد العام، واستخدم fail closed للحقول التي تغير النية |

## عائلات API المدعومة

| العائلة | المسار الرئيسي |
|---|---|
| Chat وResponses | `/v1/chat/completions`, `/v1/responses` |
| رسائل Claude الأصلية | `/v1/messages` |
| طلبات Gemini الأصلية | `/v1beta/models/{model}:generateContent` |
| الصور | `/v1/images/generations`, `/v1/images/edits` |
| الفيديو | `/v1/videos/generations` |
| الموسيقى | `/v1/music/generations` |
| Worlds | `/v1/worlds/generations` مع endpoints لحالة world وmedia asset |
| 3D | `/v1/3d/generations` |
| الصوت | `/v1/audio/speech`, `/v1/audio/transcriptions`, `/v1/audio/translations` |
| Realtime | `/v1/realtime?model={model}` |
| Embeddings وrerank | `/v1/embeddings`, `/v1/rerank` |
| ترجمة النص | `/v1/translations` |
