title: "APIキー請求情報を取得"
openapi: "GET /v1/management/api-keys/{keyId}/billing"
description: "ユーザー API キーの集計済み請求内訳を返します。"
---

## 概要

このエンドポイントは、選択した API キーに対する消費ベースの請求集計を返します。総コスト、総リクエスト数、課金単位別の使用量、リクエストで指定するモデル名（`model`）、ベンダー、`scene`、`accessChannel`、キャッシュ種別ごとの内訳、および日次コスト推移が含まれます。

## 注意事項

- チャージ、請求書、その他の組織レベルの財務記録は含まれません。

## 例

<RequestExample>
```bash cURL
curl -X GET "https://api.aisonar.dev/v1/management/api-keys/key_abc123def456/billing?startDate=2026-01-01&endDate=2026-01-31" \
  -H "Authorization: Bearer mt-your-management-token"
```
</RequestExample>

## リクエスト / レスポンス

正確なスキーマについては、上記のインタラクティブな OpenAPI パネルを使用してください。

## レスポンス例

<ResponseExample>
```json 200 OK
{
  "object": "management.billing",
  "api_key_id": "ak_abc123",
  "currency": "USD",
  "total_cost": "12.3400",
  "data": [{
    "billing_transaction_id": "billtxn_abc123",
    "request_id": "req_abc123",
    "amount": "0.0012",
    "created_at": "2026-06-27T00:00:00.000Z"
  }]
}
```
</ResponseExample>

## 重要フィールド

<ResponseField name="total_cost" type="string">請求照合に使う USD decimal string です。</ResponseField>
<ResponseField name="data" type="array">このレスポンスで返されるリストまたは payload です。</ResponseField>
<ResponseField name="billing_transaction_id" type="string">利用可能な場合に返る billing settlement identifier です。</ResponseField>
