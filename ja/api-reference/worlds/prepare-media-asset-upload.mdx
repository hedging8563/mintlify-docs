---
title: "メディアアセットアップロードを準備"
openapi: "POST /v1/worlds/media-assets/prepare-upload"
description: "World Labs メディアアセットのアップロード先を作成します"
---

World Labs のメディアアセット記録を作成し、画像または動画のアップロード URL を受け取ります。バイナリを `upload_info.upload_url` にアップロードした後、`world_prompt` で `source: "media_asset"` と返された `media_asset.media_asset_id` を指定できます。

AI Sonar はアセットの所有組織を記録します。同じ組織だけがこのメディアアセットを取得し、world 生成で利用できます。

## リクエストボディ

<ParamField body="file_name" type="string" required>
  元のファイル名。最大 64 文字。
</ParamField>

<ParamField body="kind" type="string" required>
  メディア種別: `image` または `video`。
</ParamField>

<ParamField body="extension" type="string">
  任意の拡張子。例: `png`、`jpg`、`mp4`。
</ParamField>

<ParamField body="metadata" type="object">
  上流メディアアセットに保存する任意のメタデータオブジェクト。
</ParamField>

## レスポンス

<ResponseField name="media_asset" type="object">
  準備された World Labs メディアアセットのメタデータ。
</ResponseField>

<ResponseField name="upload_info.upload_url" type="string">
  バイナリをアップロードするための上流署名付き URL。
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.aisonar.dev/v1/worlds/media-assets/prepare-upload" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "file_name": "reference.png",
    "kind": "image",
    "extension": "png"
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.aisonar.dev/v1/worlds/media-assets/prepare-upload",
    headers={"Authorization": "Bearer sk-your-api-key"},
    json={"file_name": "reference.png", "kind": "image", "extension": "png"},
)

print(response.json()["media_asset"]["media_asset_id"])
```

```javascript JavaScript
const response = await fetch('https://api.aisonar.dev/v1/worlds/media-assets/prepare-upload', {
  method: 'POST',
  headers: {
    Authorization: 'Bearer sk-your-api-key',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({ file_name: 'reference.png', kind: 'image', extension: 'png' })
});

const prepared = await response.json();
console.log(prepared.media_asset.media_asset_id, prepared.upload_info.upload_url);
```
</RequestExample>
