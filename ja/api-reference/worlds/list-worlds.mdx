---
title: "World を一覧"
openapi: "POST /v1/worlds/list"
description: "AI Sonar で作成された完了済み World Labs Marble world を一覧します"
---

AI Sonar で作成された完了済み World Labs Marble world を一覧します。結果は組織ごとに分離され、共有上流アカウント全体の一覧は公開されません。

保留中または失敗した生成タスクは `/v1/worlds/generations/{id}` を使用してください。この一覧は生成後に AI Sonar が索引化した完了済み world だけを返します。

## リクエストボディ

<ParamField body="page_size" type="integer" default="20">
  返す world 数。1 から 100。
</ParamField>

<ParamField body="page_token" type="string">
  前回レスポンスのページネーショントークン。
</ParamField>

<ParamField body="status" type="string">
  任意のステータスフィルター。AI Sonar は `SUCCEEDED` で完了済み world を返します。他のステータスは生成タスクエンドポイントで確認します。
</ParamField>

<ParamField body="model" type="string">
  任意の Marble モデルフィルター。
</ParamField>

<ParamField body="tags" type="array">
  任意のタグフィルター。
</ParamField>

<ParamField body="is_public" type="boolean">
  任意の公開範囲フィルター。
</ParamField>

<ParamField body="created_after" type="string">
  この時刻より後に作成された world のみ返します。
</ParamField>

<ParamField body="created_before" type="string">
  この時刻より前に作成された world のみ返します。
</ParamField>

<ParamField body="sort_by" type="string" default="created_at">
  `created_at` または `updated_at` で並べ替えます。
</ParamField>

<ParamField body="sort_order" type="string" default="desc">
  並び順: `asc` または `desc`。
</ParamField>

## レスポンス

<ResponseField name="worlds" type="array">
  World レコードの配列。
</ResponseField>

<ResponseField name="next_page_token" type="string">
  次ページのトークン、または `null`。
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.aisonar.dev/v1/worlds/list" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{"page_size": 20, "status": "SUCCEEDED", "model": "marble-1.1"}'
```

```python Python
import requests

response = requests.post(
    "https://api.aisonar.dev/v1/worlds/list",
    headers={"Authorization": "Bearer sk-your-api-key"},
    json={"page_size": 20, "status": "SUCCEEDED", "model": "marble-1.1"},
)
print(response.json()["worlds"])
```
