---
title: "素材を作成"
openapi: "POST /v1/videos/assets"
description: "Seedance の素材を AI Sonar 素材グループにアップロードします。"
---

Seedance の素材を AI Sonar 素材グループにアップロードします。

公開アクセス可能な素材 URL を、選択した Seedance 2.0 素材ライブラリにアップロードします。素材は認証済み組織にスコープされます。

多くのリクエストでは `library_type` だけを指定し、`group_id` は省略できます。AI Sonar はそのライブラリ種別に対して、組織のデフォルト素材グループを使用または作成します。特定の素材グループに入れる必要がある場合だけ `group_id` を指定してください。

- `aigc_avatar`: 仮想人物またはアバター素材。
- `liveness_face`: 認証済み実人物素材。

作成後、[素材を取得](/ja/api-reference/video/get-material-asset) をポーリングし、`status` が `ACTIVE` になるまで待ちます。その後、返された `id` を `material_asset_id` として、または `material_asset_ids` に含めて、[動画を作成](/ja/api-reference/video/create-video) に渡します。

素材が `FAILED` になった場合は、`error_message` を確認し、元素材を修正して新しい素材を作成してください。
