現在の組織に AI Sonar Seedance の素材グループを作成します。

Seedance 2.0 の再利用可能な素材をライブラリ種別ごとに整理したい場合に、素材グループを作成します。素材グループは認証済み組織にスコープされ、別アカウントでは使用できません。

多くの連携ではこの呼び出しを省略し、素材作成時に `group_id` を指定しなくてかまいません。AI Sonar は選択された `library_type` に対して、組織のデフォルト素材グループを使用または作成します。

- `aigc_avatar`: 仮想人物またはアバター素材。
- `liveness_face`: 認証済み実人物素材。

返された `id` は、[素材を作成](/ja/api-reference/video/create-material-asset) の `group_id` または `material_group_id` として使用できます。
