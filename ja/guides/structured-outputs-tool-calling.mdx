## 概要

OpenAI、Anthropic、Gemini 互換 route で JSON schema、tool calls、native format 機能を使います。

AI Sonar の公開 API とモデル契約 を使用し、プロバイダー実行詳細には依存しないでください。

## 重要なルール

- ユーザーへ成功を返す前に schema version, tool name, validated arguments を保存してください。
- AI Sonar の公開 API とモデル契約 を使用し、プロバイダー実行詳細には依存しないでください。
- リトライは冪等にし、重複生成や重複課金を避けてください。

## 本番チェックリスト

- ユーザーへ成功を返す前に schema version, tool name, validated arguments を保存してください。
- AI Sonar の公開 API とモデル契約 を使用し、プロバイダー実行詳細には依存しないでください。
- リトライは冪等にし、重複生成や重複課金を避けてください。
- サポートに役立つログを残しつつ、秘密情報や private payload は保存しないでください。
- 本番トラフィックを切り替える前にステージングで動作を検証してください。
