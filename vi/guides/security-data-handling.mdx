## Tổng quan

Xử lý API keys, media người dùng, log và asset tạo ra một cách an toàn trong AI Sonar.

Dùng API công khai và hợp đồng mô hình của AI Sonar; không phụ thuộc chi tiết thực thi của provider.

## Quy tắc chính

- Lưu API key scope, media URL lifetime, audit events trước khi trả thành công cho người dùng.
- Dùng API công khai và hợp đồng mô hình của AI Sonar; không phụ thuộc chi tiết thực thi của provider.
- Làm retry có tính idempotent và tránh tạo hoặc tính phí trùng lặp.
