## Cách định tuyến AI Sonar hoạt động

### Chiến lược định tuyến

| Chiến lược | Hành vi | Tốt nhất cho |
|-----------|---------|-------------|
| **PRIORITY** | Định tuyến đến nhà cung cấp ưu tiên cao nhất | Độ tin cậy |
| **COST** | Định tuyến đến nhà cung cấp rẻ nhất | Tối ưu chi phí |

### Dự phòng tự động

```
Yêu cầu: claude-sonnet-4-6
  → Provider A (chính): lỗi 503
  → Provider B (dự phòng): ✓ Thành công
```

### Định tuyến định dạng gốc

Khi model đã chọn và kênh định tuyến hỗ trợ, AI Sonar có thể định tuyến yêu cầu qua định dạng API gốc của gia đình model để tương thích tốt hơn:

| Họ mô hình | Định dạng API | URL cơ sở |
|-------------|------------|----------|
| Claude | Anthropic Messages | `https://api.aisonar.dev` |
| GPT | OpenAI Responses | `https://api.aisonar.dev/v1` |
| Gemini | Gemini Native API | `https://api.aisonar.dev` hoặc `https://api.aisonar.dev/v1beta` tùy theo đường dẫn của client |
| DeepSeek | OpenAI Chat | `https://api.aisonar.dev/v1` |

Claude Code là một đường Anthropic-native mạnh. Các API Gemini-native cũng tồn tại, nhưng bản thân Gemini CLI vẫn chỉ là một đường tương thích best-effort và không nên được xem là quy trình ổn định lâu dài.

## Cấu hình theo công cụ

<AccordionGroup>
  <Accordion title="Claude Code">
    ```bash
    export ANTHROPIC_API_KEY="sk-your-api-key"
    export ANTHROPIC_BASE_URL="https://api.aisonar.dev"
    ```
    Chọn mô hình: `claude --model claude-sonnet-4-6`

    [Hướng dẫn đầy đủ →](/vi/integrations/claude-code)
  </Accordion>

  <Accordion title="Cursor">
    Settings → Models:
    - API Key: `sk-your-api-key`
    - Base URL: `https://api.aisonar.dev/v1`

    [Hướng dẫn đầy đủ →](/vi/integrations/cursor)
  </Accordion>

  <Accordion title="Codex CLI">
    ```bash
    export OPENAI_API_KEY="sk-your-api-key"
    export OPENAI_BASE_URL="https://api.aisonar.dev/v1"
    ```
    [Hướng dẫn đầy đủ →](/vi/integrations/codex-cli)
  </Accordion>

  <Accordion title="Gemini CLI">
    ```bash
    export GEMINI_API_KEY="sk-your-api-key"
    export GOOGLE_GEMINI_BASE_URL="https://api.aisonar.dev"
    ```

    Hãy coi đây là đường tương thích best-effort chứ không phải hợp đồng upstream ổn định.
    [Hướng dẫn đầy đủ →](/vi/integrations/gemini-cli)
  </Accordion>

  <Accordion title="OpenCode">
    ```json
    {
      "provider": "openai",
      "apiKey": "sk-your-api-key",
      "baseURL": "https://api.aisonar.dev/v1"
    }
    ```
    [Hướng dẫn đầy đủ →](/vi/integrations/opencode)
  </Accordion>
