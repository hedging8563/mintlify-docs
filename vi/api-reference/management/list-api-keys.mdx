## Tổng Quan

Trả về danh sách API key của người dùng trong tổ chức gắn với token quản lý hiện tại.

## Ghi Chú

- Các key do hệ thống quản lý sẽ bị loại trừ.

## Ví Dụ

<RequestExample>
```bash cURL
curl -X GET "https://api.aisonar.dev/v1/management/api-keys" \
  -H "Authorization: Bearer mt-your-management-token"
```
</RequestExample>
