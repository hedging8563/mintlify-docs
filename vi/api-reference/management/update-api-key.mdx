## Tổng Quan

Dùng endpoint này để cập nhật tên, giới hạn sử dụng, model được phép, thời hạn hết hạn hoặc trạng thái của một API key người dùng hiện có.

## Thân Yêu Cầu

Phải cung cấp ít nhất một trường.

| Trường | Kiểu | Mặc định / Giới hạn | Ghi chú |
|-------|------|---------------------|------|
| `status` | enum | - | `active`, `inactive`, `suspended`, `revoked` |
| `name` | string | độ dài `1-50` | Tên hiển thị đã cập nhật |
| `limitAmount` | number \| null | tối thiểu `0`, input tối đa `1000000` | `null` = không giới hạn, `0` = quota bằng 0. Giá trị dương được chuẩn hoá thành trần lưu trữ không vượt quá tương đương `100000` USD |
| `limitCurrency` | enum | default `USD` | Chỉ hỗ trợ USD. Gửi `CNY` sẽ trả về `400 currency_retired`. |
| `models` | string[] | - | Danh sách cho phép model logic đã cập nhật |
| `expiresAt` | string \| null | datetime RFC3339 | `null` sẽ xoá thời hạn hết hạn |

## Ghi Chú

- Hard delete không được hỗ trợ trong Management API v1.
- `revoked` là trạng thái kết thúc đối với thao tác vận hành thông thường; hãy tạo key mới thay vì mong đợi có luồng khôi phục sau này.

## Ví Dụ

<RequestExample>
```bash cURL
curl -X PATCH "https://api.aisonar.dev/v1/management/api-keys/key_abc123def456" \
  -H "Authorization: Bearer mt-your-management-token" \
  -H "Content-Type: application/json" \
  -d '{
    "status": "inactive",
    "limitAmount": 0
  }'
```
</RequestExample>
