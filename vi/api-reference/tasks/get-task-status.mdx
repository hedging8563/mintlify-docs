## Tổng quan

Sử dụng endpoint này để polling async hợp nhất trên nhiều loại tác vụ như tạo video, hình ảnh, âm nhạc và 3D.

Nếu phản hồi create bao gồm `poll_url`, hãy gọi chính xác path đó. Đối với các workflow async mới hơn, `poll_url` có thể trỏ tới `/v1/tasks/{taskId}`.

## Tham số Đường dẫn

<ParamField path="taskId" type="string" required>
  ID tác vụ được trả về từ yêu cầu create.
</ParamField>

## Phản hồi

<ResponseField name="id" type="string">
  Định danh tác vụ.
</ResponseField>

<ResponseField name="status" type="string">
  Trạng thái tác vụ như `pending`, `processing`, `completed` hoặc `failed`. Tác vụ đã hủy được biểu diễn là `failed` với `cancelled: true`.
</ResponseField>

<ResponseField name="cancelled" type="boolean">
  `true` khi một async task đang chờ được hủy trước khi thực thi.
</ResponseField>

<ResponseField name="cancellation_status" type="string">
  Dấu hiệu hủy. Có giá trị `cancelled` khi hủy thành công.
</ResponseField>

<ResponseField name="data" type="array">
  Với tác vụ hình ảnh đã hoàn tất, kết quả hình ảnh được tạo sẽ được trả về tại đây. Tác vụ hình ảnh trả về URL trong `data[].url`.
</ResponseField>

<ResponseField name="video_url" type="string">
  URL asset kết quả khi tác vụ hoàn tất và tạo ra một video.
</ResponseField>

<ResponseField name="error" type="string">
  Thông báo lỗi khi tác vụ thất bại.
</ResponseField>

## Hành vi lỗi

Nếu tác vụ không còn tồn tại, đã hết hạn hoặc không thể được phân giải thông qua hợp đồng async-task công khai, AI Sonar sẽ trả về:

```json
{
  "error": {
    "message": "Task not found or no longer available.",
    "type": "invalid_request_error",
    "code": "async_task_not_found"
  }
}
```

## Ví dụ

<RequestExample>
```bash cURL
curl "https://api.aisonar.dev/v1/tasks/ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa" \
  -H "Authorization: Bearer sk-your-api-key"
```
</RequestExample>

```python Python
import requests

poll_url = "/v1/tasks/ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa"
response = requests.get(
    f"https://api.aisonar.dev{poll_url}",
    headers={"Authorization": "Bearer sk-your-api-key"},
)
print(response.json())
```

```javascript JavaScript
const pollUrl = '/v1/tasks/ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa';
const response = await fetch(`https://api.aisonar.dev${pollUrl}`, {
  headers: { Authorization: 'Bearer sk-your-api-key' },
});
console.log(await response.json());
```
