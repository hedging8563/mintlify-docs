## Phản hồi

<ResponseField name="object" type="string">
  Luôn là `list`.
</ResponseField>

<ResponseField name="data" type="array">
  Mảng các đối tượng model.

  Mỗi model bao gồm:
  - `id` (string): Định danh model
  - `object` (string): `model`
  - `created` (integer): Dấu thời gian tạo
  - `owned_by` (string): Nhà cung cấp model
  - `aisonar.category` (string): Danh mục model công khai
  - `aisonar.pricing_unit` (string hoặc `null`): Đơn vị giá công khai khi có
  - `aisonar.capability_flags` (object hoặc `null`): Tóm tắt năng lực công khai
  - `aisonar.supported_operations` (array hoặc `null`): Các thao tác phi-chat công khai của model
  - `aisonar.pricing_provenance` (object hoặc `null`): Metadata nguồn giá công khai
  - `aisonar.public_contract_summary` (object hoặc `null`): Tóm tắt khám phá phi-chat nhẹ, chỉ gồm `public_operations` và `request_endpoint`
  - `aisonar.agent_preferences` (object): Chỉ trả về khi có `recommended_for`
</ResponseField>

<Note>
`GET /v1/models` được tối ưu cho khám phá. Metadata chỉ có ở trang chi tiết như `capability_flags`, `pricing_provenance`, và `public_contract` đầy đủ nằm ở `GET /v1/models/{model}`.
</Note>

## Tham số Query

<ParamField query="category" type="string">
  Bộ lọc danh mục công khai tùy chọn. Hỗ trợ `chat`, `image`, `video`, `audio`, `tts`, `stt`, `music`, `3d`, `embedding`, `rerank`, và `translation`.
</ParamField>

<ParamField query="recommended_for" type="string">
  Kịch bản khuyến nghị phi-chat tùy chọn. Hỗ trợ `image`, `video`, `music`, `3d`, `tts`, `stt`, `embedding`, `rerank`, và `translation`.
</ParamField>

<ParamField query="provider" type="string">
  Bộ lọc nhà cung cấp tùy chọn như `openai`, `anthropic`, `google`, hoặc `deepseek`.
</ParamField>

<ParamField query="tag" type="string">
  Bộ lọc tag model tùy chọn như `chat`, `image`, `video`, `embedding`, hoặc `translation`.
</ParamField>

<Note>
Khi có `recommended_for`, `/v1/models` sắp xếp model phi-chat theo snapshot tỷ lệ thành công 24 giờ đã cache mới nhất. Model có `status = "insufficient_samples"` vẫn hiển thị nhưng xếp sau các model đã có điểm.
</Note>

```bash cURL
curl "https://api.aisonar.dev/v1/models" \
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.aisonar.dev/v1"
)

models = client.models.list()

for model in models.data:
    print(f"{model.id} ({model.owned_by})")
```

```go Go
package main

import (
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    req, _ := http.NewRequest("GET", "https://api.aisonar.dev/v1/models", nil)
    req.Header.Set("Authorization", "Bearer sk-your-api-key")

    resp, _ := http.DefaultClient.Do(req)
    defer resp.Body.Close()

    var payload struct {
        Data []struct {
            ID      string `json:"id"`
            OwnedBy string `json:"owned_by"`
        } `json:"data"`
    }

    json.NewDecoder(resp.Body).Decode(&payload)

    for _, model := range payload.Data {
        fmt.Printf("%s (%s)\n", model.ID, model.OwnedBy)
    }
}
```

## Lọc theo Nhà Cung Cấp

```python
# Get all OpenAI models
openai_models = [m for m in models.data if m.owned_by == "openai"]

# Get all Anthropic models
anthropic_models = [m for m in models.data if m.owned_by == "anthropic"]
```
