---
title: "Lấy thông tin Model"
openapi: "GET /v1beta/models/{model}"
description: "Lấy metadata của model bằng định dạng Google Gemini API"
---

Trả về metadata cho một model cụ thể theo định dạng Google Gemini API.

## Tham số đường dẫn

<ParamField path="model" type="string" required>
  Tên model (ví dụ: `gemini-2.5-pro`, `gemini-2.5-flash`). Các bí danh (aliases) cũng được chấp nhận.
</ParamField>

## Xác thực

Không bắt buộc với endpoint đọc metadata model công khai.

## Phản hồi

<ResponseField name="name" type="string">
  Tên tài nguyên model theo định dạng `models/{model}`.
</ResponseField>

<ResponseField name="displayName" type="string">
  Tên model hiển thị cho người dùng.
</ResponseField>

<ResponseField name="inputTokenLimit" type="integer">
  Giới hạn token đầu vào tối đa (cửa sổ ngữ cảnh).
</ResponseField>

<ResponseField name="outputTokenLimit" type="integer">
  Giới hạn token đầu ra tối đa.
</ResponseField>

<ResponseField name="supportedGenerationMethods" type="array">
  Danh sách các phương thức tạo được hỗ trợ (ví dụ: `generateContent`, `countTokens`, `embedContent`).
</ResponseField>

<ResponseField name="version" type="string">
  Chuỗi phiên bản công khai khi có.
</ResponseField>

<ResponseField name="description" type="string">
  Mô tả model dễ đọc cho người dùng khi có.
</ResponseField>

<ResponseField name="temperature" type="number">
  Giá trị temperature mặc định khi upstream công bố.
</ResponseField>

<ResponseField name="topP" type="number">
  Giá trị top-p mặc định khi có.
</ResponseField>

<ResponseField name="topK" type="integer">
  Giá trị top-k mặc định khi có.
</ResponseField>

<ResponseField name="maxTemperature" type="number">
  Temperature tối đa được hỗ trợ khi có.
</ResponseField>

## Lỗi

- Các GET vô tình chứa `:generateContent` hoặc `:streamGenerateContent` trả về `405 METHOD_NOT_ALLOWED`.
- Model không tồn tại trả về `404 NOT_FOUND`.

<RequestExample>
```bash cURL
curl "https://api.aisonar.dev/v1beta/models/gemini-2.5-pro"
```

```python Python
import google.generativeai as genai

genai.configure(
    api_key="sk-your-api-key",
    transport="rest",
    client_options={"api_endpoint": "api.aisonar.dev"}
)

model = genai.get_model("models/gemini-2.5-pro")
print(f"Name: {model.name}")
print(f"Input limit: {model.input_token_limit}")
print(f"Output limit: {model.output_token_limit}")
```
</RequestExample>
