## AI Sonar là gì?

AI Sonar là một nền tảng tổng hợp API AI cho phép bạn truy cập thống nhất tới **hàng trăm mô hình AI** từ các nhà cung cấp như OpenAI, Anthropic, Google, DeepSeek, Moonshot, MiniMax và nhiều nhà cung cấp khác.

## Các khả năng được hỗ trợ

| Khả năng | Mô tả | Ví dụ mô hình |
|------------|-------------|----------------|
| **Chat & Lập luận** | Sinh văn bản, lập kế hoạch, công cụ | `gpt-5.4`, `claude-sonnet-4-6`, `deepseek-r1` |
| **Thị giác** | Hiểu ảnh và chat đa phương thức | `gpt-4o`, `claude-sonnet-4-6`, `gemini-2.5-flash` |
| **Tạo ảnh** | Chuyển văn bản thành ảnh và chỉnh sửa ảnh | `dall-e-3`, `flux-pro`, `midjourney` |
| **Âm thanh** | TTS và chuyển âm thanh thành văn bản (transcription) | `gpt-4o-mini-tts`, `whisper-1`, `gpt-4o-transcribe` |
| **Nhúng** | Vector hóa văn bản | `text-embedding-3-small`, `text-embedding-3-large` |
| **Âm nhạc** | Sinh nhạc bằng AI | `suno` |
| **3D** | Tạo tài sản 3D | `tripo3d` |

## Chọn định dạng yêu cầu

Hãy bắt đầu bằng tuyến phù hợp với hành vi bạn cần. Dùng các tuyến native Anthropic/Gemini cho các tính năng native của nhà cung cấp; dùng `/v1/chat/completions` tương thích OpenAI khi đang chuyển đổi mã chat kiểu OpenAI hiện có.

<CodeGroup>

```bash cURL
curl https://api.aisonar.dev/v1/chat/completions \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-5.4",
    "messages": [
      {"role": "user", "content": "Explain what AI Sonar does in one sentence."}
    ]
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.aisonar.dev/v1"
)

response = client.chat.completions.create(
    model="gpt-5.4",
    messages=[{"role": "user", "content": "Explain what AI Sonar does in one sentence."}]
)

print(response.choices[0].message.content)
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.aisonar.dev/v1'
});

const response = await client.chat.completions.create({
  model: 'gpt-5.4',
  messages: [{ role: 'user', content: 'Explain what AI Sonar does in one sentence.' }],
});

console.log(response.choices[0].message.content);
```

<Note>
Chỉ sử dụng `POST /v1/responses` khi bạn thực sự cần hành vi đặc thù của Responses. Đây không phải là con đường tương thích mặc định cho tất cả các mô hình.
</Note>

</CodeGroup>
