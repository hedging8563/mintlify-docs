## 什么是 AI Sonar?

AI Sonar 是一个 AI API 聚合平台，提供对来自 OpenAI、Anthropic、Google、DeepSeek、Moonshot、MiniMax 等提供者的 **数百个 AI 模型** 的统一访问。

  </Card>
  <Card title="原生格式" icon="shuffle">
    在同一平台上调用 OpenAI、Anthropic、Gemini 和 Responses 风格的端点。
  </Card>

  </Card>
</CardGroup>

## 支持的功能

| 能力 | 描述 | 示例模型 |
|------------|-------------|----------------|
| **聊天与推理** | 文本生成、规划、工具 | `gpt-5.4`, `claude-sonnet-4-6`, `deepseek-r1` |
| **Vision** | 图像理解与多模态对话 | `gpt-4o`, `claude-sonnet-4-6`, `gemini-2.5-flash` |
| **图像生成** | 文本到图像及图像编辑 | `dall-e-3`, `flux-pro`, `midjourney` |
| **Audio** | TTS 与转录 | `gpt-4o-mini-tts`, `whisper-1`, `gpt-4o-transcribe` |
| **Embeddings** | 文本向量化 | `text-embedding-3-small`, `text-embedding-3-large` |
| **Music** | AI 音乐生成 | `suno` |
| **3D** | 3D 资产生成 | `tripo3d` |

## 先选择请求格式

从你需要的行为出发选择路由。需要提供商原生能力时使用 Anthropic/Gemini 原生路由；迁移已有 OpenAI 风格聊天代码时，使用 OpenAI 兼容的 `/v1/chat/completions`。

<CodeGroup>

```bash cURL
curl https://api.aisonar.dev/v1/chat/completions \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-5.4",
    "messages": [
      {"role": "user", "content": "Explain what AI Sonar does in one sentence."}
    ]
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.aisonar.dev/v1"
)

response = client.chat.completions.create(
    model="gpt-5.4",
    messages=[{"role": "user", "content": "Explain what AI Sonar does in one sentence."}]
)

print(response.choices[0].message.content)
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.aisonar.dev/v1'
});

const response = await client.chat.completions.create({
  model: 'gpt-5.4',
  messages: [{ role: 'user', content: 'Explain what AI Sonar does in one sentence.' }],
});

console.log(response.choices[0].message.content);
```

<Note>
仅在明确需要 Responses 特定行为时使用 `POST /v1/responses`。这并非所有模型的默认兼容路径。
</Note>

</CodeGroup>
