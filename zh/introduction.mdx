## 什么是 AI Sonar?

AI Sonar 是一个 AI API 聚合平台，提供对来自 OpenAI、Anthropic、Google、DeepSeek、Moonshot、MiniMax 等提供者的 **数百个 AI 模型** 的统一访问。

<CardGroup cols={2}>
  <Card title="原生 + 兼容" icon="plug">
    需要提供商原生行为时使用 Anthropic/Gemini 原生路由；已有 OpenAI 风格工具时使用 OpenAI 兼容的 `/v1` 路由。
  </Card>
  <Card title="原生格式" icon="shuffle">
    在同一平台上调用 OpenAI、Anthropic、Gemini 和 Responses 风格的端点。
  </Card>
  <Card title="按用量付费" icon="coins">

  </Card>
  <Card title="高可用性" icon="shield-check">
    智能路由、故障切换和多通道支持提升可靠性。
  </Card>
</CardGroup>

## 支持的功能

公开模型目录会随时间变化。请用 `/v1/models`、`recommended_for` 或模型页确认当前可用性；下面示例使用当前公开模型 ID。

| 能力 | 说明 | 示例模型 |
|---|---|---|
| 聊天与推理 | 文本生成、规划、工具调用 | `gpt-5.4`, `claude-sonnet-4-6`, `deepseek-r1` |
| 视觉与多模态 | 图像理解与多模态对话 | `gpt-4o`, `claude-sonnet-4-6`, `gemini-3.5-flash` |
| 图像生成与编辑 | 文生图与图像编辑 | `gpt-image-2`, `flux-pro`, `qwen-image` |
| 视频生成 | 文生视频、图生视频、参考视频与运动控制 | `veo3.1`, `seedance-2.0`, `kling-3.0-video` |
| 音频与 Realtime | TTS、转写、翻译和实时会话 | `gpt-4o-mini-tts`, `whisper-1`, `gpt-realtime` |
| 嵌入与 rerank | 文本向量化与重排序 | `text-embedding-3-small`, `qwen3-vl-rerank` |
| 翻译 | 文本与语音翻译流程 | `qwen-mt-plus`, `gpt-realtime-translate` |
| 音乐与 3D | 音乐和 3D 资产生成 | `suno-music`, `tripo3d-v2.5` |
| Worlds | World 生成、状态轮询和媒体资产流程 | 使用 `/v1/worlds/generations` 与 Models API 确认当前可用性 |

## 先选择请求格式

从你需要的行为出发选择路由。需要提供商原生能力时使用 Anthropic/Gemini 原生路由；迁移已有 OpenAI 风格聊天代码时，使用 OpenAI 兼容的 `/v1/chat/completions`。

<CodeGroup>

```bash cURL
curl https://api.aisonar.dev/v1/chat/completions \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-5.4",
    "messages": [
      {"role": "user", "content": "Explain what AI Sonar does in one sentence."}
    ]
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.aisonar.dev/v1"
)

response = client.chat.completions.create(
    model="gpt-5.4",
    messages=[{"role": "user", "content": "Explain what AI Sonar does in one sentence."}]
)

print(response.choices[0].message.content)
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.aisonar.dev/v1'
});

const response = await client.chat.completions.create({
  model: 'gpt-5.4',
  messages: [{ role: 'user', content: 'Explain what AI Sonar does in one sentence.' }],
});

console.log(response.choices[0].message.content);
```

<Note>
仅在明确需要 Responses 特定行为时使用 `POST /v1/responses`。这并非所有模型的默认兼容路径。
</Note>

</CodeGroup>
