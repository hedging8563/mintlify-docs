## 概述

以最小行为漂移从 OpenAI、Anthropic、Gemini、OpenRouter、LiteLLM 和媒体工作流迁移到 AI Sonar。

使用 AI Sonar 公开 API 和模型契约；不要依赖供应商执行细节。

## 关键规则

- 在向用户返回成功前持久化 old model ID, new model ID, rollout owner。
- 使用 AI Sonar 公开 API 和模型契约；不要依赖供应商执行细节。
- 让重试保持幂等，避免重复生成或重复计费。

## 生产检查清单

- 在向用户返回成功前持久化 old model ID, new model ID, rollout owner。
- 使用 AI Sonar 公开 API 和模型契约；不要依赖供应商执行细节。
- 让重试保持幂等，避免重复生成或重复计费。
- 日志要便于支持排障，但不要存储密钥或私有 payload。
- 生产流量变更前先在预发环境验证行为。
