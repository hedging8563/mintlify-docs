## 概述

围绕模型可用性、价格变化、别名和供应商能力漂移做计划。

使用 AI Sonar 公开 API 和模型契约；不要依赖供应商执行细节。

## 关键规则

- 在向用户返回成功前持久化 model ID, price snapshot, replacement model。
- 使用 AI Sonar 公开 API 和模型契约；不要依赖供应商执行细节。
- 让重试保持幂等，避免重复生成或重复计费。

## 生产检查清单

- 在向用户返回成功前持久化 model ID, price snapshot, replacement model。

- 生产流量变更前先在预发环境验证行为。
