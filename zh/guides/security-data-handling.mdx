## 概述

在 AI Sonar 集成中安全处理 API key、用户媒体、日志和生成资产。

使用 AI Sonar 公开 API 和模型契约；不要依赖供应商执行细节。

## 关键规则

- 在向用户返回成功前持久化 API key scope, media URL lifetime, audit events。
- 使用 AI Sonar 公开 API 和模型契约；不要依赖供应商执行细节。
- 让重试保持幂等，避免重复生成或重复计费。

## 生产检查清单

- 在向用户返回成功前持久化 API key scope, media URL lifetime, audit events。
- 使用 AI Sonar 公开 API 和模型契约；不要依赖供应商执行细节。
- 让重试保持幂等，避免重复生成或重复计费。
- 日志要便于支持排障，但不要存储密钥或私有 payload。
- 生产流量变更前先在预发环境验证行为。
