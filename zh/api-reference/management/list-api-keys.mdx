---

## 概览

返回当前管理令牌所属组织下的用户 API Key 列表。

## 注意事项

- 系统托管的 Key 不会出现在结果中。

## 示例

<RequestExample>
```bash cURL
curl -X GET "https://api.aisonar.dev/v1/management/api-keys" \
  -H "Authorization: Bearer mt-your-management-token"
```
</RequestExample>

## 请求 / 响应

使用上面的交互式 OpenAPI 面板获取确切的模式。

## 响应示例

<ResponseExample>
```json 200 OK
{
  "object": "list",
  "data": [{
    "id": "ak_abc123",
    "name": "Production app",
    "status": "active",
    "created_at": "2026-06-27T00:00:00.000Z",
    "last_used_at": "2026-06-27T01:00:00.000Z"
  }]
}
```
</ResponseExample>

## 重要字段

<ResponseField name="data" type="array">本次响应返回的列表或载荷。</ResponseField>
<ResponseField name="status" type="string">该资源当前的生命周期状态。</ResponseField>
