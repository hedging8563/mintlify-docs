---
title: "列出模型"
openapi: "GET /v1beta/models"
description: "使用 Google Gemini API 格式列出可用模型"
---

返回以 Google Gemini API 格式表示的可用模型列表。

## 查询参数

<ParamField query="pageSize" type="integer">
  要返回的模型最大数量。默认值：`1000`，最大值：`1000`。
</ParamField>

<ParamField query="pageToken" type="string">
  上一页返回的不透明续页令牌。将它原样传回即可获取下一页。
</ParamField>

## 身份验证

可选。支持与其他 Gemini 端点相同的身份验证方法：
- `?key=YOUR_API_KEY` 查询参数
- `x-goog-api-key: YOUR_API_KEY` 请求头
- `Authorization: Bearer YOUR_API_KEY` 请求头

## 响应

<ResponseField name="models" type="array">
  模型对象数组。
</ResponseField>

<ResponseField name="nextPageToken" type="string">
  当仍有更多模型时返回。将该值作为 `pageToken` 传回即可继续翻页。
</ResponseField>

## 错误

- 无效的 `pageToken` 会返回 `400 INVALID_ARGUMENT`。
- 非预期的列表失败会返回 `500 INTERNAL`。

<RequestExample>
```bash cURL
curl "https://api.aisonar.dev/v1beta/models?pageSize=5" \
  -H "x-goog-api-key: sk-your-api-key"
```

```python Python
import google.generativeai as genai

genai.configure(
    api_key="sk-your-api-key",
    transport="rest",
    client_options={"api_endpoint": "api.aisonar.dev"}
)

for model in genai.list_models():
    print(model.name)
```

```javascript JavaScript
const response = await fetch(
  "https://api.aisonar.dev/v1beta/models?pageSize=5",
  { headers: { "x-goog-api-key": "sk-your-api-key" } }
);
const { models } = await response.json();
models.forEach(m => console.log(m.name));
```
</RequestExample>
