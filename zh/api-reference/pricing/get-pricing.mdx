title: "获取价格"
openapi: "GET /v1/pricing"
description: "获取模型的定价信息"
---

## 概述

使用此端点可以列出当前对外可见的活跃模型定价信息。

如需查看某个模型的完整定价详情，包括结构化定价和溯源信息，请优先使用 [获取模型定价](/api-reference/models/get-model)。

## 查询参数

<ParamField query="provider" type="string">
  可选的提供方筛选，例如 `openai`、`anthropic` 或 `google`。
</ParamField>

<ParamField query="tag" type="string">
  可选的模型标签筛选，例如 `chat`、`image`、`video`、`embedding` 或 `translation`。
</ParamField>

## 响应

<ResponseField name="object" type="string">
  始终为 `pricing`。
</ResponseField>

<ResponseField name="updated_at" type="string">
  响应中包含的最新定价条目的 ISO 时间戳。
</ResponseField>

<ResponseField name="currency" type="string">
  始终为 `USD`。
</ResponseField>

<ResponseField name="data" type="array">
  定价条目数组。每一项包括 `model`、`provider`、`is_lock_price`、`capabilities`、`aliases`，以及上方交互式 schema 返回的公开标量或结构化定价摘要字段。
</ResponseField>
