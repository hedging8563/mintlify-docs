---
title: "Listar modelos"
openapi: "GET /v1beta/models"
description: "Lista los modelos disponibles usando el formato de Google Gemini API"
---

Devuelve una lista de modelos disponibles en formato de Google Gemini API.

## Parámetros de consulta

<ParamField query="pageSize" type="integer">
  Número máximo de modelos a devolver. Predeterminado: `1000`, máximo: `1000`.
</ParamField>

<ParamField query="pageToken" type="string">
  Token de continuación devuelto por una página anterior. Envíalo tal cual para obtener el siguiente bloque.
</ParamField>

## Autenticación

La autenticación no es obligatoria para los endpoints públicos de lectura del catálogo de modelos.

Los indicios de autenticación al estilo Gemini, como `?key=` o `x-goog-api-key`, no cambian el contrato de respuesta aquí.

## Respuesta

<ResponseField name="models" type="array">
  Array de objetos de modelo.
</ResponseField>

<ResponseField name="nextPageToken" type="string">
  Se devuelve cuando hay más modelos disponibles. Pasa este valor como `pageToken` para continuar la paginación.
</ResponseField>

## Errores

- Un `pageToken` inválido devuelve `400 INVALID_ARGUMENT`.
- Los fallos inesperados de listado devuelven `500 INTERNAL`.

<RequestExample>
```bash cURL
curl "https://api.aisonar.dev/v1beta/models?pageSize=5" \
  -H "x-goog-api-key: sk-your-api-key"
```

```python Python
import google.generativeai as genai

genai.configure(
    api_key="sk-your-api-key",
    transport="rest",
    client_options={"api_endpoint": "api.aisonar.dev"}
)

for model in genai.list_models():
    print(model.name)
```

```javascript JavaScript
const response = await fetch(
  "https://api.aisonar.dev/v1beta/models?pageSize=5",
  { headers: { "x-goog-api-key": "sk-your-api-key" } }
);
const { models } = await response.json();
models.forEach(m => console.log(m.name));
```
</RequestExample>
