---

## Resumen

Este endpoint devuelve los totales actuales de saldo de la organización asociada al token de gestión. Usa la misma fuente de verdad de saldo de organización que la facturación y la liquidación.

## Notas

- Los campos monetarios de verdad se devuelven en USD: `balance`, `total_recharge` y `total_used`.
- `preferred_currency` refleja la preferencia de visualización del Dashboard de la organización; no cambia los campos de verdad en USD.

## Ejemplo

<RequestExample>
```bash cURL
curl -X GET "https://api.aisonar.dev/v1/management/balance" \
  -H "Authorization: Bearer mt-your-management-token"
```
</RequestExample>

## Solicitud / Respuesta

Utiliza el panel interactivo de OpenAPI arriba para el esquema exacto.

## Ejemplo de respuesta

<ResponseExample>
```json 200 OK
{
  "object": "management.balance",
  "currency": "USD",
  "available_balance": "42.50",
  "pending_usage": "1.25",
  "updated_at": "2026-06-27T00:00:00.000Z"
}
```
</ResponseExample>

## Campos importantes

<ResponseField name="available_balance" type="string">Cadena decimal en USD para conciliación de facturación.</ResponseField>
<ResponseField name="pending_usage" type="string">Cadena decimal en USD para conciliación de facturación.</ResponseField>
<ResponseField name="updated_at" type="string">Marca de tiempo ISO o Unix devuelta por la API.</ResponseField>
