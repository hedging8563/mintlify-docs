---
title: "Obtener facturación de clave API"
openapi: "GET /v1/management/api-keys/{keyId}/billing"
description: "Devuelve desgloses agregados de facturación para una clave API de usuario."
---

## Resumen

Este endpoint devuelve agregados de facturación basados en consumo para la clave API seleccionada, incluyendo coste total, solicitudes totales, uso por unidad de facturación y desgloses por nombre de modelo solicitado (`model`), proveedor, `scene`, `accessChannel`, clase de caché y tendencia diaria de coste.

## Notas

- Las recargas, facturas y otros registros financieros a nivel de organización quedan excluidos.

## Ejemplo

<RequestExample>
```bash cURL
curl -X GET "https://api.aisonar.dev/v1/management/api-keys/key_abc123def456/billing?startDate=2026-01-01&endDate=2026-01-31" \
  -H "Authorization: Bearer mt-your-management-token"
```
</RequestExample>

## Solicitud / Respuesta

Utiliza el panel interactivo de OpenAPI arriba para el esquema exacto.

## Ejemplo de respuesta

<ResponseExample>
```json 200 OK
{
  "object": "management.billing",
  "api_key_id": "ak_abc123",
  "currency": "USD",
  "total_cost": "12.3400",
  "data": [{
    "billing_transaction_id": "billtxn_abc123",
    "request_id": "req_abc123",
    "amount": "0.0012",
    "created_at": "2026-06-27T00:00:00.000Z"
  }]

```
</ResponseExample>

## Campos importantes

<ResponseField name="total_cost" type="string">Cadena decimal en USD para conciliación de facturación.</ResponseField>
<ResponseField name="data" type="array">Lista o payload devuelto por esta respuesta.</ResponseField>
<ResponseField name="billing_transaction_id" type="string">Identificador de liquidación de facturación cuando está disponible.</ResponseField>
