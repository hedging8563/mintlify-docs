---

## Descripción general

Usa este endpoint para un sondeo async unificado entre tipos de tareas como generación de video, imagen, música y 3D.

Si una respuesta de creación incluye `poll_url`, llama a esa ruta exacta. Para flujos de trabajo async más recientes, `poll_url` puede apuntar a `/v1/tasks/{taskId}`.

## Parámetros de ruta

<ParamField path="taskId" type="string" required>
  El ID de la tarea devuelto por la solicitud de creación.
</ParamField>

## Respuesta

<ResponseField name="id" type="string">
  Identificador de la tarea.
</ResponseField>

<ResponseField name="status" type="string">
  Estado de la tarea, como `pending`, `processing`, `completed` o `failed`. Las tareas canceladas se representan como `failed` con `cancelled: true`.
</ResponseField>

<ResponseField name="cancelled" type="boolean">
  `true` cuando una tarea asíncrona en cola se canceló antes de la ejecución.
</ResponseField>

<ResponseField name="cancellation_status" type="string">
  Marcador de cancelación. Presente como `cancelled` cuando la cancelación se realiza correctamente.
</ResponseField>

<ResponseField name="data" type="array">
  Para tareas de imagen completadas, los resultados generados se devuelven aquí. Las tareas de imagen devuelven URL en `data[].url`.
</ResponseField>

<ResponseField name="video_url" type="string">
  URL del recurso resultante cuando la tarea se completa y produce un video.
</ResponseField>

<ResponseField name="error" type="string">
  Mensaje de error cuando la tarea falla.
</ResponseField>

## Comportamiento de error

Si la tarea ya no existe, ha expirado o no puede resolverse mediante el contrato público de tareas async, AI Sonar devuelve:

```json
{
  "error": {
    "message": "Task not found or no longer available.",
    "type": "invalid_request_error",
    "code": "async_task_not_found"
  }
}
```

## Ejemplos

<RequestExample>
```bash cURL
curl "https://api.aisonar.dev/v1/tasks/ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa" \
  -H "Authorization: Bearer sk-your-api-key"
```
</RequestExample>

```python Python
import requests

poll_url = "/v1/tasks/ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa"
response = requests.get(
    f"https://api.aisonar.dev{poll_url}",
    headers={"Authorization": "Bearer sk-your-api-key"},
)
print(response.json())
```

```javascript JavaScript
const pollUrl = '/v1/tasks/ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa';
const response = await fetch(`https://api.aisonar.dev${pollUrl}`, {
  headers: { Authorization: 'Bearer sk-your-api-key' },
});
console.log(await response.json());
```
