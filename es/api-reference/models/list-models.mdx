title: "Listar modelos"
openapi: "GET /v1/models"

---

## Resumen

`GET /v1/models` está optimizado para el descubrimiento. Los metadatos solo de detalle, como `capability_flags`, `pricing_provenance` y el `public_contract` completo, viven en `GET /v1/models/{model}`.

## Parámetros de consulta

<ParamField query="category" type="string">
  Filtro opcional por categoría pública. Admite `chat`, `image`, `video`, `audio`, `tts`, `stt`, `music`, `3d`, `embedding`, `rerank` y `translation`.
</ParamField>

<ParamField query="recommended_for" type="string">
  Escena opcional de recomendación no chat. Admite `image`, `video`, `music`, `3d`, `tts`, `stt`, `embedding`, `rerank` y `translation`.
</ParamField>

<ParamField query="provider" type="string">
  Filtro opcional por proveedor, por ejemplo `openai`, `anthropic`, `google` o `deepseek`.
</ParamField>

<ParamField query="tag" type="string">
  Filtro opcional por etiqueta de modelo, por ejemplo `chat`, `image`, `video`, `embedding` o `translation`.
</ParamField>

<Note>
Cuando `recommended_for` está presente, `/v1/models` ordena los modelos no chat según la instantánea más reciente de éxito en 24 horas almacenada en caché. Los modelos con `status = "insufficient_samples"` siguen visibles, pero se muestran después de los modelos con puntuación.
</Note>

## Respuesta

<ResponseField name="object" type="string">
  Siempre `list`.
</ResponseField>

<ResponseField name="data" type="array">
  Array de objetos de modelo.

  Cada modelo contiene:
  - `id` (string): Identificador del modelo
  - `object` (string): `model`
  - `created` (integer): Marca temporal de creación
  - `owned_by` (string): Proveedor del modelo
  - `aisonar.category` (string): Categoría pública del modelo
  - `aisonar.pricing_unit` (string or `null`): Unidad de precio pública cuando está disponible
  - `aisonar.capability_flags` (object or `null`): Resumen público de capacidades
  - `aisonar.supported_operations` (array or `null`): Operaciones públicas no chat compatibles con el modelo
  - `aisonar.pricing_provenance` (object or `null`): Metadatos públicos de procedencia de precios
  - `aisonar.public_contract_summary` (object or `null`): Resumen ligero de descubrimiento no chat con solo `public_operations` y `request_endpoint`
  - `aisonar.agent_preferences` (object): Solo se devuelve cuando existe `recommended_for`
</ResponseField>

```go Go
package main

import (
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    req, _ := http.NewRequest("GET", "https://api.aisonar.dev/v1/models", nil)
    req.Header.Set("Authorization", "Bearer sk-your-api-key")

    resp, _ := http.DefaultClient.Do(req)
    defer resp.Body.Close()

    var payload struct {
        Data []struct {
            ID      string `json:"id"`
            OwnedBy string `json:"owned_by"`
        } `json:"data"`
    }

    json.NewDecoder(resp.Body).Decode(&payload)

    for _, model := range payload.Data {
        fmt.Printf("%s (%s)\n", model.ID, model.OwnedBy)
    }
}
```

## Filtrado por proveedor

```python
# Obtener todos los modelos de OpenAI
openai_models = [m for m in models.data if m.owned_by == "openai"]

# Obtener todos los modelos de Anthropic
anthropic_models = [m for m in models.data if m.owned_by == "anthropic"]
```
