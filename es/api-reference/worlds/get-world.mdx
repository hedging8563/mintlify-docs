---
title: "Obtener World"
openapi: "GET /v1/worlds/{id}"
description: "Obtiene un World de World Labs Marble completado"
---

Obtén un Marble World completado creado mediante AI Sonar. El world debe pertenecer a la misma organización que la clave API.

## Parámetros de ruta

<ParamField path="id" type="string" required>
  ID de World Labs world.
</ParamField>

## Respuesta

<ResponseField name="world_id" type="string">
  ID de World Labs world.
</ResponseField>

<ResponseField name="world_marble_url" type="string">
  World Labs Marble URL.
</ResponseField>

<ResponseField name="assets" type="object">
  Activos de World generados cuando están disponibles.
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.aisonar.dev/v1/worlds/world_123" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
import requests

response = requests.get(
    "https://api.aisonar.dev/v1/worlds/world_123",
    headers={"Authorization": "Bearer sk-your-api-key"},
)
print(response.json()["world_id"])
```

```javascript JavaScript
const response = await fetch('https://api.aisonar.dev/v1/worlds/world_123', {
  headers: { Authorization: 'Bearer sk-your-api-key' }
});
console.log(await response.json());
```
</RequestExample>
