---
title: "Eliminar World"
openapi: "DELETE /v1/worlds/{id}"
description: "Elimina un World de World Labs Marble"
---

Elimina un Marble World creado mediante AI Sonar. AI Sonar reenvía la eliminación a World Labs y marca el registro local como eliminado.

La eliminación es permanente.

## Parámetros de ruta

<ParamField path="id" type="string" required>
  ID de World Labs world.
</ParamField>

## Respuesta

<ResponseField name="deleted" type="boolean">
  Si la eliminación fue aceptada.
</ResponseField>

<ResponseField name="world_id" type="string">
  ID de World Labs world.
</ResponseField>

<RequestExample>
```bash cURL
curl -X DELETE "https://api.aisonar.dev/v1/worlds/world_123" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
import requests

response = requests.delete(
    "https://api.aisonar.dev/v1/worlds/world_123",
    headers={"Authorization": "Bearer sk-your-api-key"},
)
print(response.json())
```

```javascript JavaScript
const response = await fetch('https://api.aisonar.dev/v1/worlds/world_123', {
  method: 'DELETE',
  headers: { Authorization: 'Bearer sk-your-api-key' }
});
console.log(await response.json());
```
</RequestExample>
