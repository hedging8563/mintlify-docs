---
title: "Preparar subida de medio"
openapi: "POST /v1/worlds/media-assets/prepare-upload"
description: "Crea un destino de subida para un medio de World Labs"
---

Crea un registro de medio de World Labs y recibe una URL de subida para una imagen o video. Sube el binario a `upload_info.upload_url` y luego referencia `media_asset.media_asset_id` desde `world_prompt` con `source: "media_asset"`.

AI Sonar registra el propietario del medio. Solo la misma organización puede recuperar o usar este medio en la generación de Worlds.

## Cuerpo de la solicitud

<ParamField body="file_name" type="string" required>
  Nombre de archivo original, hasta 64 caracteres.
</ParamField>

<ParamField body="kind" type="string" required>
  Tipo de medio: `image` o `video`.
</ParamField>

<ParamField body="extension" type="string">
  Extensión opcional como `png`, `jpg` o `mp4`.
</ParamField>

<ParamField body="metadata" type="object">
  Objeto opcional de metadatos guardado con el medio upstream.
</ParamField>

## Respuesta

<ResponseField name="media_asset" type="object">
  Metadatos del medio de World Labs preparado.
</ResponseField>

<ResponseField name="upload_info.upload_url" type="string">
  URL upstream firmada donde subir el archivo binario.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.aisonar.dev/v1/worlds/media-assets/prepare-upload" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "file_name": "reference.png",
    "kind": "image",
    "extension": "png"
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.aisonar.dev/v1/worlds/media-assets/prepare-upload",
    headers={"Authorization": "Bearer sk-your-api-key"},
    json={"file_name": "reference.png", "kind": "image", "extension": "png"},
)

print(response.json()["media_asset"]["media_asset_id"])
```

```javascript JavaScript
const response = await fetch('https://api.aisonar.dev/v1/worlds/media-assets/prepare-upload', {
  method: 'POST',
  headers: {
    Authorization: 'Bearer sk-your-api-key',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({ file_name: 'reference.png', kind: 'image', extension: 'png' })
});

const prepared = await response.json();
console.log(prepared.media_asset.media_asset_id, prepared.upload_info.upload_url);
```
