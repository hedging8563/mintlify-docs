---
title: "Obtener estado del World"
openapi: "GET /v1/worlds/generations/{id}"
description: "Recupera el estado y resultado de una tarea de generación de mundo World Labs Marble"
---

Consulta este endpoint para comprobar el estado de una tarea de generación World Labs Marble. Si la respuesta de creación incluyó `poll_url`, prefiere llamar esa URL exacta.

## Parámetros de ruta

<ParamField path="id" type="string" required>
  ID de la tarea de generación de mundo.
</ParamField>

## Respuesta

<ResponseField name="id" type="string">
  ID público de tarea.
</ResponseField>

<ResponseField name="task_id" type="string">
  Alias del identificador de tarea asíncrona.
</ResponseField>

<ResponseField name="operation_id" type="string">
  ID de operation upstream de World Labs.
</ResponseField>

<ResponseField name="status" type="string">
  Estado de la tarea: `pending`, `processing`, `completed`, `failed` o `timeout`.
</ResponseField>

<ResponseField name="world_id" type="string">
  World ID upstream de World Labs cuando finaliza.
</ResponseField>

<ResponseField name="world_marble_url" type="string">
  URL del mundo Marble generado cuando finaliza.
</ResponseField>

<ResponseField name="model_url" type="string">
  Alias de la URL principal del mundo/modelo generado.
</ResponseField>

<ResponseField name="glb_url" type="string">
  URL GLB de la malla collider si está disponible.
</ResponseField>

<ResponseField name="pano_url" type="string">
  URL de imagen panorámica si está disponible.
</ResponseField>

<ResponseField name="thumbnail_url" type="string">
  URL de miniatura si está disponible.
</ResponseField>

<ResponseField name="caption" type="string">
  Caption upstream del mundo generado.
</ResponseField>

<ResponseField name="cost" type="object">
  Detalles de coste de la operation upstream si están disponibles, incluido `total_credits`.
</ResponseField>

<ResponseField name="error" type="string">
  Mensaje de error cuando la tarea falla.
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.aisonar.dev/v1/worlds/generations/ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
import requests
import time

task_id = "ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa"

while True:
    response = requests.get(
        f"https://api.aisonar.dev/v1/worlds/generations/{task_id}",
        headers={"Authorization": "Bearer sk-your-api-key"},
    )
    result = response.json()
    if result["status"] in ["completed", "failed", "timeout"]:
        print(result)
        break
    time.sleep(10)
```

