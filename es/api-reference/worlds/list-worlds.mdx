---
title: "Listar Worlds"
openapi: "POST /v1/worlds/list"
description: "Lista Worlds de World Labs Marble completados creados mediante AI Sonar"
---

Lista los World Labs Marble worlds completados creados mediante AI Sonar. Los resultados se limitan a tu organización e incluyen solo recursos creados mediante AI Sonar.

Usa `/v1/worlds/generations/{id}` para tareas pendientes o fallidas. Esta lista devuelve solo Worlds completados que AI Sonar indexó después de la generación.

## Cuerpo de la solicitud

<ParamField body="page_size" type="integer" default="20">
  Cantidad de worlds a devolver, de 1 a 100.
</ParamField>

<ParamField body="page_token" type="string">
  Token de paginación de la respuesta anterior.
</ParamField>

<ParamField body="status" type="string">
  Filtro opcional de estado. AI Sonar devuelve worlds completados para `SUCCEEDED`; otros estados pertenecen al endpoint de tarea de generación.
</ParamField>

<ParamField body="model" type="string">
  Filtro opcional de modelo Marble.
</ParamField>

<ParamField body="tags" type="array">
  Filtro opcional de etiquetas.
</ParamField>

<ParamField body="is_public" type="boolean">
  Filtro opcional de visibilidad.
</ParamField>

<ParamField body="created_after" type="string">
  Solo devolver worlds creados después de esta marca de tiempo.
</ParamField>

<ParamField body="created_before" type="string">
  Solo devolver worlds creados antes de esta marca de tiempo.
</ParamField>

<ParamField body="sort_by" type="string" default="created_at">
  Ordenar por `created_at` o `updated_at`.
</ParamField>

<ParamField body="sort_order" type="string" default="desc">
  Dirección de orden: `asc` o `desc`.
</ParamField>

## Respuesta

<ResponseField name="worlds" type="array">
  Array de registros world.
</ResponseField>

<ResponseField name="next_page_token" type="string">
  Token de la página siguiente o `null`.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.aisonar.dev/v1/worlds/list" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{"page_size": 20, "status": "SUCCEEDED", "model": "marble-1.1"}'
```

```python Python
import requests

response = requests.post(
    "https://api.aisonar.dev/v1/worlds/list",
    headers={"Authorization": "Bearer sk-your-api-key"},
    json={"page_size": 20, "status": "SUCCEEDED", "model": "marble-1.1"},
)
print(response.json()["worlds"])
```
