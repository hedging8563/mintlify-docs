---
title: "Obtener estado de la música"
openapi: "GET /v1/tasks/{id}"
description: "Recupera el estado y el resultado de una tarea de generación de música"
---

<span data-mintlify-rebuild="2026-05-19-after-mdx-parse-fix" aria-hidden="true" />

Realice polling a este endpoint para verificar el estado de su tarea de generación de música.

<Info>
Si la respuesta de creación incluye `poll_url`, prioriza esa URL exacta para el polling. `id` y `task_id` deben tratarse como la misma identidad de tarea asíncrona.


Si la tarea ya no existe o ya no se puede resolver mediante el contrato público de async-task, AI Sonar devuelve `async_task_not_found` con el mensaje `Task not found or no longer available.`
</Info>

## Parámetros de ruta

<ParamField path="id" type="string" required>
  El ID de la tarea de generación de música.
</ParamField>

## Respuesta

<ResponseField name="id" type="string">
  ID de la tarea.
</ResponseField>

<ResponseField name="task_id" type="string">
  Alias del identificador de tarea asíncrona cuando está disponible.
</ResponseField>

<ResponseField name="status" type="string">
  Estado de la tarea: `pending`, `processing`, `completed` o `failed`.
</ResponseField>

<ResponseField name="progress" type="integer">
  Porcentaje de progreso (0-100).
</ResponseField>

<ResponseField name="audio_url" type="string">
  URL para descargar el archivo de audio (cuando se haya completado).
</ResponseField>

<ResponseField name="video_url" type="string">
  URL de la versión en video con visualizador (cuando se haya completado).
</ResponseField>

<ResponseField name="title" type="string">
  Título de la canción generada.
</ResponseField>

<ResponseField name="lyrics" type="string">
  Letras generadas o proporcionadas.
</ResponseField>

<ResponseField name="error" type="string">
  Mensaje de error (cuando falle).
</ResponseField>

```python Python
import requests
import time

task_id = "ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa"

while True:
    response = requests.get(
        f"https://api.aisonar.dev/v1/tasks/{task_id}",
        headers={"Authorization": "Bearer sk-your-api-key"}
    )
    result = response.json()

    if result["status"] == "completed":
        print(f"Audio URL: {result['audio_url']}")
        break
    elif result["status"] == "failed":
        print(f"Error: {result['error']}")
        break

    time.sleep(5)  # Poll every 5 seconds
```

```javascript JavaScript
const taskId = 'ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa';

async function pollForMusic() {
  while (true) {
    const response = await fetch(
      `https://api.aisonar.dev/v1/tasks/${taskId}`,
      { headers: { 'Authorization': 'Bearer sk-your-api-key' } }
    );

    const result = await response.json();

    if (result.status === 'completed') {
      console.log(`Audio URL: ${result.audio_url}`);
      return result.audio_url;
    } else if (result.status === 'failed') {
      throw new Error(result.error);
    }

    await new Promise(r => setTimeout(r, 5000));
  }
}
```

```go Go
package main

import (
    "encoding/json"
    "fmt"
    "net/http"
    "time"
)

func main() {
    taskID := "ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa"

    for {
        req, _ := http.NewRequest("GET",
            fmt.Sprintf("https://api.aisonar.dev/v1/tasks/%s", taskID), nil)
        req.Header.Set("Authorization", "Bearer sk-your-api-key")

        client := &http.Client{}
        resp, _ := client.Do(req)

        var result map[string]interface{}
        json.NewDecoder(resp.Body).Decode(&result)
        resp.Body.Close()

        if result["status"] == "completed" {
            fmt.Printf("Audio URL: %s\n", result["audio_url"])
            break
        } else if result["status"] == "failed" {
            fmt.Printf("Error: %s\n", result["error"])
            break
        }

        time.Sleep(5 * time.Second)
    }
}
```

```php PHP

$taskId = 'ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa';

while (true) {

        break;
    } elseif ($result['status'] === 'failed') {
        echo "Error: {$result['error']}\n";
        break;
    }

    sleep(5);
}
```
