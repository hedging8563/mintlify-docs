## Descripción general

Usa JSON Schema, tool calls y capacidades nativas en rutas compatibles con OpenAI, Anthropic y Gemini.

Usa las API públicas de AI Sonar y contratos de modelo; no dependas de detalles de ejecución del proveedor.

## Reglas clave

- Persiste los schema version, tool name, validated arguments antes de devolver éxito al usuario.
- Usa las API públicas de AI Sonar y contratos de modelo; no dependas de detalles de ejecución del proveedor.
- Haz que los reintentos sean idempotentes y evita generación o cobro duplicado.
