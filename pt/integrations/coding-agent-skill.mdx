## O que o skill faz

- Gera exemplos mínimos executáveis para chat, imagem, áudio, vídeo, tradução e outras famílias de API da AI Sonar.
- Usa `https://api.aisonar.dev/v1` para clientes compatíveis com OpenAI e explica quando mudar para rotas nativas Anthropic ou Gemini.
- Descobre modelos com `/v1/models`, `/llms.txt` e listas `recommended_for`, em vez de adivinhar a partir de listas antigas.
- Lê o contrato público de um modelo antes de tentar novamente solicitações não-chat, para não remover campos incompatíveis silenciosamente.
- Trata dicas de erro Agent-First como `did_you_mean`, `suggestions`, `retry_after` e `recommended_request`.

## Instalação

<Note>
Use o comando canônico de instalação não interativa:

```bash
npx skills add https://github.com/hedging8563/aisonar-skills --skill aisonar-api-integration -y
```

Isso instala o skill compartilhado `aisonar-api-integration` a partir do repositório AI Sonar skills.

Se sua ferramenta não suporta o installer, copie `skills/aisonar-api-integration/` do repositório para o diretório compartilhado de skills ou rules da ferramenta.
</Note>

### Atualizar instalações existentes

Se você instalou o skill antes da limpeza do repositório AI Sonar, execute o mesmo comando novamente. O pacote público atual é menor e não depende mais de scripts locais de busca gerados.

## Fluxo recomendado para Agent

1. Comece pelo menor exemplo funcional que corresponda à linguagem e à família de API solicitadas.
2. Quando a escolha do modelo estiver em aberto, chame `/v1/models` ou `https://api.aisonar.dev/llms.txt` antes de fixar um modelo.
3. Para trabalho não-chat, chame `/v1/models?recommended_for=<scene>` onde `<scene>` é `image`, `video`, `music`, `3d`, `tts`, `stt`, `embedding`, `rerank` ou `translation`.
4. Antes de alterar uma solicitação não-chat com falha, leia `/v1/models/:model` e alinhe `supported_operations`, `supported_parameters`, `request_endpoint`, `request_shape_mode` e `recommended_request`.
5. Se a API retornar um erro Agent-First, use os campos estruturados para corrigir a solicitação e tente novamente apenas se o erro for retryable.

## Exemplo mínimo de chat

Este exemplo usa o SDK Python da OpenAI com a base URL da AI Sonar:

```python
import os
from openai import OpenAI

client = OpenAI(
    api_key=os.environ["TOKENLAB_API_KEY"],
    base_url="https://api.aisonar.dev/v1"
)

response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)
```

Execute com:

```bash
pip install openai
export TOKENLAB_API_KEY="sk-your-api-key"
python app.py
```

## Descoberta de modelos e contratos

Prefira descoberta ao vivo em vez de listas antigas empacotadas:

```bash
# Visão geral da API legível por máquina
curl https://api.aisonar.dev/llms.txt

# Listar modelos
curl "https://api.aisonar.dev/v1/models" -H "Authorization: Bearer sk-KEY"
curl "https://api.aisonar.dev/v1/models?category=image" -H "Authorization: Bearer sk-KEY"

# Listas ranqueadas para não-chat
curl "https://api.aisonar.dev/v1/models?recommended_for=image" -H "Authorization: Bearer sk-KEY"
curl "https://api.aisonar.dev/v1/models?recommended_for=video" -H "Authorization: Bearer sk-KEY"
curl "https://api.aisonar.dev/v1/models?recommended_for=translation" -H "Authorization: Bearer sk-KEY"

# Ler o contrato público de um modelo antes de tentar novamente solicitações não-chat
curl "https://api.aisonar.dev/v1/models/gpt-image-2" -H "Authorization: Bearer sk-KEY"

# Detalhe apenas de preços
curl "https://api.aisonar.dev/v1/models/gpt-image-2/pricing" -H "Authorization: Bearer sk-KEY"
```

## Dicas de roteamento nativo

O caminho compatível com OpenAI `/v1` é o padrão para exemplos comuns de chat, Responses, imagem, embedding, áudio e rerank. Use rotas nativas Anthropic ou Gemini apenas quando a solicitação exigir comportamento específico do provedor ou quando a AI Sonar retornar the native endpoint hint header como dica de otimização.



## Recuperação de erros Agent-First

Os erros incluem campos que coding agents podem analisar sem raspar a documentação:

| Erro | Campos úteis | Ação do agent |
|---|---|---|
| Nome de modelo incorreto | `did_you_mean`, `suggestions`, `hint` | Tente novamente com o modelo corrigido ou uma alternativa sugerida |
| Saldo insuficiente | `balance_usd`, `estimated_cost_usd`, `suggestions` | Peça aprovação ou mude para um modelo acessível |
| Limite de taxa ou indisponibilidade temporária | `retryable`, `retry_after`, `alternatives` | Aguarde, tente novamente ou escolha uma alternativa listada |
| Contrato não-chat incompatível | `supported_operations`, `supported_parameters`, `request_endpoint`, `request_shape_mode`, `recommended_request` | Reconstrua a solicitação a partir do contrato público e aplique fail closed a campos que mudem a intenção |

## Famílias de API compatíveis

| Família | Caminho principal |
|---|---|
| Chat e Responses | `/v1/chat/completions`, `/v1/responses` |
| Mensagens nativas Claude | `/v1/messages` |
| Solicitações nativas Gemini | `/v1beta/models/{model}:generateContent` |
| Imagens | `/v1/images/generations`, `/v1/images/edits` |
| Vídeo | `/v1/videos/generations` |
| Música | `/v1/music/generations` |
| Worlds | `/v1/worlds/generations` mais endpoints de status world e media asset |
| 3D | `/v1/3d/generations` |
| Áudio | `/v1/audio/speech`, `/v1/audio/transcriptions`, `/v1/audio/translations` |
| Realtime | `/v1/realtime?model={model}` |
| Embeddings e rerank | `/v1/embeddings`, `/v1/rerank` |
| Tradução de texto | `/v1/translations` |

## FAQ

  <Accordion title="O skill não dispara automaticamente?">
    Mencione “AI Sonar” ou “AI Sonar API” na solicitação, por exemplo: `Use AI Sonar para adicionar geração de imagens ao meu app Node.js`.
  </Accordion>

  <Accordion title="Quais coding agents são suportados?">
    Qualquer coding agent que suporte diretórios compartilhados de skill ou rules pode usar a pasta. O installer `skills` trata agents suportados automaticamente.
  </Accordion>

  <Accordion title="Como atualizo o skill?">
    Execute novamente o comando de instalação. Ele atualiza a cópia local a partir do repositório público canônico.

    ```bash
    npx skills add https://github.com/hedging8563/aisonar-skills --skill aisonar-api-integration -y
    ```
  </Accordion>
