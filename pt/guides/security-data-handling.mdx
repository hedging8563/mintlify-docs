## Visão geral

Trate API keys, mídia de usuário, logs e assets gerados com segurança na AI Sonar.

Use as APIs públicas da AI Sonar e contratos de modelo; não dependa de detalhes de execução do provedor.

## Regras principais

- Persista os API key scope, media URL lifetime, audit events antes de retornar sucesso ao usuário.
- Use as APIs públicas da AI Sonar e contratos de modelo; não dependa de detalhes de execução do provedor.
- Faça retries idempotentes e evite geração ou cobrança duplicada.

## Checklist de produção

- Persista os API key scope, media URL lifetime, audit events antes de retornar sucesso ao usuário.
- Use as APIs públicas da AI Sonar e contratos de modelo; não dependa de detalhes de execução do provedor.
- Faça retries idempotentes e evite geração ou cobrança duplicada.
- Logs devem ajudar o suporte sem armazenar secrets ou payloads privados.
- Valide o comportamento em staging antes de alterar tráfego de produção.
