## Por que os Coding Agents são caros

Agentes como Claude Code, Cursor e Codex CLI são poderosos, mas consomem tokens a uma velocidade que surpreende a maioria das equipes.

**Contexto longo por requisição.** Cada chamada inclui system prompt, arquivos do projeto e histórico de conversa — frequentemente 10.000–100.000+ tokens antes do modelo começar a pensar.

**Chamadas de alta frequência.** Uma única sessão de codificação gera dezenas de chamadas API: geração de código, busca, revisão, autocompletar e uso de ferramentas. Uma sessão de 1 hora facilmente ultrapassa 200 requisições.

**Acumulação de conversas.** Cada turno reenvia o histórico completo de mensagens. No turno 20, você está pagando 20 vezes pelo mesmo contexto.

<Info>
Uma sessão típica de 1 hora do Claude Code pode consumir 2–5M tokens. Com tarifas diretas de API, isso é $6–30+ por hora dependendo do modelo.
</Info>
