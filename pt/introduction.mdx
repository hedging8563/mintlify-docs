## O que é AI Sonar?

AI Sonar é uma plataforma de agregação de APIs de IA que oferece acesso unificado a **centenas de modelos de IA** de fornecedores como OpenAI, Anthropic, Google, DeepSeek, Moonshot, MiniMax e outros.

## Capacidades Suportadas

| Capacidade | Descrição | Modelos de Exemplo |
|------------|-------------|----------------|
| **Chat e raciocínio** | Geração de texto, planejamento, ferramentas | `gpt-5.4`, `claude-sonnet-4-6`, `deepseek-r1` |
| **Visão** | Entendimento de imagens e chat multimodal | `gpt-4o`, `claude-sonnet-4-6`, `gemini-2.5-flash` |
| **Geração de imagens** | Texto-para-imagem e edição de imagens | `dall-e-3`, `flux-pro`, `midjourney` |
| **Áudio** | TTS e transcrição | `gpt-4o-mini-tts`, `whisper-1`, `gpt-4o-transcribe` |
| **Embeddings** | Vetorização de texto | `text-embedding-3-small`, `text-embedding-3-large` |
| **Música** | Geração de música por IA | `suno` |
| **3D** | Geração de ativos 3D | `tripo3d` |

## Escolha o formato da requisição

Comece pela rota que corresponde ao comportamento necessário. Use rotas nativas Anthropic/Gemini para recursos nativos do provedor; use /v1/chat/completions compatível com OpenAI ao migrar código de chat no estilo OpenAI.

<CodeGroup>

```bash cURL
curl https://api.aisonar.dev/v1/chat/completions \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-5.4",
    "messages": [
      {"role": "user", "content": "Explain what AI Sonar does in one sentence."}
    ]
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.aisonar.dev/v1"
)

response = client.chat.completions.create(
    model="gpt-5.4",
    messages=[{"role": "user", "content": "Explain what AI Sonar does in one sentence."}]
)

print(response.choices[0].message.content)
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.aisonar.dev/v1'
});

const response = await client.chat.completions.create({
  model: 'gpt-5.4',
  messages: [{ role: 'user', content: 'Explain what AI Sonar does in one sentence.' }],
});

console.log(response.choices[0].message.content);
```

<Note>
Use `POST /v1/responses` apenas quando você explicitamente precisar do comportamento específico do Responses. Não é o caminho de compatibilidade padrão para todos os modelos.
</Note>

</CodeGroup>
