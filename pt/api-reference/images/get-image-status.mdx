## Parâmetros de Path

<ParamField path="id" type="string" required>
  O ID da tarefa retornado da solicitação inicial de geração de imagem.
</ParamField>

<Info>
Se a resposta de criação incluir `poll_url`, prefira chamar essa URL exata para polling. Algumas tarefas de imagem podem apresentar uma `poll_url` em `/v1/tasks/{id}` em vez do caminho de status específico da imagem.

Você também pode consultar IDs públicos de tarefas de imagem como `ldtask_...` em `/v1/images/{id}`. Este alias de compatibilidade usa a mesma autorização, verificação de propriedade, cabeçalhos de cobrança, snapshots terminais e formato de resposta que `/v1/images/generations/{id}`.


Se a tarefa não existir mais ou não puder mais ser resolvida por meio do contrato público de async-task, AI Sonar retorna `async_task_not_found` com a mensagem `Task not found or no longer available.`
</Info>

## Resposta

<ResponseField name="created" type="integer">
  Timestamp Unix da criação.
</ResponseField>

<ResponseField name="task_id" type="string">
  O identificador da tarefa.
</ResponseField>

<ResponseField name="status" type="string">
  Status da tarefa: `pending`, `processing`, `completed` ou `failed`.
</ResponseField>

<ResponseField name="data" type="array">
  Array de imagens geradas (preenchido quando o `status` é `completed`).

  Cada objeto contém:
  - `url` (string): URL da imagem gerada
  - `revised_prompt` (string): O prompt utilizado para a geração
</ResponseField>

<ResponseField name="error" type="string">
  Mensagem de erro (presente apenas quando o `status` é `failed`).
</ResponseField>

## Melhores Práticas de Polling

<Tip>
**Intervalo de polling recomendado**: 3-5 segundos. A maioria das tarefas de geração de imagem é concluída em 30-120 segundos, dependendo do modelo e do caminho do provedor roteado.
</Tip>

```python
import requests
import time

def poll_image_task(task_id, api_key, max_wait=300, interval=3):
    """Poll for image generation result with timeout."""
    url = f"https://api.aisonar.dev/v1/tasks/{task_id}"
    headers = {"Authorization": f"Bearer {api_key}"}

    start_time = time.time()
    while time.time() - start_time < max_wait:
        response = requests.get(url, headers=headers)
        data = response.json()

        if data["status"] == "completed":
            return data["data"][0]["url"]
        elif data["status"] == "failed":
            raise Exception(data.get("error", "Generation failed"))

        time.sleep(interval)

    raise TimeoutError(f"Task {task_id} did not complete within {max_wait}s")

# Usage
image_url = poll_image_task("ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa", "sk-your-api-key")
print(f"Generated image: {image_url}")
```
