---
title: "Criar World"
openapi: "POST /v1/worlds/generations"
description: "Cria uma tarefa de geração de mundo World Labs Marble"
---

Gere mundos 3D exploráveis com World Labs Marble. Esta API é assíncrona: respostas de criação retornam uma identidade de tarefa e uma `poll_url` para verificar status.

Os modelos suportados são `marble-1.0`, `marble-1.1` e `marble-1.1-plus`. A World Labs documenta `marble-1.0-draft`, mas o AI Sonar ainda não expõe geração draft neste endpoint.

## Corpo da requisição

<ParamField body="model" type="string" default="marble-1.0">
  Modelo Marble a usar: `marble-1.0`, `marble-1.1` ou `marble-1.1-plus`.
</ParamField>

<ParamField body="prompt" type="string">
  Prompt de texto para geração somente texto ou como guia com entradas de imagem/vídeo.
</ParamField>

<ParamField body="world_prompt" type="object">
  Objeto nativo World Labs `world_prompt` para chamadas avançadas. Tipos suportados: `text`, `image`, `multi-image` e `video`.
</ParamField>

<ParamField body="image" type="string">
  Prompt de imagem em Base64 ou data URL.
</ParamField>

<ParamField body="image_url" type="string">
  Prompt por URL de imagem.
</ParamField>

<ParamField body="images" type="array">
  Vários prompts de imagem para geração multi-image rápida. Forneça até 4 imagens. Para o modo de reconstrução nativo da World Labs, envie `world_prompt.type="multi-image"` com `reconstruct_images: true` e até 8 imagens.
</ParamField>

<ParamField body="video_url" type="string">
  Prompt por URL de vídeo.
</ParamField>

<ParamField body="is_pano" type="boolean | string">
  Para entradas de imagem, use `true` para panorama existente, `false` para imagem única normal, ou `auto`.
</ParamField>

<ParamField body="seed" type="integer">
  Seed opcional de 0 a 4294967295.
</ParamField>

<ParamField body="display_name" type="string">
  Nome de exibição opcional para o mundo upstream, até 64 caracteres.
</ParamField>

<ParamField body="tags" type="array">
  Tags upstream opcionais. Até 10 tags, cada uma com até 32 caracteres.
</ParamField>

<ParamField body="permission" type="object">
  Objeto permission opcional da World Labs.
</ParamField>

## Resposta

<ResponseField name="id" type="string">
  ID público da tarefa para polling.
</ResponseField>

<ResponseField name="task_id" type="string">
  Alias do identificador de tarefa assíncrona.
</ResponseField>

<ResponseField name="operation_id" type="string">
  ID de operation upstream da World Labs.
</ResponseField>

<ResponseField name="poll_url" type="string">
  URL de polling preferida para esta tarefa.
</ResponseField>

<ResponseField name="status" type="string">
  Status da tarefa: `pending`, `processing`, `completed` ou `failed`.
</ResponseField>

<ResponseField name="world_marble_url" type="string">
  URL do mundo Marble gerado quando concluído.
</ResponseField>

<ResponseField name="glb_url" type="string">
  URL GLB da malha collider quando disponível.
</ResponseField>

<ResponseField name="pano_url" type="string">
  URL da imagem panorâmica quando disponível.
</ResponseField>

## Preços

A World Labs cobra em credits. O AI Sonar pré-deduz o máximo do tipo de requisição e liquida pelo `cost.total_credits` da operation concluída quando disponível. Requisições Marble padrão chegam a 1.600 credits. `marble-1.1-plus` chega a 3.100 credits.

## Escopo

Este endpoint cobre geração de mundos e polling. O AI Sonar atualmente não faz proxy de upload de media assets da World Labs, listagem de mundos, exclusão de mundos ou ferramentas pano independentes.

<RequestExample>
```bash cURL
curl -X POST "https://api.aisonar.dev/v1/worlds/generations" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "marble-1.1",
    "prompt": "Uma cidade costeira tranquila ao pôr do sol com vielas estreitas e luzes quentes"
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.aisonar.dev/v1/worlds/generations",
    headers={"Authorization": "Bearer sk-your-api-key"},
    json={
        "model": "marble-1.1",
        "prompt": "Uma cidade costeira tranquila ao pôr do sol com vielas estreitas e luzes quentes",
    },
)

task = response.json()
print(task["id"], task["poll_url"])
```

```javascript JavaScript

console.log(task.id, task.poll_url);
```
