---
title: "Obter status do World"
openapi: "GET /v1/worlds/generations/{id}"
description: "Recupera o status e o resultado de uma tarefa de geração de mundo World Labs Marble"
---

Faça polling deste endpoint para verificar o status de uma tarefa World Labs Marble. Se a resposta de criação incluiu `poll_url`, prefira chamar essa URL exata.

## Parâmetros de caminho

<ParamField path="id" type="string" required>
  ID da tarefa de geração de mundo.
</ParamField>

## Resposta

<ResponseField name="id" type="string">
  ID público da tarefa.
</ResponseField>

<ResponseField name="task_id" type="string">
  Alias do identificador de tarefa assíncrona.
</ResponseField>

<ResponseField name="operation_id" type="string">
  ID de operation upstream da World Labs.
</ResponseField>

<ResponseField name="status" type="string">
  Status da tarefa: `pending`, `processing`, `completed`, `failed` ou `timeout`.
</ResponseField>

<ResponseField name="world_id" type="string">
  World ID upstream da World Labs quando concluído.
</ResponseField>

<ResponseField name="world_marble_url" type="string">
  URL do mundo Marble gerado quando concluído.
</ResponseField>

<ResponseField name="model_url" type="string">
  Alias da URL principal do mundo/modelo gerado.
</ResponseField>

<ResponseField name="glb_url" type="string">
  URL GLB da malha collider quando disponível.
</ResponseField>

<ResponseField name="pano_url" type="string">
  URL da imagem panorâmica quando disponível.
</ResponseField>

<ResponseField name="thumbnail_url" type="string">
  URL da miniatura quando disponível.
</ResponseField>

<ResponseField name="caption" type="string">
  Caption upstream do mundo gerado.
</ResponseField>

<ResponseField name="cost" type="object">
  Detalhes de custo da operation upstream quando disponíveis, incluindo `total_credits`.
</ResponseField>

<ResponseField name="error" type="string">
  Mensagem de erro quando a tarefa falhar.
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.aisonar.dev/v1/worlds/generations/ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
import requests
import time

task_id = "ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa"

while True:
    response = requests.get(
        f"https://api.aisonar.dev/v1/worlds/generations/{task_id}",
        headers={"Authorization": "Bearer sk-your-api-key"},
    )
    result = response.json()
    if result["status"] in ["completed", "failed", "timeout"]:
        print(result)
        break
    time.sleep(10)
```

```javascript JavaScript
const taskId = 'ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa';
const response = await fetch(`https://api.aisonar.dev/v1/worlds/generations/${taskId}`, {
  headers: { Authorization: 'Bearer sk-your-api-key' }
});

const result = await response.json();
console.log(result.status, result.world_marble_url);
```
