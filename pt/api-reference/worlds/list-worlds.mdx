---
title: "Listar Worlds"
openapi: "POST /v1/worlds/list"
description: "Lista Worlds World Labs Marble concluídos criados via AI Sonar"
---

Liste Worlds World Labs Marble concluídos criados via AI Sonar. Os resultados são limitados à sua organização; AI Sonar não expõe a lista global da conta upstream compartilhada.

Use `/v1/worlds/generations/{id}` para tarefas pendentes ou com falha. Esta lista retorna apenas Worlds concluídos que AI Sonar indexou após a geração.

## Corpo da requisição

<ParamField body="page_size" type="integer" default="20">
  Quantidade de worlds retornados, de 1 a 100.
</ParamField>

<ParamField body="page_token" type="string">
  Token de paginação da resposta anterior.
</ParamField>

<ParamField body="status" type="string">
  Filtro opcional de status. AI Sonar retorna worlds concluídos para `SUCCEEDED`; outros status pertencem ao endpoint de tarefa de geração.
</ParamField>

<ParamField body="model" type="string">
  Filtro opcional de modelo Marble.
</ParamField>

<ParamField body="tags" type="array">
  Filtro opcional de tags.
</ParamField>

<ParamField body="is_public" type="boolean">
  Filtro opcional de visibilidade.
</ParamField>

<ParamField body="created_after" type="string">
  Retornar apenas worlds criados após esse timestamp.
</ParamField>

<ParamField body="created_before" type="string">
  Retornar apenas worlds criados antes desse timestamp.
</ParamField>

<ParamField body="sort_by" type="string" default="created_at">
  Ordenar por `created_at` ou `updated_at`.
</ParamField>

<ParamField body="sort_order" type="string" default="desc">
  Direção da ordenação: `asc` ou `desc`.
</ParamField>

## Resposta

<ResponseField name="worlds" type="array">
  Array de registros world.
</ResponseField>

<ResponseField name="next_page_token" type="string">
  Token da próxima página ou `null`.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.aisonar.dev/v1/worlds/list" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{"page_size": 20, "status": "SUCCEEDED", "model": "marble-1.1"}'
```

```python Python
import requests

response = requests.post(
    "https://api.aisonar.dev/v1/worlds/list",
    headers={"Authorization": "Bearer sk-your-api-key"},
    json={"page_size": 20, "status": "SUCCEEDED", "model": "marble-1.1"},
)
print(response.json()["worlds"])
```
