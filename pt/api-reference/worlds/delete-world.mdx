---
title: "Excluir World"
openapi: "DELETE /v1/worlds/{id}"
description: "Exclui um World World Labs Marble"
---

Exclua um Marble World criado via AI Sonar. AI Sonar encaminha a exclusão para a World Labs e marca o registro local como excluído.

A exclusão é permanente.

## Parâmetros de caminho

<ParamField path="id" type="string" required>
  ID do World Labs world.
</ParamField>

## Resposta

<ResponseField name="deleted" type="boolean">
  Se a exclusão foi aceita.
</ResponseField>

<ResponseField name="world_id" type="string">
  ID do World Labs world.
</ResponseField>

<RequestExample>
```bash cURL
curl -X DELETE "https://api.aisonar.dev/v1/worlds/world_123" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
import requests

response = requests.delete(
    "https://api.aisonar.dev/v1/worlds/world_123",
    headers={"Authorization": "Bearer sk-your-api-key"},
)
print(response.json())
```

```javascript JavaScript
const response = await fetch('https://api.aisonar.dev/v1/worlds/world_123', {
  method: 'DELETE',
  headers: { Authorization: 'Bearer sk-your-api-key' }
});
console.log(await response.json());
```
</RequestExample>
