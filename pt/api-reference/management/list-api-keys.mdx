---
title: "Listar chaves API"
openapi: "GET /v1/management/api-keys"
description: "Lista as chaves API gerenciadas pelo usuário para o token de gerenciamento atual da organização."
---

## Visão Geral

Retorna as chaves API de usuário da organização associada ao token de gerenciamento atual.

## Notas

- As chaves gerenciadas pelo sistema são excluídas.

## Exemplo

<RequestExample>
```bash cURL
curl -X GET "https://api.aisonar.dev/v1/management/api-keys" \
  -H "Authorization: Bearer mt-your-management-token"
```
</RequestExample>

## Solicitação / Resposta

Use o painel interativo OpenAPI acima para o esquema exato.

## Exemplo de resposta

<ResponseExample>
```json 200 OK
{
  "object": "list",
  "data": [{
    "id": "ak_abc123",
    "name": "Production app",
    "status": "active",
    "created_at": "2026-06-27T00:00:00.000Z",
    "last_used_at": "2026-06-27T01:00:00.000Z"
  }]
}
```
</ResponseExample>

## Campos importantes

<ResponseField name="data" type="array">Lista ou payload retornado por esta resposta.</ResponseField>
<ResponseField name="status" type="string">Estado atual do ciclo de vida deste recurso.</ResponseField>
