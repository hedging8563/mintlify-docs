## Visão Geral

Este endpoint retorna os totais atuais de saldo da organização associada ao token de gerenciamento. Ele usa a mesma fonte da verdade de saldo da organização usada por faturamento e liquidação.

## Notas

- Os campos monetários de verdade são retornados em USD: `balance`, `total_recharge` e `total_used`.
- `preferred_currency` reflete a preferência de exibição da organização no Dashboard; ela não altera os campos de verdade em USD.

## Exemplo

<RequestExample>
```bash cURL
curl -X GET "https://api.aisonar.dev/v1/management/balance" \
  -H "Authorization: Bearer mt-your-management-token"
```
</RequestExample>

## Solicitação / Resposta

Use o painel interativo OpenAPI acima para o esquema exato.

## Exemplo de resposta

<ResponseExample>
```json 200 OK
{
  "object": "management.balance",
  "currency": "USD",
  "available_balance": "42.50",
  "pending_usage": "1.25",
  "updated_at": "2026-06-27T00:00:00.000Z"
}
```
</ResponseExample>

## Campos importantes

<ResponseField name="available_balance" type="string">String decimal em USD para reconciliação de cobrança.</ResponseField>
<ResponseField name="pending_usage" type="string">String decimal em USD para reconciliação de cobrança.</ResponseField>
<ResponseField name="updated_at" type="string">Timestamp ISO ou Unix retornado pela API.</ResponseField>
