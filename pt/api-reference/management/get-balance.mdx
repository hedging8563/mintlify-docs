## Visão Geral

Este endpoint retorna os totais atuais de saldo da organização associada ao token de gerenciamento. Ele usa a mesma fonte da verdade de saldo da organização usada por faturamento e liquidação.

## Notas

- Os campos monetários de verdade são retornados em USD: `balance`, `total_recharge` e `total_used`.
- `preferred_currency` reflete a preferência de exibição da organização no Dashboard; ela não altera os campos de verdade em USD.

## Exemplo

<RequestExample>
```bash cURL
curl -X GET "https://api.aisonar.dev/v1/management/balance" \
  -H "Authorization: Bearer mt-your-management-token"
```
</RequestExample>
