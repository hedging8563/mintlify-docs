---

## Visão Geral

Este endpoint é parte da API de Lote compatível com OpenAI da AI Sonar. Trabalhos em lote utilizam arquivos JSONL carregados, são executados de forma assíncrona e podem retornar arquivos de saída/erro posteriormente.

## Notas

- Os arquivos de entrada do lote devem usar `purpose=batch`.
- `completion_window` é atualmente `24h`.
- A ordenação da saída não é garantida; sempre combine pelo `custom_id`.
- Streaming não é suportado dentro dos itens do lote.

## Exemplo

<RequestExample>
```bash cURL
curl -X GET "https://api.aisonar.dev/v1/batches/batch_abc123" \
  -H "Authorization: Bearer sk-your-api-key"
```
</RequestExample>

## Solicitação / Resposta

Use o painel interativo do OpenAPI acima para o esquema exato.

## Dicas Operacionais

- Use `custom_id` para reconciliação idempotente a montante.
- Espere `output_file_id` e `error_file_id` apenas após o trabalhador finalizar o lote.
- A precificação do lote pode diferir da precificação síncrona porque as regras de desconto `isBatchRequest=true` se aplicam.

## Exemplo de resposta

<ResponseExample>
```json 200 OK

  "id": "batch_abc123",
  "object": "batch",
  "status": "completed",

```
</ResponseExample>
