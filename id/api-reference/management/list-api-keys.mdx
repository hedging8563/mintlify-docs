---
title: "Daftar API Key"
openapi: "GET /v1/management/api-keys"
description: "Mendaftarkan API key yang dikelola pengguna untuk token manajemen organisasi saat ini."
---

## Ikhtisar

Mengembalikan daftar API key pengguna di organisasi yang terkait dengan token manajemen saat ini.

## Catatan

- Key yang dikelola sistem tidak disertakan.

## Contoh

<RequestExample>
```bash cURL
curl -X GET "https://api.aisonar.dev/v1/management/api-keys" \
  -H "Authorization: Bearer mt-your-management-token"
```
</RequestExample>

## Permintaan / Respons

Gunakan panel OpenAPI interaktif di atas untuk skema yang tepat.

## Contoh respons

<ResponseExample>
```json 200 OK
{
  "object": "list",
  "data": [{
    "id": "ak_abc123",
    "name": "Production app",
    "status": "active",
    "created_at": "2026-06-27T00:00:00.000Z",
    "last_used_at": "2026-06-27T01:00:00.000Z"
  }]
}
```
</ResponseExample>

## Field penting

<ResponseField name="data" type="array">Daftar atau payload yang dikembalikan respons ini.</ResponseField>
<ResponseField name="status" type="string">Status lifecycle saat ini untuk resource ini.</ResponseField>
