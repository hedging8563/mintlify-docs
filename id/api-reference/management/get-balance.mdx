## Ikhtisar

Endpoint ini mengembalikan total saldo saat ini untuk organisasi yang terkait dengan management token. Endpoint ini memakai sumber kebenaran saldo organisasi yang sama dengan billing dan settlement.

## Catatan

- Field moneter utama dikembalikan dalam USD: `balance`, `total_recharge`, dan `total_used`.
- `preferred_currency` mencerminkan preferensi tampilan Dashboard organisasi; field utama USD tidak berubah.

## Contoh

<RequestExample>
```bash cURL
curl -X GET "https://api.aisonar.dev/v1/management/balance" \
  -H "Authorization: Bearer mt-your-management-token"
```
</RequestExample>

## Permintaan / Respons

Gunakan panel OpenAPI interaktif di atas untuk skema yang tepat.

## Contoh respons

<ResponseExample>
```json 200 OK
{
  "object": "management.balance",
  "currency": "USD",
  "available_balance": "42.50",
  "pending_usage": "1.25",
  "updated_at": "2026-06-27T00:00:00.000Z"
}
```
</ResponseExample>

## Field penting

<ResponseField name="available_balance" type="string">String desimal USD untuk rekonsiliasi billing.</ResponseField>
<ResponseField name="pending_usage" type="string">String desimal USD untuk rekonsiliasi billing.</ResponseField>
<ResponseField name="updated_at" type="string">Waktu ISO atau Unix timestamp dari API.</ResponseField>
