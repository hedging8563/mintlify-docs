---
title: "Hapus World"
openapi: "DELETE /v1/worlds/{id}"
description: "Menghapus World Labs Marble world"
---

Hapus Marble World yang dibuat melalui AI Sonar. AI Sonar meneruskan penghapusan ke World Labs dan menandai catatan world lokal sebagai terhapus.

Penghapusan bersifat permanen.

## Parameter path

<ParamField path="id" type="string" required>
  World Labs world ID.
</ParamField>

## Respons

<ResponseField name="deleted" type="boolean">
  Apakah penghapusan diterima.
</ResponseField>

<ResponseField name="world_id" type="string">
  World Labs world ID.
</ResponseField>

<RequestExample>
```bash cURL
curl -X DELETE "https://api.aisonar.dev/v1/worlds/world_123" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
import requests

response = requests.delete(
    "https://api.aisonar.dev/v1/worlds/world_123",
    headers={"Authorization": "Bearer sk-your-api-key"},
)
print(response.json())
```

```javascript JavaScript
const response = await fetch('https://api.aisonar.dev/v1/worlds/world_123', {
```
