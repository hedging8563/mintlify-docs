---
title: "Daftar Worlds"
openapi: "POST /v1/worlds/list"
description: "Menampilkan World Labs Marble world selesai yang dibuat melalui AI Sonar"
---

Menampilkan World Labs Marble worlds yang selesai dibuat melalui AI Sonar. Hasil dibatasi untuk organisasi Anda dan hanya mencakup resource yang dibuat melalui AI Sonar.

Gunakan `/v1/worlds/generations/{id}` untuk tugas yang pending atau gagal. Daftar ini hanya mengembalikan world selesai yang telah diindeks AI Sonar setelah generasi.

## Body permintaan

<ParamField body="page_size" type="integer" default="20">
  Jumlah world yang dikembalikan, 1 sampai 100.
</ParamField>

<ParamField body="page_token" type="string">
  Token paginasi dari respons sebelumnya.
</ParamField>

<ParamField body="status" type="string">
  Filter status opsional. AI Sonar mengembalikan world selesai untuk `SUCCEEDED`; status lain berada di endpoint tugas generasi.
</ParamField>

<ParamField body="model" type="string">
  Filter model Marble opsional.
</ParamField>

<ParamField body="tags" type="array">
  Filter tag opsional.
</ParamField>

<ParamField body="is_public" type="boolean">
  Filter visibilitas opsional.
</ParamField>

<ParamField body="created_after" type="string">
  Hanya kembalikan world yang dibuat setelah timestamp ini.
</ParamField>

<ParamField body="created_before" type="string">
  Hanya kembalikan world yang dibuat sebelum timestamp ini.
</ParamField>

<ParamField body="sort_by" type="string" default="created_at">
  Urutkan berdasarkan `created_at` atau `updated_at`.
</ParamField>

<ParamField body="sort_order" type="string" default="desc">
  Arah urutan: `asc` atau `desc`.
</ParamField>

## Respons

<ResponseField name="worlds" type="array">
  Array catatan world.
</ResponseField>

<ResponseField name="next_page_token" type="string">
  Token halaman berikutnya atau `null`.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.aisonar.dev/v1/worlds/list" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{"page_size": 20, "status": "SUCCEEDED", "model": "marble-1.1"}'
```

```python Python
import requests

response = requests.post(
    "https://api.aisonar.dev/v1/worlds/list",
    headers={"Authorization": "Bearer sk-your-api-key"},
    json={"page_size": 20, "status": "SUCCEEDED", "model": "marble-1.1"},
)
print(response.json()["worlds"])
```
