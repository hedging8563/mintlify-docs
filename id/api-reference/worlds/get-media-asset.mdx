---
title: "Ambil aset media"
openapi: "GET /v1/worlds/media-assets/{id}"
description: "Mengambil aset media World Labs yang disiapkan melalui AI Sonar"
---

Ambil metadata aset media yang disiapkan melalui AI Sonar. Akses dibatasi pada organisasi yang membuat aset tersebut.

## Parameter path

<ParamField path="id" type="string" required>
  ID aset media World Labs.
</ParamField>

## Respons

<ResponseField name="media_asset_id" type="string">
  ID aset media World Labs.
</ResponseField>

<ResponseField name="file_name" type="string">
  Nama file asli, hingga 64 karakter.
</ResponseField>

<ResponseField name="kind" type="string">
  Jenis media: `image` atau `video`.
</ResponseField>

<ResponseField name="metadata" type="object">
  Objek metadata opsional yang disimpan bersama aset media upstream.
</ResponseField>
