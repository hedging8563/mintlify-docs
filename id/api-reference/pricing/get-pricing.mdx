## Ikhtisar

Gunakan endpoint ini untuk menampilkan permukaan harga publik saat ini untuk model yang aktif.

Untuk detail harga lengkap satu model, termasuk harga terstruktur dan provenance, utamakan [Dapatkan Harga Model](/id/api-reference/models/get-model).

## Parameter Query

<ParamField query="provider" type="string">
  Filter provider opsional seperti `openai`, `anthropic`, atau `google`.
</ParamField>

<ParamField query="tag" type="string">
  Filter tag model opsional seperti `chat`, `image`, `video`, `embedding`, atau `translation`.
</ParamField>

## Respons

<ResponseField name="object" type="string">
  Selalu `pricing`.
</ResponseField>

<ResponseField name="updated_at" type="string">
  Timestamp ISO untuk entri harga terbaru yang disertakan dalam respons.
</ResponseField>

<ResponseField name="currency" type="string">
  Selalu `USD`.
</ResponseField>

<ResponseField name="data" type="array">
  Array entri harga. Setiap item mencakup `model`, `provider`, `is_lock_price`, `capabilities`, `aliases`, dan field ringkasan harga publik berbentuk scalar atau terstruktur yang dikembalikan skema interaktif di atas.
</ResponseField>

<Note>
Gunakan panel OpenAPI interaktif di atas untuk skema respons yang tepat. Entri harga publik juga dapat menampilkan `lock_price_per_request`, `has_complex_pricing`, `supported_operations`, `pricing_summary`, dan `pricing_provenance`.
</Note>
