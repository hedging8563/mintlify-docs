## Ikhtisar

Gunakan `/v1/translations` untuk **terjemahan teks ke teks**.

<Note>
Endpoint ini berbeda dari [Terjemahan Audio](/id/api-reference/audio/create-translation), yang menerima file audio dan selalu menghasilkan teks bahasa Inggris.
</Note>

Untuk alur agent, temukan dulu model terjemahan yang direkomendasikan saat ini:

```bash
curl "https://api.aisonar.dev/v1/models?recommended_for=translation" \
  -H "Authorization: Bearer sk-your-api-key"
```

Lalu kirim model yang dipilih secara eksplisit ke `/v1/translations`.

<Note>
`recommended_for=translation` hanya berlaku untuk **terjemahan teks** (`POST /v1/translations`). Tidak berlaku untuk [terjemahan audio](/id/api-reference/audio/create-translation).
</Note>

## Body Request

<ParamField body="model" type="string" required>
  ID model terjemahan yang dikembalikan dari `/v1/models?recommended_for=translation`.
</ParamField>

<ParamField body="text" type="string" required>
  Teks sumber yang ingin diterjemahkan.
</ParamField>

<ParamField body="target_language" type="string" required>
  Kode bahasa target atau nama bahasa, tergantung dukungan model.
</ParamField>

<ParamField body="source_language" type="string">
  Petunjuk bahasa sumber opsional.
</ParamField>

<ParamField body="mime_type" type="string" default="text/plain">
  Format teks sumber. Nilai yang didukung: `text/plain`, `text/html`.
</ParamField>

<ParamField body="user" type="string">
  Identifier end-user opsional untuk monitoring abuse dan pelacakan request.
</ParamField>

## Respons

<ResponseField name="text" type="string">
  Teks yang sudah diterjemahkan.
</ResponseField>

<ResponseField name="model" type="string">
  Model yang menghasilkan terjemahan.
</ResponseField>

<ResponseField name="source_language" type="string | null">
  Bahasa sumber yang dideteksi atau diterima model.
</ResponseField>

<ResponseField name="target_language" type="string">
  Bahasa target yang digunakan untuk request.
</ResponseField>
