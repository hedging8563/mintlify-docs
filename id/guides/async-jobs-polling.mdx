## Ikhtisar

Lacak workflow media, batch, dan task tanpa bergantung pada endpoint status khusus provider.

Gunakan API publik AI Sonar dan kontrak model; jangan bergantung pada detail eksekusi provider.

## Aturan utama

- Simpan `task_id`, `poll_url`, terminal status sebelum mengembalikan sukses ke pengguna.
- Gunakan API publik AI Sonar dan kontrak model; jangan bergantung pada detail eksekusi provider.
- Buat retry idempotent dan hindari generasi atau tagihan ganda.

## Checklist produksi

- Simpan `task_id`, `poll_url`, terminal status sebelum mengembalikan sukses ke pengguna.
- Gunakan API publik AI Sonar dan kontrak model; jangan bergantung pada detail eksekusi provider.
- Buat retry idempotent dan hindari generasi atau tagihan ganda.
- Log harus membantu support tanpa menyimpan secret atau payload privat.
- Verifikasi perilaku di staging sebelum mengubah traffic produksi.
