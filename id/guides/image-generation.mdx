API gambar AI Sonar mencakup beberapa keluarga model dengan bentuk permintaan yang berbeda. Gunakan kontrak model publik dan referensi API spesifik endpoint sebelum mengasumsikan bahwa suatu field berfungsi di setiap model gambar.

## Pilih Endpoint

| Tujuan | Endpoint | Catatan |
|--------|----------|---------|
| Teks-ke-gambar | `POST /v1/images/generations` | Gunakan `prompt` dan `model` yang eksplisit |
| Gambar-ke-gambar / gambar referensi | `POST /v1/images/generations` | Gunakan `operation: "image-to-image"` ditambah `image_url`, `image_urls`, atau `reference_image_urls` ketika model mendukungnya |
| Pengeditan Gambar GPT | `POST /v1/images/edits` | Mendukung multipart `image`, JSON `image_url` / `image_urls`, dan referensi resmi `images[]` untuk model edit yang didukung |
| Variasi warisan | `POST /v1/images/variations` | Utamakan rute generasi atau edit saat ini untuk pekerjaan baru |
| Status Async | `GET /v1/tasks/{id}` | Gunakan `poll_url` terlebih dahulu jika ada |

## Penemuan Model

```bash
curl "https://api.aisonar.dev/v1/models?recommended_for=image" \\
  -H "Authorization: Bearer sk-your-api-key"
```

Parameter model gambar sangat bervariasi. Bacalah kontrak model yang dipilih sebelum mengirim `size`, `quality`, `aspect_ratio`, `resolution`, `async`, `background`, atau `output_format`.

## Input Dan Field Spesifik Model

- `gpt-image-2` menggunakan `size` dan `quality` gaya OpenAI; hilangkan `size` atau `quality` untuk membiarkan AI Sonar menggunakan `auto` di mana didukung.
- Keluarga gambar Gemini dan Nano Banana umumnya menggunakan `aspect_ratio`; hanya kirim `resolution` ketika model yang dipilih mengeksposnya.
- Permintaan gambar referensi Nano Banana harus berada di `/v1/images/generations` dengan `operation: "image-to-image"` dan `image_urls`, bukan di `/v1/images/edits`.
- `/v1/images/generations` tidak menerima `images[]` atau `file_id` di tingkat atas; itu adalah bentuk alur edit.
- URL gambar jarak jauh harus berupa URL publik `http`/`https`. Hindari jaringan pribadi, kredensial yang tertanam, fragmen, dan URL bertanda tangan yang berumur pendek.

## Buat Permintaan

```bash
curl https://api.aisonar.dev/v1/images/generations \\
  -H "Authorization: Bearer sk-your-api-key" \\
  -H "Content-Type: application/json" \\
  -d '{
    "model": "gpt-image-2",
    "prompt": "Sebuah foto produk bersih dari cangkir kopi keramik di atas meja kenari",
    "size": "1024x1024",
    "response_format": "url"
  }'
```

## Memantau Hasil

Beberapa rute gambar mengembalikan `data[].url` atau `data[].b64_json` secara inline. Rute lainnya mengembalikan `status: "pending"`, `task_id`, dan `poll_url`. Jika Anda meminta generasi atau pengeditan gambar async, hasil tugas yang selesai mengembalikan URL; gunakan permintaan sinkron ketika Anda secara khusus membutuhkan `b64_json`.

```bash
curl "https://api.aisonar.dev/v1/tasks/$TASK_ID" \\
  -H "Authorization: Bearer sk-your-api-key"
```

## Kesalahan Umum

- Jangan mengasumsikan setiap model gambar bersifat sinkron.
- Jangan meminta `b64_json` dari tugas gambar async; hasil async berorientasi URL.
- Jangan mengirim `resolution` ke model gambar Gemini yang hanya menggunakan rasio.
- Jangan gunakan `/v1/images/edits` untuk alur gambar-ke-gambar Nano Banana.
- Atur waktu tunggu klien HTTP setidaknya `120s` untuk permintaan gambar resolusi tinggi yang sinkron.
