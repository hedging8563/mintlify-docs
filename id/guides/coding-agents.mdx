## Mengapa Coding Agents Mahal

Coding agents seperti Claude Code, Cursor, dan Codex CLI sangat powerful — tetapi mereka mengonsumsi token dengan kecepatan yang mengejutkan banyak tim.

**Konteks panjang per permintaan.** Setiap panggilan menyertakan system prompt, file proyek, dan riwayat percakapan — seringkali 10.000–100.000+ token sebelum model mulai berpikir.

**Panggilan dengan frekuensi tinggi.** Satu sesi coding memicu puluhan panggilan API: pembuatan kode, pencarian, review, autocomplete, dan penggunaan tool. Sesi 1 jam bisa dengan mudah mencapai 200+ permintaan.

**Akumulasi percakapan.** Setiap giliran mengirim ulang seluruh riwayat pesan. Pada giliran ke-20, Anda membayar untuk konteks yang sama sebanyak 20 kali.

<Info>
Sesi Claude Code selama 1 jam yang tipikal dapat mengonsumsi 2–5 juta token. Dengan tarif API langsung, itu berarti $6–30+ per jam tergantung modelnya.
</Info>
