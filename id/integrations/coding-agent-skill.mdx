## Apa yang dilakukan skill ini

- Membuat contoh minimal yang dapat dijalankan untuk keluarga API AI Sonar seperti chat, image, audio, video, translation, dan lainnya.
- Menggunakan `https://api.aisonar.dev/v1` untuk client kompatibel OpenAI dan menjelaskan kapan berpindah ke route native Anthropic atau Gemini.
- Menemukan model lewat `/v1/models`, `/llms.txt`, dan daftar `recommended_for` alih-alih menebak dari daftar usang.
- Membaca kontrak publik model sebelum mencoba ulang permintaan non-chat agar field yang tidak didukung tidak dibuang diam-diam.
- Menangani petunjuk error Agent-First seperti `did_you_mean`, `suggestions`, `retry_after`, dan `recommended_request`.

## Instalasi

<Note>
Gunakan perintah instalasi kanonis non-interaktif:

```bash
npx skills add https://github.com/hedging8563/aisonar-skills --skill aisonar-api-integration -y
```

Ini memasang skill bersama `aisonar-api-integration` dari repository AI Sonar skills.

Jika alat Anda tidak mendukung installer, salin `skills/aisonar-api-integration/` dari repository ke direktori skills atau rules bersama milik alat Anda.
</Note>

### Perbarui instalasi yang ada

Jika Anda memasang skill sebelum pembersihan repository AI Sonar, jalankan lagi perintah yang sama. Paket publik saat ini lebih kecil dan tidak lagi bergantung pada script pencarian lokal yang digenerasi.

### Verifikasi instalasi

Tanyakan ke coding agent Anda:

```
Skills apa saja yang tersedia?
```

Jika `aisonar-api-integration` terlihat, instalasi berhasil.

## Alur Agent yang disarankan

1. Mulai dari contoh kerja terkecil yang sesuai dengan bahasa dan keluarga API yang diminta pengguna.
2. Saat pilihan model terbuka, panggil `/v1/models` atau `https://api.aisonar.dev/llms.txt` sebelum meng-hardcode model.
3. Untuk pekerjaan non-chat, panggil `/v1/models?recommended_for=<scene>` dengan `<scene>` berupa `image`, `video`, `music`, `3d`, `tts`, `stt`, `embedding`, `rerank`, atau `translation`.
4. Sebelum mengubah permintaan non-chat yang gagal, baca `/v1/models/:model` dan selaraskan dengan `supported_operations`, `supported_parameters`, `request_endpoint`, `request_shape_mode`, dan `recommended_request`.
5. Jika API mengembalikan error Agent-First, gunakan field terstruktur untuk memperbaiki permintaan dan coba ulang hanya jika error retryable.

## Contoh chat minimal

Contoh ini memakai OpenAI Python SDK dengan base URL AI Sonar:

```python
import os
from openai import OpenAI

client = OpenAI(
    api_key=os.environ["TOKENLAB_API_KEY"],
    base_url="https://api.aisonar.dev/v1"
)

response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)
```

Jalankan dengan:

```bash
pip install openai
export TOKENLAB_API_KEY="sk-your-api-key"
python app.py
```

## Penemuan model dan kontrak publik

Utamakan penemuan live daripada daftar bawaan yang usang:

```bash
# Ringkasan API yang dapat dibaca mesin
curl https://api.aisonar.dev/llms.txt

# Daftar model
curl "https://api.aisonar.dev/v1/models" -H "Authorization: Bearer sk-KEY"
curl "https://api.aisonar.dev/v1/models?category=image" -H "Authorization: Bearer sk-KEY"

# Daftar pendek non-chat berperingkat
curl "https://api.aisonar.dev/v1/models?recommended_for=image" -H "Authorization: Bearer sk-KEY"
curl "https://api.aisonar.dev/v1/models?recommended_for=video" -H "Authorization: Bearer sk-KEY"
curl "https://api.aisonar.dev/v1/models?recommended_for=translation" -H "Authorization: Bearer sk-KEY"

# Baca kontrak publik satu model sebelum mencoba ulang permintaan non-chat
curl "https://api.aisonar.dev/v1/models/gpt-image-2" -H "Authorization: Bearer sk-KEY"

# Detail khusus harga
curl "https://api.aisonar.dev/v1/models/gpt-image-2/pricing" -H "Authorization: Bearer sk-KEY"
```

## Petunjuk routing native

Jalur kompatibel OpenAI `/v1` adalah default untuk contoh umum chat, Responses, image, embedding, audio, dan rerank. Gunakan route native Anthropic atau Gemini hanya ketika permintaan membutuhkan perilaku khusus provider atau AI Sonar mengembalikan the native endpoint hint header sebagai petunjuk optimasi.



## Pemulihan error Agent-First

Error menyertakan field yang dapat diparse coding agent tanpa mengorek dokumentasi:

| Error | Field berguna | Tindakan agent |
|---|---|---|
| Nama model salah | `did_you_mean`, `suggestions`, `hint` | Coba ulang dengan model yang diperbaiki atau satu alternatif yang disarankan |
| Saldo tidak cukup | `balance_usd`, `estimated_cost_usd`, `suggestions` | Minta persetujuan atau pindah ke model yang lebih terjangkau |
| Batas laju atau sementara tidak tersedia | `retryable`, `retry_after`, `alternatives` | Tunggu, coba ulang, atau pilih alternatif yang tercantum |
| Kontrak non-chat tidak cocok | `supported_operations`, `supported_parameters`, `request_endpoint`, `request_shape_mode`, `recommended_request` | Bangun ulang permintaan dari kontrak publik dan fail closed pada field yang mengubah niat |

## Keluarga API yang didukung

| Keluarga | Path utama |
|---|---|
| Chat dan Responses | `/v1/chat/completions`, `/v1/responses` |
| Claude-native messages | `/v1/messages` |
| Permintaan Gemini-native | `/v1beta/models/{model}:generateContent` |
| Gambar | `/v1/images/generations`, `/v1/images/edits` |
| Video | `/v1/videos/generations` |
| Musik | `/v1/music/generations` |
| Worlds | `/v1/worlds/generations` plus endpoint status world dan media asset |
| 3D | `/v1/3d/generations` |
| Audio | `/v1/audio/speech`, `/v1/audio/transcriptions`, `/v1/audio/translations` |
| Realtime | `/v1/realtime?model={model}` |
| Embeddings dan rerank | `/v1/embeddings`, `/v1/rerank` |
| Terjemahan teks | `/v1/translations` |
