## Apa itu AI Sonar?

AI Sonar adalah platform agregasi API AI yang memberi Anda akses terpadu ke **ratusan model AI** dari penyedia seperti OpenAI, Anthropic, Google, DeepSeek, Moonshot, MiniMax, dan lainnya.

<CardGroup cols={2}>
  <Card title="Native + kompatibel" icon="plug">
    Gunakan rute native Anthropic/Gemini saat perilaku penyedia penting, atau rute `/v1` yang kompatibel dengan OpenAI untuk alat bergaya OpenAI yang sudah ada.
  </Card>
  <Card title="Format Native" icon="shuffle">
    Panggil endpoint bergaya OpenAI, Anthropic, Gemini, dan Responses dari satu platform.

## Kemampuan yang Didukung

Katalog publik berubah seiring waktu. Periksa ketersediaan saat ini lewat `/v1/models`, `recommended_for`, atau halaman Models; contoh di bawah memakai ID model publik saat ini.

| Kemampuan | Deskripsi | Contoh model |
|---|---|---|
| Chat dan penalaran | Pembuatan teks, perencanaan, tools | `gpt-5.4`, `claude-sonnet-4-6`, `deepseek-r1` |
| Vision dan multimodal | Pemahaman gambar dan chat multimodal | `gpt-4o`, `claude-sonnet-4-6`, `gemini-3.5-flash` |
| Pembuatan dan edit gambar | Text-to-image dan edit gambar | `gpt-image-2`, `flux-pro`, `qwen-image` |
| Pembuatan video | Text/image/reference-to-video dan kontrol gerak | `veo3.1`, `seedance-2.0`, `kling-3.0-video` |
| Audio dan Realtime | TTS, transkripsi, terjemahan, dan sesi realtime | `gpt-4o-mini-tts`, `whisper-1`, `gpt-realtime` |
| Embeddings dan rerank | Vektorisasi teks dan pemeringkatan ulang | `text-embedding-3-small`, `qwen3-vl-rerank` |
| Terjemahan | Alur terjemahan teks dan ucapan | `qwen-mt-plus`, `gpt-realtime-translate` |
| Musik dan 3D | Pembuatan musik dan asset 3D | `suno-music`, `tripo3d-v2.5` |
| Worlds | Pembuatan world, polling status, dan alur media asset | Periksa ketersediaan lewat `/v1/worlds/generations` dan Models API |

## Pilih Format Permintaan

Mulailah dengan rute yang sesuai dengan perilaku yang Anda butuhkan. Gunakan rute native Anthropic/Gemini untuk fitur native penyedia; gunakan `/v1/chat/completions` yang kompatibel dengan OpenAI saat memigrasikan kode chat bergaya OpenAI yang sudah ada.

<CodeGroup>

```bash cURL
curl https://api.aisonar.dev/v1/chat/completions \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-5.4",
    "messages": [
      {"role": "user", "content": "Explain what AI Sonar does in one sentence."}
    ]
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="sk-your-api-key",
    base_url="https://api.aisonar.dev/v1"
)

response = client.chat.completions.create(
    model="gpt-5.4",
    messages=[{"role": "user", "content": "Explain what AI Sonar does in one sentence."}]
)

print(response.choices[0].message.content)
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'sk-your-api-key',
  baseURL: 'https://api.aisonar.dev/v1'
});

const response = await client.chat.completions.create({
  model: 'gpt-5.4',
  messages: [{ role: 'user', content: 'Explain what AI Sonar does in one sentence.' }],
});

console.log(response.choices[0].message.content);
```

<Note>
Gunakan `POST /v1/responses` hanya ketika Anda secara eksplisit memerlukan perilaku khusus Responses. Itu bukan jalur kompatibilitas default di semua model.
</Note>

</CodeGroup>
