## Übersicht

Verwalten Sie API keys, Nutzermedien, Logs und generierte Assets sicher in AI Sonar-Integrationen.

Nutzen Sie öffentliche AI Sonar-APIs und Modellverträge; verlassen Sie sich nicht auf Provider-Ausführungsdetails.

## Wichtige Regeln

- Persistieren Sie API key scope, media URL lifetime, audit events, bevor Sie Nutzern Erfolg melden.
- Nutzen Sie öffentliche AI Sonar-APIs und Modellverträge; verlassen Sie sich nicht auf Provider-Ausführungsdetails.
- Machen Sie Wiederholungen idempotent und vermeiden Sie doppelte Generierung oder doppelte Abrechnung.

## Produktions-Checkliste

- Persistieren Sie API key scope, media URL lifetime, audit events, bevor Sie Nutzern Erfolg melden.
- Nutzen Sie öffentliche AI Sonar-APIs und Modellverträge; verlassen Sie sich nicht auf Provider-Ausführungsdetails.
- Machen Sie Wiederholungen idempotent und vermeiden Sie doppelte Generierung oder doppelte Abrechnung.
- Logs sollen dem Support helfen, aber keine Secrets oder privaten Payloads speichern.
- Prüfen Sie das Verhalten in Staging, bevor Produktionsverkehr geändert wird.
