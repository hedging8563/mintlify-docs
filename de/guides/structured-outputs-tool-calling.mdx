## Übersicht

Nutzen Sie JSON Schema, tool calls und native Formatfunktionen über OpenAI-, Anthropic- und Gemini-kompatible Routen.

Nutzen Sie öffentliche AI Sonar-APIs und Modellverträge; verlassen Sie sich nicht auf Provider-Ausführungsdetails.

## Wichtige Regeln

- Persistieren Sie schema version, tool name, validated arguments, bevor Sie Nutzern Erfolg melden.
- Nutzen Sie öffentliche AI Sonar-APIs und Modellverträge; verlassen Sie sich nicht auf Provider-Ausführungsdetails.
- Machen Sie Wiederholungen idempotent und vermeiden Sie doppelte Generierung oder doppelte Abrechnung.

## Produktions-Checkliste

- Persistieren Sie schema version, tool name, validated arguments, bevor Sie Nutzern Erfolg melden.
- Nutzen Sie öffentliche AI Sonar-APIs und Modellverträge; verlassen Sie sich nicht auf Provider-Ausführungsdetails.
