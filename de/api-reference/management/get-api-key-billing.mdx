---
title: "API-Schlüssel-Abrechnung abrufen"
openapi: "GET /v1/management/api-keys/{keyId}/billing"
description: "Gibt aggregierte Abrechnungsaufschlüsselungen für einen Benutzer-API-Schlüssel zurück."
---

## Übersicht

Dieser Endpunkt liefert verbrauchsbasierte Abrechnungsaggregate für den ausgewählten API-Schlüssel, einschließlich Gesamtkosten, Gesamtanfragen, Nutzung nach Abrechnungseinheit sowie Aufschlüsselungen nach angefordertem Modellnamen (`model`), Anbieter, `scene`, `accessChannel`, Cache-Klasse und täglichem Kostentrend.

## Hinweise

- Aufladungen, Rechnungen und andere Finanzdatensätze auf Organisationsebene sind ausdrücklich ausgeschlossen.

## Beispiel

<RequestExample>
```bash cURL
curl -X GET "https://api.aisonar.dev/v1/management/api-keys/key_abc123def456/billing?startDate=2026-01-01&endDate=2026-01-31" \
  -H "Authorization: Bearer mt-your-management-token"
```
</RequestExample>

## Anfrage / Antwort

Verwenden Sie das interaktive OpenAPI-Panel oben für das genaue Schema.

{

</ResponseExample>

## Wichtige Felder

<ResponseField name="total_cost" type="string">USD-Dezimalzeichenfolge für die Abrechnung.</ResponseField>
<ResponseField name="data" type="array">Zurückgegebene Liste oder Payload dieser Antwort.</ResponseField>
<ResponseField name="billing_transaction_id" type="string">Abrechnungskennung, wenn verfügbar.</ResponseField>
