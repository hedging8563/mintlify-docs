## Pfadparameter

<ParamField path="id" type="string" required>
  Die Task-ID, die von der ursprünglichen Bildgenerierungsanfrage zurückgegeben wurde.
</ParamField>

<Info>
Wenn die Antwort der Erstellung eine `poll_url` enthält, sollte vorzugsweise genau diese URL für das Polling aufgerufen werden. Einige Bildaufgaben können eine `poll_url` unter `/v1/tasks/{id}` anstelle des bildspezifischen Statuspfads bereitstellen.

Du kannst öffentliche Bild-Task-IDs wie `ldtask_...` auch über `/v1/images/{id}` abfragen. Dieser Kompatibilitätsalias verwendet dieselbe Authentifizierung, Ownership-Prüfung, Billing-Header, terminalen Snapshots und Antwortform wie `/v1/images/generations/{id}`.


Wenn die Aufgabe nicht mehr existiert oder sich über den öffentlichen async-task-Vertrag nicht mehr auflösen lässt, gibt AI Sonar `async_task_not_found` mit der Meldung `Task not found or no longer available.` zurück
</Info>

## Antwort

<ResponseField name="created" type="integer">
  Unix-Zeitstempel der Erstellung.
</ResponseField>

<ResponseField name="task_id" type="string">
  Die Aufgabenkennung.
</ResponseField>

<ResponseField name="status" type="string">
  Aufgabenstatus: `pending`, `processing`, `completed` oder `failed`.
</ResponseField>

<ResponseField name="data" type="array">
  Array der generierten Bilder (gefüllt, wenn der `status` `completed` ist).

  Jedes Objekt enthält:
  - `url` (string): URL des generierten Bildes
  - `revised_prompt` (string): Der für die Generierung verwendete Prompt
</ResponseField>

<ResponseField name="error" type="string">
  Fehlermeldung (nur vorhanden, wenn der `status` `failed` ist).
</ResponseField>

import requests

response = requests.get(
    "https://api.aisonar.dev/v1/tasks/ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa",
    headers={"Authorization": "Bearer sk-your-api-key"}
)

data = response.json()
print(f"Status: {data['status']}")

if data["status"] == "completed":
    print(f"Image URL: {data['data'][0]['url']}")

## Best Practices für Polling

<Tip>
**Empfohlenes Polling-Intervall**: 3-5 Sekunden. Die meisten Bildgenerierungsaufgaben werden je nach Modell und gewähltem Provider-Pfad innerhalb von 30-120 Sekunden abgeschlossen.
</Tip>

```python
import requests
import time

def poll_image_task(task_id, api_key, max_wait=300, interval=3):
    """Poll for image generation result with timeout."""
    url = f"https://api.aisonar.dev/v1/tasks/{task_id}"
    headers = {"Authorization": f"Bearer {api_key}"}

    start_time = time.time()
    while time.time() - start_time < max_wait:
        response = requests.get(url, headers=headers)
        data = response.json()

        if data["status"] == "completed":
            return data["data"][0]["url"]
        elif data["status"] == "failed":
            raise Exception(data.get("error", "Generation failed"))

        time.sleep(interval)

    raise TimeoutError(f"Task {task_id} did not complete within {max_wait}s")

# Usage
image_url = poll_image_task("ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa", "sk-your-api-key")
print(f"Generated image: {image_url}")
```
