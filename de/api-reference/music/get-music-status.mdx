openapi: "GET /v1/tasks/{id}"

---

<span data-mintlify-rebuild="2026-05-19-after-mdx-parse-fix" aria-hidden="true" />

Fragen Sie diesen Endpunkt ab, um den Status Ihrer Musik-Generierungsaufgabe zu überprüfen.

<Info>
Wenn die Create-Response `poll_url` enthält, sollten Sie zum Polling bevorzugt genau diese URL verwenden. `id` und `task_id` sollten als dieselbe asynchrone Task-ID behandelt werden.


Wenn die Aufgabe nicht mehr existiert oder über den öffentlichen Async-Task-Vertrag nicht mehr aufgelöst werden kann, gibt AI Sonar `async_task_not_found` mit der Meldung `Task not found or no longer available.` zurück
</Info>

## Pfad-Parameter

<ParamField path="id" type="string" required>
  Die ID der Musik-Generierungsaufgabe.
</ParamField>

## Antwort

<ResponseField name="id" type="string">
  Aufgaben-ID.
</ResponseField>

<ResponseField name="task_id" type="string">
  Alias der asynchronen Task-ID, wenn verfügbar.
</ResponseField>

<ResponseField name="status" type="string">
  Aufgabenstatus: `pending`, `processing`, `completed` oder `failed`.
</ResponseField>

<ResponseField name="progress" type="integer">
  Fortschritt in Prozent (0-100).
</ResponseField>

<ResponseField name="audio_url" type="string">
  URL zum Herunterladen der Audiodatei (wenn abgeschlossen).
</ResponseField>

<ResponseField name="video_url" type="string">
  URL zur Videoversion mit Visualizer (wenn abgeschlossen).
</ResponseField>

<ResponseField name="title" type="string">
  Generierter Songtitel.
</ResponseField>

<ResponseField name="lyrics" type="string">
  Generierter oder bereitgestellter Songtext.
</ResponseField>

<ResponseField name="error" type="string">
  Fehlermeldung (wenn fehlgeschlagen).
</ResponseField>

```python Python
import requests
import time

task_id = "ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa"

while True:
    response = requests.get(
        f"https://api.aisonar.dev/v1/tasks/{task_id}",
        headers={"Authorization": "Bearer sk-your-api-key"}
    )
    result = response.json()

    if result["status"] == "completed":
        print(f"Audio URL: {result['audio_url']}")
        break
    elif result["status"] == "failed":
        print(f"Error: {result['error']}")
        break

    time.sleep(5)  # Poll every 5 seconds
```

```javascript JavaScript
const taskId = 'ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa';

async function pollForMusic() {
  while (true) {
    const response = await fetch(
      `https://api.aisonar.dev/v1/tasks/${taskId}`,
      { headers: { 'Authorization': 'Bearer sk-your-api-key' } }
    );

    const result = await response.json();

    if (result.status === 'completed') {
      console.log(`Audio URL: ${result.audio_url}`);
      return result.audio_url;
    } else if (result.status === 'failed') {
      throw new Error(result.error);
    }

    await new Promise(r => setTimeout(r, 5000));
  }
}
```

```go Go
package main

import (
    "encoding/json"
    "fmt"
    "net/http"
    "time"
)

func main() {
    taskID := "ldtask_aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa"

    for {
        req, _ := http.NewRequest("GET",
            fmt.Sprintf("https://api.aisonar.dev/v1/tasks/%s", taskID), nil)
        req.Header.Set("Authorization", "Bearer sk-your-api-key")

        client := &http.Client{}
        resp, _ := client.Do(req)

        var result map[string]interface{}
        json.NewDecoder(resp.Body).Decode(&result)
        resp.Body.Close()

        if result["status"] == "completed" {
            fmt.Printf("Audio URL: %s\n", result["audio_url"])
            break
        } else if result["status"] == "failed" {
            fmt.Printf("Error: %s\n", result["error"])
            break
        }

        time.Sleep(5 * time.Second)
    }
}
```
