---
title: "Worlds auflisten"
openapi: "POST /v1/worlds/list"
description: "Listet abgeschlossene World Labs Marble Worlds aus AI Sonar auf"
---

Liste abgeschlossene World Labs Marble Worlds auf, die über AI Sonar erstellt wurden. Ergebnisse sind auf deine Organisation begrenzt; AI Sonar veröffentlicht nicht die kontoübergreifende Upstream-Liste.

Nutze `/v1/worlds/generations/{id}` für ausstehende oder fehlgeschlagene Generierungsaufgaben. Diese Liste enthält nur abgeschlossene Worlds, die AI Sonar nach der Generierung indexiert hat.

## Request Body

<ParamField body="page_size" type="integer" default="20">
  Anzahl der zurückgegebenen Worlds, 1 bis 100.
</ParamField>

<ParamField body="page_token" type="string">
  Paginierungs-Token aus der vorherigen Antwort.
</ParamField>

<ParamField body="status" type="string">
  Optionaler Statusfilter. AI Sonar gibt für `SUCCEEDED` abgeschlossene Worlds zurück; andere Status gehören zum Generierungs-Endpunkt.
</ParamField>

<ParamField body="model" type="string">
  Optionaler Marble-Modellfilter.
</ParamField>

<ParamField body="tags" type="array">
  Optionaler Tag-Filter.
</ParamField>

<ParamField body="is_public" type="boolean">
  Optionaler Sichtbarkeitsfilter.
</ParamField>

<ParamField body="created_after" type="string">
  Gibt nur Worlds zurück, die nach diesem Zeitpunkt erstellt wurden.
</ParamField>

<ParamField body="created_before" type="string">
  Gibt nur Worlds zurück, die vor diesem Zeitpunkt erstellt wurden.
</ParamField>

<ParamField body="sort_by" type="string" default="created_at">
  Sortieren nach `created_at` oder `updated_at`.
</ParamField>

<ParamField body="sort_order" type="string" default="desc">
  Sortierrichtung: `asc` oder `desc`.
</ParamField>

## Response

<ResponseField name="worlds" type="array">
  Array von World-Datensätzen.
</ResponseField>

<ResponseField name="next_page_token" type="string">
  Token für die nächste Seite oder `null`.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST "https://api.aisonar.dev/v1/worlds/list" \
  -H "Authorization: Bearer sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{"page_size": 20, "status": "SUCCEEDED", "model": "marble-1.1"}'
```

```python Python
import requests

response = requests.post(
    "https://api.aisonar.dev/v1/worlds/list",
    headers={"Authorization": "Bearer sk-your-api-key"},
    json={"page_size": 20, "status": "SUCCEEDED", "model": "marble-1.1"},
)
print(response.json()["worlds"])
```
