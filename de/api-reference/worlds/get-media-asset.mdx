---
title: "Medien-Asset abrufen"
openapi: "GET /v1/worlds/media-assets/{id}"
description: "Ruft ein über AI Sonar vorbereitetes World Labs Medien-Asset ab"
---

Rufe Metadaten für ein über AI Sonar vorbereitetes Medien-Asset ab. Der Zugriff ist auf die Organisation beschränkt, die das Asset erstellt hat.

## Path Parameters

<ParamField path="id" type="string" required>
  World Labs Medien-Asset-ID.
</ParamField>

## Response

<ResponseField name="media_asset_id" type="string">
  World Labs Medien-Asset-ID.
</ResponseField>

<ResponseField name="file_name" type="string">
  Ursprünglicher Dateiname, bis zu 64 Zeichen.
</ResponseField>

<ResponseField name="kind" type="string">
  Medientyp: `image` oder `video`.
</ResponseField>

<ResponseField name="metadata" type="object">
  Optionales Metadatenobjekt für das Upstream-Medien-Asset.
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.aisonar.dev/v1/worlds/media-assets/media_asset_123" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
import requests

response = requests.get(
    "https://api.aisonar.dev/v1/worlds/media-assets/media_asset_123",
    headers={"Authorization": "Bearer sk-your-api-key"},
)
print(response.json())
```

```javascript JavaScript
const response = await fetch('https://api.aisonar.dev/v1/worlds/media-assets/media_asset_123', {
  headers: { Authorization: 'Bearer sk-your-api-key' }
});
console.log(await response.json());
```
</RequestExample>
