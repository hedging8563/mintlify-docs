---
title: "World löschen"
openapi: "DELETE /v1/worlds/{id}"
description: "Löscht eine World Labs Marble World"
---

Lösche eine über AI Sonar erstellte Marble World. AI Sonar leitet die Löschung an World Labs weiter und markiert den lokalen World-Datensatz als gelöscht.

Das Löschen ist dauerhaft.

## Path Parameters

<ParamField path="id" type="string" required>
  World Labs world ID.
</ParamField>

## Response

<ResponseField name="deleted" type="boolean">
  Ob die Löschung akzeptiert wurde.
</ResponseField>

<ResponseField name="world_id" type="string">
  World Labs world ID.
</ResponseField>

<RequestExample>
```bash cURL
curl -X DELETE "https://api.aisonar.dev/v1/worlds/world_123" \
  -H "Authorization: Bearer sk-your-api-key"
```

```python Python
import requests

response = requests.delete(
    "https://api.aisonar.dev/v1/worlds/world_123",
    headers={"Authorization": "Bearer sk-your-api-key"},
)
print(response.json())
```

```javascript JavaScript
const response = await fetch('https://api.aisonar.dev/v1/worlds/world_123', {
  method: 'DELETE',
  headers: { Authorization: 'Bearer sk-your-api-key' }
});
console.log(await response.json());
```
</RequestExample>
