## Übersicht

Verwenden Sie diesen Endpunkt, um die aktuelle öffentliche Preisansicht für aktive Modelle abzurufen.

Für vollständige Preisdaten zu einem einzelnen Modell, einschließlich strukturierter Preise und Herkunft, verwenden Sie besser [Get Model Pricing](/de/api-reference/models/get-model).

## Query-Parameter

<ParamField query="provider" type="string">
  Optionaler Filter nach Anbieter, z. B. `openai`, `anthropic` oder `google`.
</ParamField>

<ParamField query="tag" type="string">
  Optionaler Modell-Tag-Filter, z. B. `chat`, `image`, `video`, `embedding` oder `translation`.
</ParamField>

## Antwort

<ResponseField name="object" type="string">
  Immer `pricing`.
</ResponseField>

<ResponseField name="updated_at" type="string">
  ISO-Zeitstempel für den neuesten Preis-Eintrag in der Antwort.
</ResponseField>

<ResponseField name="currency" type="string">
  Immer `USD`.
</ResponseField>

<ResponseField name="data" type="array">
  Array von Preis-Einträgen. Jeder Eintrag enthält `model`, `provider`, `is_lock_price`, `capabilities`, `aliases` sowie die öffentlichen skalaren oder strukturierten Preisfelder aus dem interaktiven Schema oben.
</ResponseField>

<RequestExample>

```bash cURL
curl "https://api.aisonar.dev/v1/pricing?provider=openai&tag=chat" \
```

```python Python
import requests

response = requests.get(
    "https://api.aisonar.dev/v1/pricing",
    headers={"Authorization": "Bearer sk-your-api-key"},
    params={"provider": "openai", "tag": "chat"}
)

print(response.json())
```

```go Go
package main

import (
    "encoding/json"
    "fmt"
    "net/http"
)

func main() {
    req, _ := http.NewRequest("GET", "https://api.aisonar.dev/v1/pricing?provider=openai&tag=chat", nil)
    req.Header.Set("Authorization", "Bearer sk-your-api-key")

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Println(result)
}
```

<Note>
Verwenden Sie das interaktive OpenAPI-Panel oben für das exakte Antwortschema. Öffentliche Preiseinträge können außerdem `lock_price_per_request`, `has_complex_pricing`, `supported_operations`, `pricing_summary` und `pricing_provenance` enthalten.
</Note>
