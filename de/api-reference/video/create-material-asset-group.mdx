Erstellt eine AI Sonar Seedance-Materialgruppe für die aktuelle Organisation.

Erstellen Sie eine Materialgruppe, wenn Sie wiederverwendbare Seedance 2.0-Materialien nach Bibliothekstyp organisieren möchten. Gruppen sind auf die authentifizierte Organisation beschränkt und können nicht kontoübergreifend verwendet werden.

Die meisten Integrationen können diesen Aufruf überspringen und beim Erstellen eines Materials `group_id` weglassen. AI Sonar verwendet oder erstellt die Standardgruppe der Organisation für den ausgewählten `library_type`.

- `aigc_avatar`: Materialien für virtuelle Personen oder Avatare.
- `liveness_face`: zertifizierte Materialien realer Personen.

Verwenden Sie die zurückgegebene `id` als `group_id` oder `material_group_id` in [Material erstellen](/de/api-reference/video/create-material-asset).
